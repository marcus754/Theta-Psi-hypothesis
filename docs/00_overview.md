# 00. Конспект Θ–Ψ с формулами (v2)

Этот файл теперь играет роль карты модулей. Он фиксирует только базу гипотезы
и связи между слоями:
- `ядро`: что предполагается о времени, гравитации и рефрактивном отклике;
- `фон`: как редуцированный FRW-контур строится поверх ядра;
- `observables`: как из фона и strong-field решения извлекаются наблюдаемые.

Сравнение с внешними моделями не входит в этот файл как часть физического
базиса. Оно относится только к отдельному observational-слою.


## Канон времени

- В рамках гипотезы в качестве фундаментального объекта времени принимается тензор `Θ_{μν}` (см. `docs/11_derivation_path.md`).
- В формулах этого раздела используется редуцированная FRW-переменная
  `θ ≡ θ_eff`, где `θ_eff^2 = Θ_{μν} u^μ u^ν`, `u^μ u_μ = -1`.
- Все записи вида `n(θ)` в этом файле следует читать как `n(θ_eff)`.
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику `Θ_{μν}`.

## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются `Θ_{μν}` и `Ψ`; редуцированный контур не
требует внешнего порога и не использует ручное отсечение веток.

```
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_{ΘΨ} + L_refr(Θ,Ψ) ].
```

Источник режима (без Weyl-порога в каноне):

```
I_grad(Θ,Ψ) = l_θ^2 · |∇_⊥θ|^2 + l_Ψ^2 · |∇_⊥Ψ|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^μ).
```

Смысл:
- В FRW (однородный фон): `I_grad=0`, поэтому рефрактивный сектор не получает
  источника и остаётся выключенным.
- В strong-field (большие пространственные градиенты): `I_grad>0`, и
  преломляющий сектор включается автоматически как прямое следствие
  взаимодействия `Θ` и `Ψ`.
- Время в гипотезе понимается как физический объект; поэтому в рабочем каноне допустимая
  динамика не должна переводить временной режим в нуль или отрицательную
  ветвь.

Рабочая редуцированная реализация этого принципа:

```
J_refr(Θ,Ψ)=Φ_eff(θ_eff)·I_grad(Θ,Ψ),
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr).
```

Невалидная ветка фона исключена параметризацией (а не фильтром):

```
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).
```

Переход режима понимается как прямое следствие взаимодействия `Θ` и `Ψ`, а не
как внешний переключатель.

## Ядро (seed)
- Единицы: 8πG=1, c=1. Пространство плоское; «геометрия» — язык, не онтология.
- Переносчик причинности — фотон; единственная физическая скорость — c=1 (dx/dτ=1).
- Преломление вакуума: dτ = dt / n(θ_eff). Это растяжение координатного времени и оптической длины пути, не «вторая скорость».
- Физическая ветвь модели: индекс времени остаётся конечным на реализованной ветке; поэтому истинный горизонт событий не формируется.
- Эффекты одной n(θ_eff): a = −∇ ln n; t = ∫ n(θ_eff) dl; Δν/ν ≈ Δ ln n(θ_eff); α ≈ ∫ ∇_⊥ ln n(θ_eff) dl.
- Однородный L_eff: 3/2(θ̇/θ)² + 1/2Ψ̇² − 1/2V0Ψ² + 1/2γ²θ̇²; Ψ=R e^{iφ}, Q=a³R²φ̇.
- ADM high‑k: K=G=diag(γ²+3/θ², 1, R²); M_RR = V0 + 3Q²/(a⁶R⁴); ΔM_ab — k⁰‑поправки (H, Ḣ, θ̇, Ṙ, Q).
- Скаляры luminal в high‑k; сверхсвет запрещён. Тензоры: c_T=1.
- Совместимость: универсальность n, изотропия, почти без дисперсии; weak-field
  matching остаётся условием на наблюдаемые, а не частью онтологии.
- Код — инструмент верификации; источники истины — этот документ и background/frw_symbolic.py.

Этот файл — краткое человеческое изложение гипотезы на базе `Θ_{μν}, Ψ` с опорными формулами. Код служит инструментом для проверки, но источником истины считаем согласованные записи здесь и в `background/frw_symbolic.py`.

## Модульность гипотезы

1. База гипотезы:
   `Θ_{μν}`, `Ψ`, положительность времени, `J_refr`, `n`, `ĝ_{μν}`.
2. Гравитационный слой:
   гравитация как отклик на неоднородность временного режима и причинной
   проводимости вакуума.
3. Фоновая космология:
   отдельная FRW-редукция из базиса, но не весь базис.
4. Strong-field:
   отдельный стационарный и волновой слой, использующий тот же `J_refr`.
5. Observables / fitting:
   отдельный слой сравнения с данными и внешними baseline.

## Единицы и параметры
- Единицы: выбраны такие, что уравнение Фридмана имеет вид `H^2 = ρ_tot / 3` (т.е. 8πG=1, c=1).
- Параметры модели: `γ` — связь времени и энергии (в реальности γ_phys ≲ 10⁻¹⁸), `V0` — квадратичный потенциал субстрата, `Q` — фазовый заряд комплексной Ψ.
- Конверсия `H0` из км/с/Мпк в эти единицы: см. `background/components.py` (`omegas_from_km_s_Mpc`).

## Каузальность
- Переносчик причинности — фотон. Единственная физическая скорость — световая, c=1.
- Гравитация — преломление вакуума: вакуум ведёт себя как среда с показателем n(θ_eff), который растягивает координатное время: dτ = dt / n(θ_eff). Это растяжение времени пути, а не изменение максимальной скорости.
- Отсюда следуют классические эффекты: t = ∫ n(θ_eff) dl (задержка Шапиро), Δν/ν ≈ Δ ln n(θ_eff) (красное смещение), α ≈ ∫ ∇_⊥ ln n(θ_eff) dl (линзирование).
- Тензорный сектор немодифицирован: c_T = 1. Скалярные моды в high‑k люминальны (K=G ⇒ c_s = 1). Любая сверхсветовость (c_s²>1) запрещена.

## Интерпретация (аналоговая картина)
- Геометрия — язык. Физика здесь: вакуум с показателем преломления n(θ_eff) растягивает координатное время (dτ = dt / n(θ_eff)).
- Единственная физическая скорость — c=1; преломление — это оптическая длина пути t = ∫ n(θ_eff) dl, а не «другая скорость света».
- Все эффекты гравитации сводятся к градиенту ln n(θ_eff): свободное падение, красное смещение, задержка Шапиро, линзирование.

## Минимальный рабочий класс преломляющего сектора

Чтобы переход

```
Θ, Ψ  ->  J_refr(Θ,Ψ)  ->  n(θ_eff,Θ,Ψ)  ->  ĝ_{μν}
```

был зафиксирован не только словами, в каноне берётся минимальный класс

```
L_refr = M_*^4 · F(J_refr),
J_refr = Φ_eff(θ_eff) · I_grad(Θ,Ψ),
Φ_eff(θ_eff) = ln(θ_eff / θ_0).
```

Смысл:
- `Φ_eff` — канонический weak-field показатель временного режима;
- `I_grad` измеряет силу пространственной неоднородности взаимодействия `Θ`
  и `Ψ`;
- `F` задаёт универсальный отклик вакуума, одинаковый для света, часов и
  массивных тел через общую эффективную метрику `ĝ_{μν}`.

В рабочей записи это записывается как

```
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr).
```

Тем самым `n` выступает не произвольной ручкой, а сжатой записью универсального
рефрактивного отклика вакуума на прямой инвариант взаимодействия `Θ` и `Ψ`.

Канонический normalized representative-form:

```
F_v1(J) = 1 - exp(-J).
```

Он выбран потому, что сразу даёт:
- `F(0)=0` и отсутствие искусственного фонового включения;
- `F'(J)>0` и монотонное усиление режима;
- `F''(J)<0` и плавное насыщение без порога;
- включение strong-field ветви напрямую через рост `I_grad`, без третьего поля.

Но это пока не вывод из действия: действие фиксирует `J_refr` и требования к
`F`, а не единственную функцию `F(J)`. Явный `J_*` здесь не нужен и считается
обычной нормировкой `J_refr`.

Минимальные леммы канона:
- при `I_grad=0` рефрактивный сектор на FRW не получает источника;
- при `𝒩'(J)>0` и конечном насыщении всегда `1 <= n(J) < ∞`;
- при конечной энергии и `θ(t0)>0` состояние `θ=0` не достигается за конечное
  время;
- универсальность `ĝ_{μν}` делает свободное падение следствием, а не отдельным
  постулатом.

## Откуда берётся гравитация
- Источник гравитации в этой рамке — не отдельная "сила притяжения", а
  пространственная неоднородность временного режима, задаваемого `Θ_{μν}`.
- В редуцированном описании это читается как неоднородность `θ_eff(x)` и,
  следовательно, `n(θ_eff(x))`.
- Если `n` зависит от положения, то локальный ход процессов, частот и
  оптической длины пути зависит от положения тоже. Именно этот градиент и
  даёт наблюдаемое ускорение:

```
a = -∇ ln n(θ_eff).
```

- Поэтому гравитация здесь не первичная силовая сущность, а отклик тел и
  сигналов на неоднородную причинно-временную структуру вакуума.

## Что значит «преломление вакуума»
- Это не утверждение, что "ломается только свет". В первую очередь меняется
  сама проводимость вакуума по отношению к причинным процессам.
- Свет проявляет это как изгиб луча и увеличение времени пролёта.
- Частицы проявляют это как изменение траектории.
- Часы проявляют это как замедление локального хода времени.
- Поэтому слово "преломление" в гипотезе надо читать широко: преломляется не
  один фотон, а вакуумный режим переноса причинности и развёртывания процессов.

## Положительность времени и компактные объекты
Физический тезис гипотезы: время не является пустой координатой и потому не
может физически обнулиться вдоль допустимой траектории. В текущем редуцированном
контуре это кодируется, в частности, условием:

```
1 ≤ n(θ_eff) < ∞.
```

Следствия:

```
1 + z = n_emit / n_obs < ∞,
Δt = ∫_Γ n(θ_eff) dl < ∞.
```

То есть сигнал с поверхности сверхкомпактного объекта может выйти всегда, но при очень большом `n_emit` будет чудовищно красносмещён и практически ненаблюдаем. Регуляризованная форма здесь — рабочая запись более общего требования: временной режим остаётся физически допустимым и положительным.

## Фоновый модуль

Этот раздел относится только к FRW-редукции. Он не исчерпывает гипотезу и не
должен читаться как её полный канон.

## Однородный Лагранжиан и основные величины
Поля: `θ(t)` — норма поля времени Θ; `R(t)` — амплитуда комплексного энергетического субстрата `Ψ = R e^{iφ}`. Эффективный однородный лагранжиан:

```
L_eff = 3/2 (θ̇/θ)^2 + 1/2 Ṙ^2 − 1/2 V0 R^2 + 1/2 γ^2 θ̇^2.
```

Из него следуют плотность энергии и давление (с точностью до фактора `a^3`):

```
ρ = (∂L/∂θ̇) θ̇ + (∂L/∂Ψ̇) Ψ̇ − L,
p = L.

Полезные тождества:  ρ − p = V0 R^2;  ρ + p = γ^2 θ̇^2 + Ṙ^2 + 3 (θ̇/θ)^2.
```

Расширение задаётся 
```
H^2 = (ρ_fields + ρ_m + ρ_r + ρ_Λ) / 3,
```
где `ρ_fields` — вклад Θ–Ψ, а фоновые компоненты задаются `Omegas` (материя, излучение, Λ).

## Регуляризующая переменная x
Уравнение движения для θ содержит особую координатную точку при `1 − γ²θ²/3 = 0`. Вводим безопасную переменную

```
θ = (√3/γ) tanh x,
θ̇ = (√3/γ) sech²x · ẋ,
```

тогда она переносится в предел `|x| → ∞`. Это координатная (численная) регуляризация уравнений, а не отдельный физический постулат о горизонте. Уравнение в x (при произвольном H(t)):

```
ẍ + 3H ẋ = ẋ² ( 2 tanh x + coth x ).
```

## Комплексная Ψ = R e^{iφ}
Вводим заряд `Q = a³ R² φ̇` (интеграл движения). Тогда

```
R̈ + 3H Ṙ − Q²/(a⁶ R³) + V0 R = 0,
φ̇ = Q / (a³ R²),
ρ_phase = p_phase = 1/2 · Q²/(a⁶ R²).
```

В пределе `Q → 0` остаётся реальный прототип с чисто «пружинным» членом V0 R.

## Интегрирование по N = ln a
Для численной устойчивости удобно перейти к N. Обозначим `y = dx/dN`, `u = dR/dN`. Тогда

```
ρ_fields = A(x,y,u) · H² + 1/2 V0 R²,   где
A(x,y,u) = 3/2 · (sech⁴x / tanh²x) · y² + 1/2 · u² + 3/2 · sech⁴x · y².

H² = [ ρ_m(a) + ρ_r(a) + ρ_Λ(a) + 1/2 V0 R² + 1/2 Q²/(a⁶ R²) ] / (3 − A).

Ḣ = −1/2 [ (γ² θ̇² + Ṙ² + 3 (θ̇/θ)²) + ρ_m + 4/3 ρ_r + Q²/(a⁶ R²) ].
```

Из уравнений для ẍ и R̈ получаем уравнения в N для `y′(N), u′(N)` и `t′(N)=1/H` (см. `background/frw_background.py`).

## Скалярные возмущения (ADM)
Квадратичное действие для скаляров в базовом виде:

```
L^(2) = 1/2 · δq̇ᵀ K δq̇ − 1/2 · δqᵀ [ G (k²/a²) + M ] δq,
δq = (δθ, δR, δφ).
```

Диагональная двухсущностная high‑k модель (согласована с фоном):

```
Kθθ = 3/θ² + γ²,
KRR = 1,
Kφφ = R²,

Gθθ = Kθθ,
GRR = 1,
Gφφ = R²,

MRR = V0 + 3 Q²/(a⁶ R⁴),  Mθθ = Mφφ = 0.
```

Смешения (малые ε) и диагональные поправки вида `+ α_i (k²/a²)` допускаются и заданы явно в интерфейсе (`theory/adm_symbolic.py`).

Условия здоровья:
- Нет призраков: `K ≻ 0` (положительно определена).
- Нет градиентных инстабильностей: собственные значения `K⁻¹ G` положительны.
- Субсветовость: `c_s² ≤ 1` (с небольшим запасом над единицей во избежание ложных флагов из‑за округления).
- Для конечных k: `ω²(k)` — собственные значения `K⁻¹ [ G (k²/a²) + M ]` положительны.

Примечание: в коде реализована численно устойчивая оценка собственных значений через холецки‑редукцию (см. `src/linear_modes_adm.py`). Символический каркас для будущего полного исключения lapse/shift — в `theory/adm_symbolic.py`.

Исключение lapse/shift (c = (δN, B)):

```
L^(2) ⊃ − δqᵀ B c − 1/2 · cᵀ C c  ⇒  c* = − C⁻¹ Bᵀ δq,
A_eff = A − B C⁻¹ Bᵀ,   A = G (k²/a²) + M.
```

Отсюда восстанавливаются G_eff и M_eff (слагаемые при k²/a² и постоянные). В численном API передача `elim={...}` включает этот шаг (см. `src/linear_modes_adm.py`).

## Принцип эквивалентности: «почему» здесь работает

В этой рамке эквивалентность не постулируется, а вытекает из одного «индекса времени» вакуума `n(θ_eff)`:

```
dτ = dt / n(θ_eff),
L ≈ (m/2n) v² − m/n  ⇒  a = −∇ ln n(θ_eff),
```

масса сокращается, и все тела падают одинаково (mграв = mинерт). Фотоны видят тот же `n(θ_eff)` — красное смещение, задержка Шапиро и линзирование следуют из той же функции. Подробно см. `docs/01_interpretation.md`.

## Проверки здоровья и отсечки
- Конечные k: положительность ω²(k) для всех мод (если включено).
- (Опционально) эвристики для быстрых сканов: EFT‑срез и PPN‑прокси; по умолчанию отключены,
  основными считаются только условия из лагранжиана и уравнений движения.
- BBN: монотонное убывание H(a) на `a ∈ [1e−10, 1e−6]` (радиационная эпоха).
- Суперсветовость: `max eig(K⁻¹ G) ≤ 1 + tol` (2×2 или ADM 3×3).

Все проверки собраны в единый агрегатор `checks/health.py`.

## Наблюдаемые и фит
- Фоновые: `H(z)`, расстояния (SN, BAO). Для примера — `fitting/` с демо CC‑данными и лёгким Metropolis.
- Для реальных H0 используйте конверсию `omegas_from_km_s_Mpc`.
- Скан параметров (γ, V0[, Q]) с «жёстким стопом» по здоровью: `scripts/run_grid_scan.py` (`--health`, `--adm`, `--finite_k`, `--lnN`).
