# 00. Конспект Theta–Psi с формулами (v2)

Файл играет роль карты модулей. Он фиксирует только базу гипотезы и связи между слоями:
- ядро: что предполагается о времени, гравитации и рефрактивном отклике;
- фон: как редуцированный FRW-контур строится поверх ядра;
- наблюдаемые: как из фона и сильное поле решения извлекаются наблюдаемые.

Сравнение с внешними моделями не входит в этот файл как часть физического базиса. Оно относится только к отдельному observational-слою.

## Канон времени

- В рамках гипотезы в качестве фундаментального объекта времени принимается тензор Theta_munu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная
  theta ≡ theta_eff, где theta_eff^2 = Theta_munu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную тензорную динамику Theta_munu.

## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются Theta_munu и Psi; редуцированный контур не требует внешнего порога и не использует ручное отсечение веток.
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + L_refr(Theta,Psi) ].

Источник режима (без Weyl-порога в каноне):
I_grad(Theta,Psi) = l_theta² |∇_⊥theta|² + l_Psi² |∇_⊥Psi|²
∇_⊥ = пространственная часть градиента (ортогонально локальному u^mu).

Смысл:
- В FRW (однородный фон): I_grad=0, поэтому рефрактивный сектор не получает источника и остаётся выключенным.
- В сильное поле (большие пространственные градиенты): I_grad>0, и преломляющий сектор включается автоматически как прямое следствие взаимодействия Theta и Psi.
- Время в гипотезе понимается как физический объект; поэтому в рабочем каноне допустимая динамика не должна переводить временной режим в нуль или отрицательную ветвь.

Рабочая редуцированная реализация этого принципа:
J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi),
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).

Невалидная ветка фона исключена параметризацией (а не фильтром):
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).

Переход режима понимается как прямое следствие взаимодействия Theta и Psi, а не как внешний переключатель.

## Ядро (seed)
- Единицы: 8πG=1, c=1. Пространство плоское; «геометрия» — язык, не онтология.
- Причинная структура задаётся общим конусом распространения; единственная физическая скорость — c=1 (dx/dtau=1).
- Преломление вакуума: dtau = dt / n(theta_eff). Это растяжение координатного времени и оптической длины пути, не «вторая скорость».
- Физическая ветвь модели: индекс времени остаётся конечным на реализованной ветке; поэтому истинный горизонт событий не формируется.
- Эффекты одной n(theta_eff): a = −∇ ln n; t = ∫ n(theta_eff) dl; Δnu/nu ≈ Δ ln n(theta_eff); α ≈ ∫ ∇_⊥ ln n(theta_eff) dl.
- Однородный L_eff: 3/2(thetȧ/theta)² + 1/2Psi̇² − 1/2V0Psi² + 1/2gamma²thetȧ²; Psi=R e^iφ, Q=a³R²φ̇.
- ADM high‑k: K=G=diag(gamma²+3/theta², 1, R²); M_RR = V0 + 3Q²/(a⁶R⁴); ΔM_ab — k⁰‑поправки (H, Ḣ, thetȧ, Ṙ, Q).
- Скаляры luminal в high‑k; сверхсвет запрещён. Тензоры: c_T=1.
- Совместимость: универсальность n, изотропия, почти без дисперсии; слабое поле
  согласование остаётся условием на наблюдаемые, а не частью онтологии.
- Код — инструмент верификации; опорные источники определения модели — этот документ и background/frw_symbolic.py.

Этот файл — краткое человеческое изложение гипотезы на базе Theta_munu, Psi с опорными формулами. Код служит инструментом для проверки, но каноническими источниками определения модели считаются согласованные записи здесь и в background/frw_symbolic.py.

## Модульность гипотезы

1. База гипотезы:
   Theta_munu, Psi, положительность времени, J_refr, n, g_hat_munu.
2. Гравитационный слой: гравитация как отклик на неоднородность временного режима и причинной проводимости вакуума.
3. Фоновая космология: отдельная FRW-редукция из базиса, но не весь базис.
4. Сильное поле: отдельный стационарный и волновой слой, использующий тот же J_refr.
5. Наблюдаемые / fitting: отдельный слой сравнения с данными и внешними baseline.

## Единицы и параметры
- Единицы: выбраны такие, что уравнение Фридмана имеет вид H^2 = ρ_tot / 3 (т.е. 8πG=1, c=1).
- Параметры модели: gamma — связь времени и энергии (в реальности gamma_phys ≲ 10⁻¹⁸), V0 — квадратичный потенциал субстрата, Q — фазовый заряд комплексной Psi.
- Конверсия H0 из км/с/Мпк в эти единицы: см. background/components.py (omegas_from_km_s_Mpc).

## Каузальность
- Причинная структура задаётся общим конусом распространения. Единственная физическая скорость — световая, c=1.
- Гравитация — преломление вакуума: вакуум приобретает эффективный показатель преломления с показателем n(theta_eff), который растягивает координатное время: dtau = dt / n(theta_eff). Это растяжение времени пути, а не изменение максимальной скорости.
- Отсюда следуют классические эффекты: t = ∫ n(theta_eff) dl (задержка Шапиро), Δnu/nu ≈ Δ ln n(theta_eff) (красное смещение), α ≈ ∫ ∇_⊥ ln n(theta_eff) dl (линзирование).
- Тензорный сектор немодифицирован: c_T = 1. Скалярные моды в high‑k распространяются со световой скоростью (K=G ⇒ c_s = 1). Любая сверхсветовость (c_s²>1) запрещена.

## Интерпретация (аналоговая картина)
- Геометрия — язык. Физика здесь: вакуум с показателем преломления n(theta_eff) растягивает координатное время (dtau = dt / n(theta_eff)).
- Единственная физическая скорость — c=1; преломление — это оптическая длина пути t = ∫ n(theta_eff) dl, а не «другая скорость света».
- Все эффекты гравитации сводятся к градиенту ln n(theta_eff): свободное падение, красное смещение, задержка Шапиро, линзирование.

## Минимальный рабочий класс преломляющего сектора

Чтобы переход
Theta, Psi  ->  J_refr(Theta,Psi)  ->  n(theta_eff,Theta,Psi)  ->  g_hat_munu

был зафиксирован не только словами, в каноне берётся минимальный класс
L_refr = M_*^4 · F(J_refr),
J_refr = Phi_eff(theta_eff) · I_grad(Theta,Psi),
Phi_eff(theta_eff) = ln(theta_eff / theta_0).

Смысл:
- Phi_eff — канонический слабое поле показатель временного режима;
- I_grad измеряет силу пространственной неоднородности взаимодействия Theta  и Psi;
- F задаёт универсальный отклик вакуума, одинаковый для света, часов и массивных тел через общую эффективную метрику g_hat_munu.

В рабочей записи это записывается как
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).

Тем самым n выступает не произвольной ручкой, а сжатой записью универсального рефрактивного отклика вакуума на прямой инвариант взаимодействия Theta и Psi.

Канонический normalized representative-form:
F_v1(J) = 1 - exp(-J).

Он выбран потому, что сразу даёт:
- F(0)=0 и отсутствие искусственного фонового включения;
- F'(J)>0 и монотонное усиление режима;
- F''(J)<0 и плавное насыщение без порога;
- включение сильное поле ветви напрямую через рост I_grad, без третьего поля.

Но это пока не вывод из действия: действие фиксирует J_refr и требования к F, а не единственную функцию F(J). Явный J_* здесь не нужен и рассматривается как нормировка J_refr.

Минимальные леммы канона:
- при I_grad=0 рефрактивный сектор на FRW не получает источника;
- при 𝒩'(J)>0 и конечном насыщении всегда 1 <= n(J) < ∞;
- при конечной энергии и theta(t0)>0 состояние theta=0 не достигается за конечное время;
- универсальность g_hat_munu делает свободное падение следствием, а не отдельным постулатом.

## Откуда берётся гравитация
- Источник гравитации в этой рамке — не отдельная сила притяжения, а пространственная неоднородность временного режима, задаваемого Theta_munu.
- В редуцированном описании это читается как неоднородность theta_eff(x) и, следовательно, n(theta_eff(x)).
- Если n зависит от положения, то локальный ход процессов, частот и оптической длины пути зависит от положения тоже. Именно этот градиент и даёт наблюдаемое ускорение: 
a = -∇ ln n(theta_eff).
- Поэтому гравитация здесь не первичная силовая сущность, а отклик тел и сигналов на неоднородную причинно-временную структуру вакуума.

## Что значит «преломление вакуума»
- Это не утверждение, что "ломается только свет". В первую очередь меняется сама проводимость вакуума по отношению к причинным процессам.
- Свет проявляет это как изгиб луча и увеличение времени пролёта.
- Частицы проявляют это как изменение траектории.
- Часы проявляют это как замедление локального хода времени.
- Поэтому слово "преломление" в гипотезе надо читать широко: преломляется не один фотон, а вакуумный режим переноса причинности и развёртывания процессов.

## Положительность времени и компактные объекты
Физический тезис гипотезы: время не является пустой координатой и потому не может физически обнулиться вдоль допустимой траектории. В текущем редуцированном контуре это кодируется, в частности, условием:
1 ≤ n(theta_eff) < ∞.

Следствия:
1 + z = n_emit / n_obs < ∞,
Δt = ∫_Γ n(theta_eff) dl < ∞.

То есть сигнал с поверхности сверхкомпактного объекта может выйти всегда, но при очень большом n_emit будет чудовищно красносмещён и практически ненаблюдаем. Регуляризованная форма здесь — рабочая запись более общего требования: временной режим остаётся физически допустимым и положительным.

## Фоновый модуль

Этот раздел относится только к FRW-редукции. Он не исчерпывает гипотезу и не должен читаться как её полный канон.

## Однородный Лагранжиан и основные величины
Поля: theta(t) — норма поля времени Theta; R(t) — амплитуда комплексного энергетического субстрата Psi = R e^iφ. Эффективный однородный лагранжиан:
L_eff = 3/2 (thetȧ/theta)^2 + 1/2 Ṙ^2 − 1/2 V0 R^2 + 1/2 gamma^2 thetȧ^2.

Из него следуют плотность энергии и давление (с точностью до фактора a^3):
ρ = (∂L/∂thetȧ) thetȧ + (∂L/∂Psi̇) Psi̇ − L,
p = L.

Полезные тождества:  ρ − p = V0 R^2;  ρ + p = gamma^2 thetȧ^2 + Ṙ^2 + 3 (thetȧ/theta)^2.
Расширение задаётся 
H^2 = (ρ_fields + ρ_m + ρ_r + ρ_Λ) / 3,
где ρ_fields — вклад Theta–Psi, а фоновые компоненты задаются Omegas (материя, излучение, Λ).

## Регуляризующая переменная x
Уравнение движения для theta содержит особую координатную точку при 1 − gamma²theta²/3 = 0. Вводим безопасную переменную
theta = (√3/gamma) tanh x,
thetȧ = (√3/gamma) sech²x · ẋ,

тогда она переносится в предел |x| → ∞. Это координатная (численная) регуляризация уравнений, а не отдельный физический постулат о горизонте. Уравнение в x (при произвольном H(t)):
ẍ + 3H ẋ = ẋ² ( 2 tanh x + coth x ).

## Комплексная Psi = R e^iφ
Вводим заряд Q = a³ R² φ̇ (интеграл движения). Тогда
R̈ + 3H Ṙ − Q²/(a⁶ R³) + V0 R = 0,
φ̇ = Q / (a³ R²),
ρ_phase = p_phase = 1/2 · Q²/(a⁶ R²).

В пределе Q → 0 остаётся реальный прототип с чисто «пружинным» членом V0 R.

## Интегрирование по N = ln a
Для численной устойчивости удобно перейти к N. Обозначим y = dx/dN, u = dR/dN. Тогда
ρ_fields = A(x,y,u) · H² + 1/2 V0 R²,   где
A(x,y,u) = 3/2 · (sech⁴x / tanh²x) · y² + 1/2 · u² + 3/2 · sech⁴x · y².
H² = [ ρ_m(a) + ρ_r(a) + ρ_Λ(a) + 1/2 V0 R² + 1/2 Q²/(a⁶ R²) ] / (3 − A).
Ḣ = −1/2 [ (gamma² thetȧ² + Ṙ² + 3 (thetȧ/theta)²) + ρ_m + 4/3 ρ_r + Q²/(a⁶ R²) ].

Из уравнений для ẍ и R̈ получаем уравнения в N для y′(N), u′(N) и t′(N)=1/H (см. background/frw_background.py).

## Скалярные возмущения (ADM)
Квадратичное действие для скаляров в базовом виде:
L^(2) = 1/2 · dq̇ᵀ K dq̇ − 1/2 · dqᵀ [ G (k²/a²) + M ] dq,
dq = (d_theta, d_R, deltaφ).

Диагональная двухсущностная high‑k модель (согласована с фоном):
Kthetatheta = 3/theta² + gamma²,
KRR = 1,
Kφφ = R²,

Gthetatheta = Kthetatheta,
GRR = 1,
Gφφ = R²,

MRR = V0 + 3 Q²/(a⁶ R⁴),  Mthetatheta = Mφφ = 0.

Смешения (малые ε) и диагональные поправки вида + α_i (k²/a²) допускаются и заданы явно в интерфейсе (theory/adm_symbolic.py).

Условия здоровья:
- Нет призраков: K ≻ 0 (положительно определена).
- Нет градиентных инстабильностей: собственные значения K⁻¹ G положительны.
- Субсветовость: c_s² ≤ 1 (с учётом численной погрешности).
- Для конечных k: ω²(k) — собственные значения K⁻¹ [ G (k²/a²) + M ] положительны.

Примечание: в коде реализована численно устойчивая оценка собственных значений через холецки‑редукцию (см. src/linear_modes_adm.py). Символический каркас для будущего полного исключения Лапса/Шифта — в theory/adm_symbolic.py.

Исключение Лапса/Шифта (c = (d_N, B)):
L^(2) ⊃ − dqᵀ B c − 1/2 · cᵀ C c  ⇒  c* = − C⁻¹ Bᵀ dq,
A_eff = A − B C⁻¹ Bᵀ,   A = G (k²/a²) + M.

Отсюда восстанавливаются G_eff и M_eff (слагаемые при k²/a² и постоянные). В численном API передача elim=... включает этот шаг (см. src/linear_modes_adm.py).

## Принцип эквивалентности: «почему» здесь работает

В этой рамке эквивалентность не постулируется, а вытекает из одного «индекса времени» вакуума n(theta_eff):
dtau = dt / n(theta_eff),
L ≈ (m/2n) v² − m/n  ⇒  a = −∇ ln n(theta_eff),

масса сокращается, и все тела падают одинаково (mграв = mинерт). Фотоны видят тот же n(theta_eff) — красное смещение, задержка Шапиро и линзирование следуют из той же функции. Подробно см. docs/01_interpretation.md.

## Проверки здоровья и отсечки
- Конечные k: положительность ω²(k) для всех мод (если включено).
- (Опционально) эвристики для быстрых сканов: предел эффективной теории и PPN‑прокси; по умолчанию отключены, основными считаются только условия из лагранжиана и уравнений движения.
- BBN: монотонное убывание H(a) на a ∈ [1e−10, 1e−6] (радиационная эпоха).
- Суперсветовость: max eig(K⁻¹ G) ≤ 1 + tol (2×2 или ADM 3×3).

Все проверки собраны в единый агрегатор checks/health.py.
