# 07. Тензорный сектор, оптическая метрика и PPN


## Канон времени

- Фундаментальный объект времени в гипотезе: тензор `Θ_{μν}` (см. `docs/11_derivation_path.md`).
- В формулах этого раздела используется редуцированная FRW-переменная
  `θ ≡ θ_eff`, где `θ_eff^2 = Θ_{μν} u^μ u^ν`, `u^μ u_μ = -1`.
- Все записи вида `n(θ)` в этом файле следует читать как `n(θ_eff)`.
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику `Θ_{μν}`.

## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются `Θ_{μν}` и `Ψ`; редуцированный контур не
требует внешнего порога и не использует ручное отсечение веток.

```
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_{ΘΨ}
                   + M_*^4 F(J_refr) ],
J_refr(Θ,Ψ)=Φ_eff(θ_eff)·I_grad(Θ,Ψ).
```

Источник режима (без Weyl-порога в каноне):

```
I_grad(Θ,Ψ) = l_θ^2 · |∇_⊥θ|^2 + l_Ψ^2 · |∇_⊥Ψ|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^μ).
```

Смысл:
- В FRW (однородный фон): `I_grad=0`, поэтому `J_refr=0`.
- В strong-field (большие пространственные градиенты): `I_grad>0`, поэтому
  преломляющий сектор включается напрямую через `F(J_refr)`.
- Отдельного режимного поля в каноне нет.

Индекс вакуума ограничен конструктивно:

```
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr).
```

Невалидная ветка фона исключена параметризацией (а не фильтром):

```
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).
```

В наблюдательных проверках оценивается не «ручное включение», а прямой ответ
на пространственные неоднородности `Θ_{μν}, Ψ`.

Раздел фиксирует связку: `c_T = 1`, универсальная оптическая метрика `n(θ_eff)` и слабопольный PPN-предел `γ=β=1`.

## 1) Скорость тензорных волн (c_T)

Ключевой вопрос: не изменяет ли модель скорость гравитационных волн (c_T),
сильно ограниченную наблюдениями LIGO/Virgo (|c_T/c − 1| ≲ 10⁻¹⁵)?

В текущей версии гипотезы на базе `Θ_{μν}, Ψ` гравитационный сектор не модифицирован;
поля (θ, Ψ) относятся к «материи»
(источнике для фона и скалярных возмущений). В эффективной параметризации
модифицированной гравитации это соответствует α_T = 0, что строго даёт
c_T = 1 для тензорного сектора на FRW-фоне.

Практический вывод:
- В коде (theory/tensor.py) заданы свойства тензорного сектора с c_T^2 = 1.
- Тест (tests/test_tensor_speed.py) проверяет, что возвращается c_T^2 = 1.
- Если в будущем будут добавляться операторы, меняющие кинетику гравитации,
  они должны быть явно включены и немедленно проверяться на совместимость с
  ограничениями LIGO/Virgo, т.е. α_T ≈ 0 в поздние эпохи.

## 2) Оптическая метрика из n(θ_eff) и эмерджентные тензорные волны

Цель: зафиксировать «из первых принципов» связку
- один индекс времени вакуума `n(θ_eff)` (преломление), принцип наименьшего времени;
- единая оптическая метрика для всей материи;
- возникновение тензорного сектора (две TT‑поляризации, c_T = 1) как
  универсального низкоэнергетического ответа среды.

## 1) Один `n(θ_eff)` и принцип Ферма

Постулат: локальный ход собственного времени у материи и света растянут одним и тем же индексом

```
dτ = dt / n(θ_eff).
```

Физическая постановка берёт ограниченный индекс:

```
1 ≤ n(θ_eff) < ∞.
```

Отсюда в сильном поле получаем не «истинный горизонт», а безгоризонтный предел с очень большим, но конечным красным смещением.

Лучи и траектории минимизируют оптическую длину (принцип Ферма)

```
	L_opt[Γ] = ∫_Γ n(θ_eff) dl,
```

что уже даёт три классических эффекта слабого поля из одного `n(θ_eff)`:
красное смещение, задержку Шапиро, линзирование (см. 01_interpretation.md).

## 2) Оптическая метрика ĝ(n, u)

Вариационный принцип удобно записать ковариантно через «оптическую метрику»
типа Гордона. Пусть

- `u_μ ≡ ∂_μ θ / √(∂θ·∂θ)` — выделенное таймлайк‑направление («часы»);
- базовая фонова метрика в локально инерциальных координатах — η.

Тогда эффективная метрика, по геодезикам которой идут свет и материя, может быть записана как

```
ĝ_{μν}(θ) = η_{μν} + (n(θ_eff)^2 − 1) u_μ u_ν
```

(эквивалентные записи встречаются и с коэффициентом при `u_μ u_ν`, зависящим от выбранной нормировки; существенен сам факт существования единой ĝ, кодирующей `n(θ_eff)`). В локально комовинговом кадре `u^μ = (1,0,0,0)` и `ĝ_{00} = n^2`, так что `dτ^2 = dt^2 / n^2`.

Следствие: «преломление вакуума» сводится к геодезикам одной и той же ĝ(θ) для всех полей материи. Это и есть строгая форма принципа наименьшего времени в терминах действия.

Для компактных объектов критерий наблюдаемости:

```
1+z = n_emit/n_obs < ∞,   Δt = ∫ n dl < ∞.
```

## 3) Эмерджентные тензоры из единого закона распространения

Дальше работает общая EFT‑логика. Если вся материя и Ψ взаимодействуют через одну ĝ(θ):

```
S_total = S_{ΘΨ}[θ, Ψ] + Σ_i S_matter[Φ_i; ĝ(θ)],
```

то квантовый эффективный функционал при низких энергиях неизбежно содержит геометрические инварианты от ĝ (механизм Сахарова):

```
Γ_eff[ĝ] ⊃ ∫ d^4x √−ĝ [ Λ_ind^4 + (M_ind^2/2) R(ĝ) + O(∂^4) ].
```

Аргумент минимальности: единственный релевантный (по производным) тензорный
инвариант второго порядка — скаляр кривизны `R(ĝ)`. Его коэффициент `M_ind^2`
определяется интегрированием быстрых мод материи/Ψ. Это не добавляет новую
онтологию: `ĝ` остается сжатой записью того, что все процессы используют один
и тот же clock-sector response.

Из линейной вариации EH‑члена по TT‑части `δĝ_{ij}^{TT}` следует стандартное уравнение гравитационных волн на FRW‑фоне (в конформном времени):

```
h''_{ij} + 2 ℋ h'_{ij} + k^2 h_{ij} = 0,
```

то есть две поперечные тензорные поляризации с `c_T = 1`.

Итог: «наблюдаемая тензорность» — это TT‑возмущения составной метрики ĝ(θ), порождённой все тем же `n(θ_eff)`. Θ остаётся скаляром‑часами; тензорным является не Θ, а коллективный объект ĝ.

### Слабопольный matching наблюдаемых (конформальная форма)

Пусть в статике и изотропии конформальный коэффициент `A(θ_eff) = n(θ_eff)^2` представим как

```
A(θ_eff) = e^{2Φ(x)} = 1 + 2Φ + 2Φ^2 + O(Φ^3),
```

где `Φ` — слабый потенциал. Тогда оптическая метрика

```
ĝ_{00} =  A = 1 + 2Φ + 2Φ^2 + ...,
ĝ_{ij} = −A δ_{ij} = −(1 + 2Φ) δ_{ij} + O(Φ^2).
```

Сопоставляя с наблюдательной слабопольной параметризацией в изотропной калибровке,

```
g_{00} = 1 + 2Φ + 2β Φ^2 + ...,
g_{ij} = −(1 − 2γ Φ) δ_{ij} + ...,
```

получаем требуемые weak-field коэффициенты `γ = 1`, `β = 1`. В Θ–Ψ это
читается не как онтологическая привязка к геометрии, а как санитарное условие:
один и тот же `n(θ_eff)` должен давать правильные коэффициенты красного
смещения, задержки, линзирования и орбитальной динамики.

## 4) Совместимость с базой `Θ_{μν}, Ψ`

- Однородный предел: minisuperspace‑лагранжиан (см. `background/frw_symbolic.py`) описывает нулевые моды Θ и амплитуды Ψ на фоне FRW; при этом ведущая тензорная динамика исходит из индуцированного EH‑члена для ĝ и даёт `c_T=1` (см. docs/07_tensor_optical_ppn.md и тесты).
- Скалярные возмущения: текущие матрицы K,G,M минимального двухсущностного
  reduced-контура — это сектор скалярных фононов той же среды. Условия
  гиперболичности и субсветовости (K≻0, eig(K⁻¹G)≤1, ω²>0) уже автоматизированы
  в `checks/`.

## 5) Что проверять и где риски

- PPN/диполь: вывести γ_PPN, β_PPN и отсутствие дипольного излучения как
  следствие универсального `n` и единой `ĝ`.
- c_T и поляризации: `c_T=1` (уже зафиксировано), отсутствие доминирующих скалярных/векторных примесей в ГВ.
- Сверхсвет/призраки: контролируются условиями на K,G,M (у нас — `checks/health.py`).
- Λ_ind и УФ‑срез: феноменологический контроль (EFT‑режим `|θ̇/θ|, |γ θ̇| < Λ`).

## 6) Дорожная карта к «камню»

1) Формально связать текущий `n(θ_eff)` с явной ĝ(θ) и выписать TT‑уравнение на FRW (короткий расчёт, c_T=1).
2) Довести PPN‑модуль до формул (γ_PPN, β_PPN), добавить тесты.
3) Развить ADM‑ветку: скалярный сектор на фоне ĝ(θ), проверки ω²(k)>0 и запрета сверхсвета при исключении lapse/shift.

Эта записка фиксирует фундамент: «один `n(θ_eff)` → одна ĝ(θ) → тензоры как эмерджентные TT‑моды с c_T=1». Все классические эффекты и наблюдаемая тензорная гравитация следуют из одного и того же объекта — индекса времени вакуума.

## 3) PPN как санитарный weak-field тест: γ_PPN = β_PPN = 1

Цель: показать, что при универсальной связи материи через один и тот же
индекс времени `n(θ_eff)` weak-field наблюдаемые не ломаются: параметры ППН
в слабом, статическом поле принимают значения `γ=1`, `β=1`, а дипольное
излучение отсутствует. Это проверка совместимости с измерениями, а не
онтологическое основание гипотезы.

## 1) Универсальная эффективная метрика

В слабом поле, статике и изотропии удобно взять универсальную конформальную
форму (согласованную с принципом эквивалентности):

```
\tilde g_{\mu\nu}(x) = A(θ_eff(x))\, \eta_{\mu\nu},   \qquad  A(θ_eff)=n(θ_eff)^2,
```

где `η_{μν}` — Минковский, а `A(θ_eff)>0` кодирует один и тот же `n(θ_eff)` для всех
полей материи. Такая форма:
- фиксирует растяжение собственного времени `d\tau^2 = dt^2/n^2`;
- даёт совпадающие коэффициенты при возмущениях `\tilde g_{00}` и `\tilde g_{ij}`
  на первом порядке, что приводит к `γ=1`.
- в полной (не только PN) постановке используется ограниченный индекс
  `1 ≤ n(θ_eff) < ∞`, что исключает истинный горизонт и оставляет лишь
  конечный экстремальный redshift у сверхкомпактных объектов.

Примечание. В Θ–Ψ первичной является не метрика, а единый индекс времени
`n(θ_eff)`. Метрическая запись нужна только для компактного учета того, что
свет, часы и массивная материя используют один и тот же локальный закон
распространения.

## 2) Линеаризация и сопоставление с PPN

Пишем `A(θ_eff) = 1 + 2\Phi(x) + O(\Phi^2)`, где `\Phi` — слабый потенциал
(пропорционален `\ln n`). Тогда

```
\tilde g_{00} = 1 + 2\Phi + O(\Phi^2),
\quad
\tilde g_{ij} = -\bigl(1 + 2\Phi\bigr)\delta_{ij} + O(\Phi^2).
```

Наблюдательная PPN‑запись в изотропной калибровке:

```
g_{00} = 1 + 2\Phi + 2\beta\,\Phi^2 + O(\Phi^3),
\quad
g_{ij} = -\bigl(1 - 2\gamma\,\Phi\bigr)\delta_{ij} + O(\Phi^2).
```

Сопоставляя первый порядок, получаем `\gamma = 1`.
На втором порядке, при отсутствии дополнительных нелокальных/нелинейных
источников в `A(θ_eff)`, справедливо `\beta = 1` (конформальная форма одинаково
масштабирует `g_{00}` и `g_{ij}` и не вводит особых квадратичных поправок).

Итого: `\gamma_{PPN} = 1`, `\beta_{PPN} = 1`.

## 3) Отсутствие дипольного излучения

Скалярная «зарядка» материи одинакова (универсальная связь), поэтому
нет разделения тел по «скалярному заряду» — дипольный момент системы
не излучает на ведущем порядке. Радиционные потери начинаются с квадрупольного
канала, что совместимо с пульсарными тестами.

## 4) Связь с n(θ_eff) и классическими тестами

Из раздела 01_interpretation.md: классические эффекты слабого поля
следуют из одного `n(θ_eff)` через `\Delta\nu/\nu \approx \Delta\ln n`,
`t = \int n(θ_eff)\,dl`, `\alpha \approx \int \nabla_\perp \ln n\, dl`.
Выбор нормировки `\ln n \approx -2\Phi` согласует численные коэффициенты
красного смещения, задержки Шапиро и линзирования с наблюдаемыми и совместим
с `\gamma=1`.

## 5) Ограничения и условия применимости

- Статика/слабое поле/изотропия: `|\Phi| \ll 1`, отсутствуют сильные
  анизотропные градиенты `θ`.
- Универсальность связи: вся материя использует один и тот же `n(θ_eff)` и
  одну и ту же effective propagation запись `\tilde g(θ)`; иначе принцип
  эквивалентности нарушается и PPN‑параметры уходят от наблюдаемых значений.
- Вне слабого поля возможны конформальные/дизформальные поправки высокого
  порядка — они должны сохранять локальные тесты (ограничения на параметры).
- Безгоризонтный сильнопольный режим формулируется как `n(θ_eff)` конечен сверху;
  тогда `1+z` и `∫ n dl` конечны.

Таким образом, при универсальном `n(θ_eff)` и единой effective propagation
записи `\tilde g = n(θ_eff)^2 \eta` слабопольные проверки проходят:
`γ=β=1`, скалярного диполя нет.
