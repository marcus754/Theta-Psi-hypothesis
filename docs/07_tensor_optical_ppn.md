# 07. Тензорный сектор, оптическая метрика и PPN

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор Theta_mu_nu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная theta ≡ theta_eff, где theta_eff^2 = Theta_mu_nu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную тензорную динамику Theta_mu_nu.

## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются Theta_mu_nu и Psi; редуцированный контур не требует внешнего порога и не использует ручное отсечение веток.

S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + M_*^4 F(J_refr) ],
J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi).

Источник режима (без Weyl-порога в каноне):
I_grad(Theta,Psi) = l_theta^2 · |∇_⊥theta|^2 + l_Psi^2 · |∇_⊥Psi|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^mu).

Смысл:
- В FRW (однородный фон): I_grad=0, поэтому J_refr=0.
- В сильное поле (большие пространственные градиенты): I_grad>0, поэтому преломляющий сектор включается напрямую через F(J_refr).
- Отдельного режимного поля в каноне нет.

Индекс вакуума ограничен конструктивно:
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).

Невалидная ветка фона исключена параметризацией (а не фильтром):
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).

В наблюдательных проверках оценивается не «ручное включение», а прямой ответ на пространственные неоднородности Theta_mu_nu, Psi.
Раздел фиксирует связку: c_T = 1, универсальная оптическая метрика n(theta_eff) и слабопольный PPN-предел gamma=beta=1.

## 1) Скорость тензорных волн (c_T)

Ключевой вопрос: не изменяет ли модель скорость гравитационных волн (c_T), сильно ограниченную наблюдениями LIGO/Virgo (|c_T/c − 1| ≲ 10⁻¹⁵)?

В текущей версии гипотезы на базе Theta_mu_nu, Psi гравитационный сектор не модифицирован; поля (theta, Psi) относятся к «материи»
(источнике для фона и скалярных возмущений). В эффективной параметризации модифицированной гравитации это соответствует α_T = 0, что строго даёт c_T = 1 для тензорного сектора на FRW-фоне.

Практический вывод:
- В коде (theory/tensor.py) заданы свойства тензорного сектора с c_T^2 = 1.
- Тест (tests/test_tensor_speed.py) проверяет, что возвращается c_T^2 = 1.
- Если в будущем будут добавляться операторы, меняющие кинетику гравитации, они должны быть явно включены и немедленно проверяться на совместимость с ограничениями LIGO/Virgo, т.е. α_T ≈ 0 в поздние эпохи.

## 2) Оптическая метрика из n(theta_eff) и эмерджентные тензорные волны

Цель: зафиксировать «из первых принципов» связку
- один индекс времени вакуума n(theta_eff) (преломление), принцип наименьшего времени;
- единая оптическая метрика для всей материи;
- возникновение тензорного сектора (две TT‑поляризации, c_T = 1) как универсального низкоэнергетического ответа среды.

### 2.1) Один n(theta_eff) и принцип Ферма

Постулат: локальный ход собственного времени у материи и света растянут одним и тем же индексом
dtau = dt / n(theta_eff).

Физическая постановка берёт ограниченный индекс:
1 ≤ n(theta_eff) < ∞.

Отсюда в сильном поле получаем не «истинный горизонт», а безгоризонтный предел с очень большим, но конечным красным смещением.
Лучи и траектории минимизируют оптическую длину (принцип Ферма)

L_opt[Γ] = ∫_Γ n(theta_eff) dl,

что уже даёт три классических эффекта слабого поля из одного n(theta_eff): красное смещение, задержку Шапиро, линзирование (см. 01_interpretation.md).

### 2.2) Оптическая метрика g_hat(n, u)

Вариационный принцип удобно записать ковариантно через «оптическую метрику»
типа Гордона. Пусть
- u_mu ≡ ∂_mu theta / √(∂theta·∂theta) — выделенное таймлайк‑направление («часы»);
- базовая фонова метрика в локально инерциальных координатах — eta.

Тогда эффективная метрика, по геодезикам которой идут свет и материя, может быть записана как
g_hat_mu_nu(theta) = eta_mu_nu + (n(theta_eff)^2 − 1) u_mu u_nu

(эквивалентные записи встречаются и с коэффициентом при u_mu u_nu, зависящим от выбранной нормировки; существенен сам факт существования единой g_hat, кодирующей n(theta_eff)). В локально комовинговом кадре u^mu = (1,0,0,0) и g_hat_00 = n^2, так что dtau^2 = dt^2 / n^2.

Следствие: «преломление вакуума» сводится к геодезикам одной и той же g_hat(theta) для всех полей материи. Это и есть строгая форма принципа наименьшего времени в терминах действия.

Для компактных объектов критерий наблюдаемости:
1+z = n_emit/n_obs < ∞,   Δt = ∫ n dl < ∞.

### 2.3) Эмерджентные тензоры из единого закона распространения

Дальше работает общая EFT‑логика. Если вся материя и Psi взаимодействуют через одну g_hat(theta):
S_total = S_ThetaPsi[theta, Psi] + sum_i S_matter[Phi_i; g_hat(theta)],

то квантовый эффективный функционал при низких энергиях неизбежно содержит геометрические инварианты от g_hat (механизм Сахарова):
Γ_eff[g_hat] ⊃ ∫ d^4x √−g_hat [ Λ_ind^4 + (M_ind^2/2) R(g_hat) + O(∂^4) ].

Аргумент минимальности: единственный релевантный (по производным) тензорный инвариант второго порядка — скаляр кривизны R(g_hat). Его коэффициент M_ind^2 определяется интегрированием быстрых мод материи/Psi. Это не добавляет новую онтологию: g_hat остается сжатой записью того, что все процессы используют один и тот же clock-sector response.

Из линейной вариации EH‑члена по TT‑части deltag_hat_ij^TT следует стандартное уравнение гравитационных волн на FRW‑фоне (в конформном времени):
h''_ij + 2 H_conf h'_ij + k^2 h_ij = 0,

то есть две поперечные тензорные поляризации с c_T = 1.

Итог: «наблюдаемая тензорность» — это TT‑возмущения составной метрики g_hat(theta), порождённой все тем же n(theta_eff). Theta остаётся скаляром‑часами; тензорным является не Theta, а коллективный объект g_hat.

### Слабопольный согласование наблюдаемых (конформальная форма)

Пусть в статике и изотропии конформальный коэффициент A(theta_eff) = n(theta_eff)^2 представим как
A(theta_eff) = e^2Phi(x) = 1 + 2Phi + 2Phi^2 + O(Phi^3),

где Phi — слабый потенциал. Тогда оптическая метрика

g_hat_00 =  A = 1 + 2Phi + 2Phi^2 + ...,
g_hat_ij = -A delta_ij = -(1 - 2Phi) delta_ij + O(Phi^2).

Сопоставляя с наблюдательной слабопольной параметризацией в изотропной калибровке,
g_00 = 1 + 2Phi + 2beta Phi^2 + ...,
g_ij = −(1 − 2gamma Phi) delta_ij + ...,

получаем требуемые слабое поле коэффициенты gamma = 1, beta = 1. В Theta–Psi это читается не как онтологическая привязка к геометрии, а как санитарное условие: один и тот же n(theta_eff) должен давать правильные коэффициенты красного смещения, задержки, линзирования и орбитальной динамики.

### 2.4) Совместимость с базой Theta_mu_nu, Psi

- Однородный предел: минисуперпространство‑лагранжиан (см. background/frw_symbolic.py) описывает нулевые моды Theta и амплитуды Psi на фоне FRW; при этом ведущая тензорная динамика исходит из индуцированного EH‑члена для g_hat и даёт c_T=1 (см. docs/07_tensor_optical_ppn.md и тесты).
- Скалярные возмущения: текущие матрицы K,G,M минимального двухсущностного reduced-контура — это сектор скалярных фононов той же среды. Условия гиперболичности и субсветовости (K≻0, eig(K⁻¹G)≤1, ω²>0) уже автоматизированы в checks/.

### 2.5) Что проверять и где риски

- PPN/диполь: вывести gamma_PPN, beta_PPN и отсутствие дипольного излучения как следствие универсального n и единой g_hat.
- c_T и поляризации: c_T=1 (уже зафиксировано), отсутствие доминирующих скалярных/векторных примесей в ГВ.
- Сверхсвет/призраки: контролируются условиями на K,G,M (у нас — checks/health.py).
- Λ_ind и УФ‑срез: феноменологический контроль (EFT‑режим |thetȧ/theta|, |gamma thetȧ| < Λ).

### 2.6) Дорожная карта к «камню»

1) Формально связать текущий n(theta_eff) с явной g_hat(theta) и выписать TT‑уравнение на FRW (короткий расчёт, c_T=1).
2) Довести PPN‑модуль до формул (gamma_PPN, beta_PPN), добавить тесты.
3) Развить ADM‑ветку: скалярный сектор на фоне g_hat(theta), проверки ω²(k)>0 и запрета сверхсвета при исключении Лапса/Шифта.

Эта записка фиксирует фундамент: «один n(theta_eff) → одна g_hat(theta) → тензоры как эмерджентные TT‑моды с c_T=1». Все классические эффекты и наблюдаемая тензорная гравитация следуют из одного и того же объекта — индекса времени вакуума.

## 3) PPN: санитарный слабое поле тест gamma_PPN = beta_PPN = 1

Цель: показать, что при универсальной связи материи через один и тот же индекс времени n(theta_eff) слабое поле наблюдаемые не ломаются: параметры ППН в слабом, статическом поле принимают значения gamma=1, beta=1, а дипольное излучение отсутствует. Это проверка совместимости с измерениями, а не онтологическое основание гипотезы.

### 3.1) Универсальная эффективная метрика

В слабом поле, статике и изотропии удобно взять универсальную конформальную форму (согласованную с принципом эквивалентности):
g_munu(x) = A(theta_eff(x)) eta_munu,     A(theta_eff)=n(theta_eff)^2,

где eta_mu_nu — Минковский, а A(theta_eff)>0 кодирует один и тот же n(theta_eff) для всех
полей материи. Такая форма:
- фиксирует растяжение собственного времени dtau^2 = dt^2/n^2;
- даёт совпадающие коэффициенты при возмущениях g_00 и g_ij на первом порядке, что приводит к gamma=1.
- в полной (не только PN) постановке используется ограниченный индекс 1 ≤ n(theta_eff) < ∞, что исключает истинный горизонт и оставляет лишь конечный экстремальный redshift у сверхкомпактных объектов.

Примечание. В Theta–Psi первичной является не метрика, а единый индекс времени n(theta_eff). Метрическая запись нужна только для компактного учета того, что свет, часы и массивная материя используют один и тот же локальный закон распространения.

### 3.2) Линеаризация и сопоставление с PPN

Пишем A(theta_eff) = 1 + 2Phi(x) + O(Phi^2), где Phi — слабый потенциал (пропорционален ln n). Тогда

g_00 = 1 + 2Phi + O(Phi^2),
g_ij = -(1 - 2Phi)delta_ij + O(Phi^2).

Наблюдательная PPN‑запись в изотропной калибровке:
g_00 = 1 + 2Phi + 2betaPhi^2 + O(Phi^3),

g_ij = -(1 - 2gammaPhi)delta_ij + O(Phi^2).

Сопоставляя первый порядок, получаем gamma = 1.
На втором порядке, при отсутствии дополнительных нелокальных/нелинейных источников в A(theta_eff), справедливо beta = 1 (конформальная форма одинаково масштабирует g_00 и g_ij и не вводит особых квадратичных поправок).

Итого: gamma_PPN = 1, beta_PPN = 1.

### 3.3) Отсутствие дипольного излучения

Скалярная «зарядка» материи одинакова (универсальная связь), поэтому нет разделения тел по «скалярному заряду» — дипольный момент системы не излучает на ведущем порядке. Радиционные потери начинаются с квадрупольного канала, что совместимо с пульсарными тестами.

### 3.4) Связь с n(theta_eff) и классическими тестами

Из раздела 01_interpretation.md: классические эффекты слабого поля следуют из одного n(theta_eff) через Deltanu/nu approx Deltaln n,
t = int n(theta_eff)dl, alpha approx int nabla_perp ln n dl. Выбор нормировки ln n approx -2Phi согласует численные коэффициенты
красного смещения, задержки Шапиро и линзирования с наблюдаемыми и совместим с gamma=1.

### 3.5) Ограничения и условия применимости

- Статика/слабое поле/изотропия: |Phi| ll 1, отсутствуют сильные анизотропные градиенты theta.
- Универсальность связи: вся материя использует один и тот же n(theta_eff) и одну и ту же effective propagation запись g(theta); иначе принцип эквивалентности нарушается и PPN‑параметры уходят от наблюдаемых значений.
- Вне слабого поля возможны конформальные/дизформальные поправки высокого порядка — они должны сохранять локальные тесты (ограничения на параметры).
- Безгоризонтный сильнопольный режим формулируется как n(theta_eff) конечен сверху; тогда 1+z и ∫ n dl конечны.

Таким образом, при универсальном n(theta_eff) и единой effective propagation записи g = n(theta_eff)^2 eta слабопольные проверки проходят: gamma=beta=1, скалярного диполя нет.
