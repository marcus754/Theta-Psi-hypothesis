# 11. Линия вывода Theta–Psi: тензорное время как канон

Цель: зафиксировать без двусмысленности, что в канонической версии фундаментальным объектом времени является тензор Theta_munu, а скаляр theta — только его редуцированная проекция для текущих вычислений FRW.

## 1) Постулаты канона

1. В качестве фундаментального объекта времени принимается Theta_munu (симметричный тензор 2-го ранга).
2. Энергетический сектор моделируется комплексным полем Psi.
3. Предполагается, что наблюдаемая динамика материи задается эффективным откликом вакуума, который
   определяется инвариантами Theta_munu.
4. В слабое поле пределе рабочая версия теории должна воспроизводить проверенные наблюдаемые тесты.
5. Поскольку время в гипотезе понимается как физический объект, допустимая динамика не должна
   переводить временной режим в нуль или отрицательную ветвь.

## 2) Базовое ковариантное действие (структура)

S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_Theta(Theta_munu, g) + L_Psi(Psi, g) + L_int(Theta,Psi,g) ].


### 2.1) Единое редуцированное действие для FRW и сильного поля

Фиксируем единое редуцированное действие, которое воспроизводит оба рабочих предела:
S_red = ∫ d^4x √(-g) [
  (M_pl²/2)R
  + L_Theta
  + L_R
  - U_red(theta_eff,R)
  + M_*^4 F(J_refr)
].


В этом каноне:
L_Theta = 3/2 · ∇_mu ln theta_eff ∇^mu ln theta_eff + 1/2 · gamma² ∇_mu theta_eff ∇^mu theta_eff
L_R     = 1/2 · ∇_mu R ∇^mu R


С минимальным потенциалом:
U_red(theta_eff,R) = 1/2 m_theta² theta_eff² + λ_theta/4 theta_eff⁴ + 1/2 V0 R² + λ_ThetaPsi/2 theta_eff² R².


Смысл этой записи:
1. На FRW-фоне ∇_i theta_eff = ∇_i Psi = 0, поэтому J_refr = 0 и остаётся
   однородный лагранжиан.
2. В стационарном неоднородном режиме градиенты времени/энергии включают
   F(J_refr), давая ответ в сильном поле из того же действия.
3. Это единое редуцированное ковариантное действие, а не два независимых лагранжиана.

### 2.2) Текущий реестр ковариантного действия

В репозитории этот переход теперь поддержан не только текстом, но и отдельным
символическим модулем реестра:
- theory/covariant_action.py

Его роль:
1. зафиксировать theta_eff² = Theta_munuu^muu^nu;
2. хранить каноническое Phi_eff^can = ln(theta_eff/theta_0);
3. держать один символьный источник для S_red, стационарный J_refr и редуцированные
   блоки энергии-импульса;
4. позволить проверять, что FRW и стационарное сильное поле действительно читаются
   как два предела одного редуцированного действия, а не как две независимые подстановки.

Это канонический реестр действия: он существует, тестируется и задаёт рабочий
канон S_red.

### 2.3) Уравнения движения и редуцированный T_munu

Из S_red для однородного FRW-среза фиксируем единый реестр минисуперпространства:
E_theta = (3/theta² + gamma²) thetä - 3 thetȧ² / theta³ = 0,
E_Psi = Psï + V0 Psi = 0.


Соответствующие плотность энергии и давление:
ρ_red = 3/2 · (thetȧ/theta)² + 1/2 · Ṙ² + 1/2 · V0 R² + 1/2 · gamma² thetȧ²,
p_red = 3/2 · (thetȧ/theta)² + 1/2 · Ṙ² - 1/2 · V0 R² + 1/2 · gamma² thetȧ².


Следовательно:
T^(red)_00 = ρ_red,
T^(red)_ii = a² p_red  (без суммирования по i).


В редуцированном каноне это и есть минимальный замкнутый набор: действие,
вариация по полям и сводка по T_munu в одном источнике.

### 2.4) Полное ковариантное замыкание

Полный ковариантный объект канона фиксируется как

S_cov = S_red + S_m[g_hat, matter],
g_hat = g + (n²(theta_eff,Theta,Psi) - 1) u ⊗ u.


Здесь g_hat — не новая онтологическая геометрия, а компактная запись
рефракционного и часового отклика вакуума. S_m[g_hat, matter] означает,
что материя и излучение получают один и тот же закон распространения через
этот отклик. Это и есть полное ковариантное замыкание для текущего канона:
редуцированное ядро плюс единое правило связи материи.

В этой записи g и g_hat имеют статус языка учета, а не первичных объектов
гипотезы. Первичными остаются Theta_munu и Psi. Метрическая форма нужна только
для того, чтобы записать локальный закон распространения процессов без
разных правил для света, часов и массивной материи.

Вариация по метрике:
G_munu = 8πG · (T^(m)_munu + T^(Theta)_munu + T^(Psi)_munu + T^(int)_munu).


Критически важно: наблюдаемые тензорные возмущения должны быть коллективными
возмущениями эффективного закона распространения, а не следствием одной скалярной
редукции theta_eff.

## 3) Тензорный лагранжиан L_Theta: минимальная форма

### 3.1) Требования к L_Theta

1. **Диффеоморфизм-инвариантность**: скаляр относительно координатных преобразований.
2. **Минимальное число производных**: не более 2-го порядка для гиперболичности.
3. **Положительная энергия**: отсутствие призраков (ghost-free).
4. **Корректный инертный предел**: при Theta_munu → const эффективный отклик вакуума
   перестаёт вносить дополнительную неоднородность.

### 3.2) Минимальный лагранжиан на пространстве тензоров

Наиболее общая форма без призраков для симметричного тензора 2-го ранга:
L_Theta = 1/2 · G^munuρσ(Theta) · ∇_mu Theta_nuρ · ∇_σ Theta_αbeta · g^αbeta - U(Theta_munuTheta^munu),


где G^munuρσ — метрика на пространстве конфигураций Theta.

Для изотропного FRW-фона с выделенным временем u^mu = (1,0,0,0) в комовинговом кадре:
Theta_munu = diag(theta²(t), 0, 0, 0)  (в синхронной калибровке),


тогда единственный нетривиальный инвариант:
I_1 = Theta_munu u^mu u^nu = theta²(t).


### 3.3) Редукция к скаляру theta_eff

Определяем:
theta_eff ≡ √(Theta_munu u^mu u^nu),   u^mu u_mu = -1.


В FRW метрике ds² = -dt² + a²(t) dx²:
u^mu = (1, 0, 0, 0),   u_mu = (-1, 0, 0, 0),
Theta_00 = theta²(t),   Theta_ij = 0,
theta_eff = √(Theta_00 u^0 u^0) = √(theta² · 1 · 1) = theta(t).


### 3.4) Вывод кинетического члена

Из тензорного лагранжиана:
L_Theta ⊃ 1/2 · G^0000(Theta) · ∂_0 Theta_00 · ∂_0 Theta_00.


Для Theta_00 = theta²:
∂_0 Theta_00 = 2theta thetȧ,
L_Theta ⊃ 1/2 · G^0000 · (2theta thetȧ)² = 2 G^0000 · theta² thetȧ².


Требование канонической нормировки (см. 08_foundations.md):
L_eff ⊃ 3/2 · (thetȧ/theta)² = 3/2 · theta^-2 · thetȧ².


Сравнивая:
2 G^0000 · theta² = 3/2 · theta^-2  ⇒  G^0000 = 3/4 · theta^-4.


**Вывод:** метрика конфигурационного пространства на Theta:
G^munuρσ(Theta) = 3/4 · (Theta_αbeta u^α u^beta)^-2 · P^munuρσ,


где P^munuρσ — проектор на временную компоненту.

### 3.5) Член смешения gamma² thetȧ²

Этот член возникает из взаимодействия Theta и Psi:
L_int ⊃ gamma²/2 · (∇_mu Theta_nuρ) (∇^mu Psi) (∇^nu Psi) (∇^ρ Psi) / (Psī Psi).


В FRW пределе при Psi = Psi(t):
L_int ⊃ gamma²/2 · thetȧ² · Psi̇² / |Psi|².


Для однородного фона Psi̇²/|Psi|² ≈ 1 (нормировка поля), остаётся:
L_int ⊃ gamma²/2 · thetȧ².

Полагаем, что на однородном фоне поле Psi нормировано так, что Psi̇²/|Psi|² = 1. При этом условии из L_int следует gamma²/2 · thetȧ².

**Физический смысл:** gamma — безразмерная связь между градиентом времени и градиентом энергии.

## 4) Итоговый эффективный лагранжиан

После редукции Theta_munu → theta_eff:
L_eff = 3/2 · (thetȧ/theta)² + 1/2 · Ṙ² - 1/2 · V0 · R² + 1/2 · gamma² · thetȧ².


### 4.1) Проверка пределов

| Предел | L_eff | Физика |
|--------|-------|--------|
| gamma → 0 | 3/2 (thetȧ/theta)² + 1/2 Ṙ² - 1/2 V0 R² | Развязанные Theta и Psi |
| theta → const | 1/2 Ṙ² - 1/2 V0 R² | инертный предел Psi-сектора |
| Psi → 0 | 3/2 (thetȧ/theta)² + 1/2 gamma² thetȧ² | Чистое Theta-поле (stiff matter) |
| thetȧ → 0 | 1/2 Ṙ² - 1/2 V0 R² | статический временной режим |

### 4.2) Энергия и давление

ρ = (∂L/∂thetȧ)thetȧ + (∂L/∂Psi̇)Psi̇ - L
  = 3/2 (thetȧ/theta)² + 1/2 Ṙ² + 1/2 V0 R² + 1/2 gamma² thetȧ²,

p = L = 3/2 (thetȧ/theta)² + 1/2 Ṙ² - 1/2 V0 R² + 1/2 gamma² thetȧ².


Уравнение состояния:
w = p/ρ = [3/2 (thetȧ/theta)² + 1/2 Ṙ² - 1/2 V0 R² + 1/2 gamma² thetȧ²] / [3/2 (thetȧ/theta)² + 1/2 Ṙ² + 1/2 V0 R² + 1/2 gamma² thetȧ²].


Пределы:
- Кинетическое доминирование: w → +1 (stiff matter)
- Потенциальное доминирование: w → -1 (dark energy)
- thetȧ = 0, Psi̇ = 0: w = -1 (космологическая постоянная)

## 5) Оптическая метрика из тензора Theta_munu

### 5.1) Эффективная метрика для материи

Материя движется по геодезикам эффективной метрики:
g_hat_munu = g_munu + (n²(theta_eff) - 1) · u_mu u_nu,


где u_mu = ∂_mu theta_eff / √(∂theta·∂theta) — выделенное временное направление.

В комовинговом кадре u^mu = (1,0,0,0):
g_hat_00 = n²(theta_eff),   g_hat_ij = -delta_ij.


### 5.1.1) Минимальный рабочий класс L_refr(Theta,Psi)

Связка

Theta, Psi  ->  J_refr(Theta,Psi)  ->  n(theta_eff,Theta,Psi)  ->  g_hat_munu


фиксирует минимальный рабочий класс преломляющего сектора. Его задача
заключается в том, чтобы задать, как из временного объекта и энергетического
субстрата возникает универсальный рефрактивный отклик вакуума.

Требования к L_refr:
1. Он зависит от Theta и Psi только через инварианты, совместимые с каноном.
2. Он универсален для материи и излучения.
3. На FRW-фоне его вклад выключается автоматически при I_grad = 0.
4. В слабое поле пределе он даёт линейное соответствие ln n  κ Phi_eff.
5. В сильное поле режиме он насыщается и не уводит n в патологию.

Минимальная запись:
L_refr = M_*^4 · F(J_refr),
J_refr = Phi_eff(theta_eff) · I_grad(Theta,Psi),
Phi_eff(theta_eff) = ln(theta_eff / theta_0).


Здесь:
- theta_0 — слабое поле опорное значение временного режима;
- Phi_eff — не ньютоновский потенциал сам по себе, а канонический scalar bridge,
  который в слабом поле совпадает с ньютоновским после калибровки;
- I_grad — прямой инвариант пространственной неоднородности системы Theta–Psi.

Тогда эффективный индекс не вводится отдельно "сверху", а определяется как
универсальный отклик вакуума на J_refr:
n - 1 = 𝒩(J_refr),


где 𝒩(0)=0, 𝒩'(0)=κ, 𝒩(J)>=0, 𝒩(J) монотонна и имеет конечное
насыщение.

Эквивалентная рабочая форма:
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).


Эта форма фиксирует сразу три свойства:
1. при I_grad=0 получаем J_refr=0 и n=1, то есть FRW не включает
   рефрактивный сектор;
2. при малом J_refr имеем слабое поле закон n ≈ 1 + κ J_refr;
3. при большом J_refr отклик насыщается, а численная итерация не выходит в
   неустойчивую область.

### 5.1.1a) Статус Phi_eff: канон против численной регуляризации

Чтобы убрать двусмысленность:
1. **Каноническое определение**:
Phi_eff^can(theta_eff) = ln(theta_eff / theta_0),   theta_eff > 0.


2. **Численная регуляризация в решатель'ах**:
Phi_eff^num(theta_eff) = гладкая положительная аппроксимация(theta_eff/theta_0),


которая совпадает с канонической формой в слабое поле согласование и используется
только там, где нужна гладкая положительная функция без логарифмической
жёсткости у машинного theta_eff  0.

Следствие: канон гипотезы не зависит от конкретной гладкой аппроксимации при
выполнении трёх условий:
- Phi_eff^num >= 0,
- Phi_eff^num -> 0 при theta_eff -> theta_0,
- в рабочем слабом поле Phi_eff^num = Phi_eff^can + O((theta_eff-theta_0)^2).

### 5.1.2) Как L_refr входит в действие материи

L_refr перестраивает эффективный локальный clock sector вакуума. Рабочая
запись:
S_m  ->  S_m[g_hat_munu, matter],
g_hat_munu = g_munu + (n²(theta_eff,Theta,Psi)-1) u_mu u_nu.


### 5.1.2a) Тензор энергии-импульса reduced сектора

Для reduced action определяем:
T_munu^(red) = -2 / √(-g) · delta(√(-g)L_red) / deltag^munu
               = T_munu^(theta) + T_munu^(Psi) + T_munu^(refr) - g_munuU_red.


Минимальные куски:
T_munu^(theta) =
  3 · ∇_mu ln theta_eff ∇_nu ln theta_eff
  + gamma² ∇_mu theta_eff ∇_nu theta_eff
  - g_munu[ 3/2 (∇lntheta_eff)^2 + 1/2 gamma² (∇theta_eff)^2 ],

T_munu^(Psi) =
  ∇_mu Psi ∇_nu Psi
  - g_munu[ 1/2 (∇Psi)^2 ],

T_munu^(refr) =
  -g_munu M_*^4 F(J_refr)
  + 2 M_*^4 F'(J_refr) · deltaJ_refr/deltag^munu.


На FRW-фоне:
I_grad = 0  =>  J_refr = 0  =>  T_munu^(refr) = 0,


и тензор энергии-импульса сводится к однородному Theta–Psi сектору. В stationary
режиме T_munu^(refr) играет роль добавочного stress-sector, который и
 создаёт конечный optical response без введения отдельного поля-переключателя.

### 5.1.2b) Линейная устойчивость reduced сектора

Минимальные критерии до полного ADM-замыкания:
1. **FRW no-ghost**:
K_FRW = diag( 3/theta_eff² + gamma², 1 )  ≻ 0.


2. **FRW no-gradient-instability**:
G_FRW = diag( c_theta² K_theta, c_Psi² )  ≻ 0.


3. **Stationary ellipticity** для радиального two-field решатель:
M_11 > 0,  M_22 > 0,  det(M) > 0,


где M — локальная квазилинейная матрица из п. 5.1.3.

4. **No extra modes**:
no_extra_modes = no_ghost ∧ no_gradient_instability ∧ elliptic.


Это даёт простой и проверяемый критерий: рабочая область параметров должна
быть одновременно no-ghost на FRW, эллиптичной в stationary сильное поле
секторе и без признаков скрытой добавочной моды.

### 5.1.3) Минимальный стационарный сектор и уравнения Эйлера–Лагранжа

Чтобы сильное поле решатель был лагранжево замкнутым, фиксируем
минимальный стационарный радиальный функционал для редуцированного двухсущностного
сектора:
L_stat = 1/2 · I_grad - U(theta,Psi) + α · F(J_refr),
I_grad = theta'^2 + w_Psi Psi'^2,
J_refr = Phi_eff(theta) · I_grad,
Phi_eff(theta) = ln(theta/theta_0) или его гладкая положительная аппроксимация.


Здесь:
- α — сила рефрактивной обратной связи;
- w_Psi > 0 — относительный вес градиентного вклада Psi;
- штрих означает производную по стационарной радиальной координате r.

Минимальный рабочий стационарный потенциал, согласованный с текущим
вычислительным решатель:
U(theta,Psi) =
  1/2 m² theta² + λ/4 theta⁴
  - 1/2 m² Psi² + c_ThetaPsi λ/2 · theta² Psi².


Тогда:
U_,theta = m² theta + λ theta³ + c_ThetaPsi λ theta Psi²,
U_,Psi = -m² Psi + c_ThetaPsi λ theta² Psi.


Для сферически-симметричного стационарного профиля уравнения Эйлера–Лагранжа:
1/r² · d/dr [ r² A theta' ] + U_,theta - α F'(J_refr) Phi'_(eff) I_grad = 0,
1/r² · d/dr [ r² A w_Psi Psi' ] + U_,Psi = 0,


где

A(theta,theta',Psi') = 1 + 2 α Phi_eff(theta) F'(J_refr).


Поскольку A зависит от градиентов через J_refr, система является
квазилинейной. Её удобно переписать как локальную матричную систему для
вторых производных:
M_11 theta'' + M_12 Psi'' = R_theta,
M_21 theta'' + M_22 Psi'' = R_Psi,


с коэффициентами

A_,theta   = 2 α Phi'_(eff) [F' + Phi_eff I_grad F''],
A_,theta'  = 4 α Phi_eff² F'' theta',
A_,Psi'  = 4 α Phi_eff² F'' w_Psi Psi',

M_11 = A + A_,theta' theta',
M_12 = A_,Psi' theta',
M_21 = A_,theta' w_Psi Psi',
M_22 = w_Psi (A + A_,Psi' Psi').


Правая часть:
R_theta = -A_,theta theta'^2 - 2Atheta'/r - U_,theta + αF' Phi'_(eff) I_grad,
R_Psi = -A_,theta w_Psi theta'Psi' - 2Aw_PsiPsi'/r - U_,Psi.


Это и есть минимальная форма, в которой текущий two-field решатель уже можно
считать лагранжево мотивированным, а не просто ad hoc замыкание.

### 5.1.4) Слабое поле и FRW пределы стационарного сектора

Если I_grad -> 0, то:
J_refr -> 0,
F'(J_refr) -> F'(0),
A -> 1,


и радиальная система сводится к обычному двум слабо связанным уравнениям
скалярного профиля. На FRW-фоне пространственных градиентов нет, поэтому
стационарный рефрактивный блок автоматически выключен.


То есть L_refr задаёт универсальный скалярный отклик n, а уже через g_hat
этот отклик читается всеми секторами материи одинаково.

Это и есть содержательная причина, почему:
- свет линзируется,
- часы замедляются,
- массивные тела испытывают ускорение,

но при этом не вводится три разных механизма для трёх разных явлений.

### 5.1.3) FRW-предел и сильное поле включение

На однородном FRW-фоне:
I_grad = 0  =>  J_refr = 0.


Поэтому FRW-фон сам по себе не включает рефрактивный сектор.

В сильное поле или квазистационарном режиме:
I_grad > 0  =>  J_refr = Phi_eff I_grad > 0


и сильнопольный режим включается напрямую, без промежуточного поля режима.

Значит включение сильное поле режима происходит так:
1. градиенты Theta, Psi создают I_grad;
2. I_grad вместе с Phi_eff(theta_eff) создаёт J_refr;
3. J_refr включает конечный рефрактивный отклик n(theta_eff,Theta,Psi).

### 5.1.4) Как из этого получается насыщение n

Для малого J_refr:
F(J) = J + O(J²),


поэтому

n - 1 = 𝒩(J_refr) ≈ κ J_refr = κ Phi_eff I_grad.


То есть в слабом сильное поле режиме отклик растёт аналитически и контролируемо.

Для большого J_refr функция 𝒩(J) обязана насыщаться:
𝒩(J) -> 𝒩_∞ < ∞   при J -> +∞.


Следовательно, даже при росте I_grad временной отклик не уходит в
патологическую ветвь. Это и есть математический механизм того, как из прямого
взаимодействия Theta и Psi рождается конечный сильное поле режим без ручного
переключателя.

### 5.1.5) Канонический класс функций F(J)

Чтобы сектор режима был не произвольным, фиксируем минимальные допустимые
свойства функций ещё до фиттинга и сильное поле тестов.

Для F(J) канонически требуем:
F(0) = 0,
F'(J) > 0  для J >= 0,
F''(J) <= 0  для J >= 0,
F(J) -> F_∞ < ∞  при J -> +∞.


Смысл:
- F(0)=0 гарантирует отсутствие искусственного рефрактивного сдвига на фоне;
- F'(J)>0 гарантирует, что усиление режима не уменьшает отклик вакуума;
- F''(J)<=0 делает отклик насыщаемым, а не взрывным;
- конечный F_∞ исключает уход в бесконечную оптическую ветвь.

Канонический normalized family-v1:
F_v1(J) = 1 - exp(-J).


Любой явный масштаб J_* можно убрать перенормировкой J_refr до
безразмерной переменной; после этого он не является физическим параметром.

Чтобы сама экспонента стала выводом, нужен отдельный принцип для остатка
отклика:
S(J) = 1 - F(J),
S(J1 + J2) = S(J1) S(J2).


То есть независимые приращения J действуют без памяти и мультипликативно
на оставшуюся незадействованную часть отклика. При непрерывности такой закон
даёт

S(J) = exp(-J/J_*),
F(J) = 1 - exp(-J/J_*).


Именно этот шаг делает экспоненту следствием принципа полугруппы для
остатка отклика, а не отдельным выбором формы.

### 5.1.6) Почему именно этот класс считается каноническим

Для family-v1 получаем:
F_v1'(J) = exp(-J) > 0,
F_v1''(J) = -exp(-J) < 0,
F_v1(∞) = 1.


То есть все требования выше выполнены автоматически.

Кроме того, при малом J:
F_v1(J) = J + O(J²),


и слабое поле режим получается аналитическим и контролируемым. При большом J
отклик насыщается, что делает family-v1 минимальным гладким кандидатом без
изломов и без дополнительных порогов.

### 5.1.7) Связь family-v1 с рабочей формой для n

Если ввести численно регуляризованную family

𝒩(J) = saturating_response(J),


то family для n принимает вид

n(theta_eff,Theta,Psi) = 1 + 𝒩(J_refr),
J_refr = Phi_eff(theta_eff) I_grad(Theta,Psi).


Это делает структуру однородной:
- F(J) управляет тем, как рефрактивный сектор входит в effective action;
- 𝒩(J) управляет тем, как этот сектор читается как наблюдаемый индекс в
  решатель-реализации;
- обе функции принадлежат одному и тому же классу: монотонный, вогнутый,
  saturating response без порога и без смены ветви.

### 5.1.8) Что допускается как альтернатива, а что нет

Допускается как blind-test family, если зафиксировано заранее:
F_alt,1(J) = J / (1 + J),
F_alt,2(J) = tanh(J),


при тех же требованиях монотонности и насыщения.

Не допускается в каноне:
1. функции с F'(J) < 0 на части области;
2. функции с неограниченным ростом F(J);
3. кусочные пороги вида "если J>J_c, то включить режим";
4. переопределение family после просмотра данных.

### 5.1.9) Леммы канона

Ниже фиксируются минимальные утверждения, которые должны быть истинны для
любого допустимого F(J_refr). F_v1 в нормированной записи 1-exp(-J)
является каноническим representative-form; явный J_* — только нормировка.

#### Лемма A. Выключенность рефрактивного сектора на FRW

Предпосылки:
I_grad = 0,
F(0)=0.


Тогда

J_refr = 0,   n = 1.


Смысл: FRW не включает рефрактивный сектор.

#### Лемма B. Монотонность и конечность рефрактивного отклика

Предпосылки:
J >= 0,
𝒩(0)=0,
𝒩'(J) > 0,
𝒩''(J) <= 0,
lim_J->∞ 𝒩(J) < ∞.


Тогда

1 <= n(J) = 1 + 𝒩(J),
dn/dJ > 0.


Следовательно, усиление режима повышает индекс монотонно, но не позволяет ему
уйти в бесконечность на физической ветке решения; численная regularization
служит только для устойчивости решатель'а.

#### Лемма C. Слабое поле аналитичность

Если F(J) и 𝒩(J) аналитичны в нуле и

F(J) = J + O(J²),
𝒩(J) = κ J + O(J²),


то при малом I_grad и слабом Phi_eff:
n - 1 = κ Phi_eff I_grad + O((Phi_eff I_grad)²).


Следовательно, слабое поле отклик не содержит скрытого порога и включается
гладко.

#### Лемма D. Недостижимость истинного горизонта в редуцированном контуре

Предпосылки:
ρ(t) <= ρ_max < ∞,
L_eff ⊃ 3/2 (thetȧ/theta)²,
theta(t0) > 0.


Тогда

|d ln theta / dt| <= sqrt(2ρ_max/3),


и после интегрирования:
theta(t) >= theta(t0) exp(-K |t-t0|) > 0


для любого конечного t.

Следовательно, временной режим не достигает физического обнуления за конечный
ход, а в конечной optical branch это означает конечные redshift и delay.

#### Лемма E. Эквивалентность как следствие универсального g_hat_munu

Если материя входит только через

S_m[g_hat_munu, matter],
g_hat_munu = g_munu + (n²-1) u_mu u_nu,


и функция n одинакова для всех полей материи, то в нерелятивистском пределе
получаем

L ≈ (m/2n) v² - m/n,
a = -∇ ln n,


и масса пробного тела сокращается из уравнения движения.

Следовательно, универсальность свободного падения здесь является не отдельным
постулатом, а следствием универсального отклика вакуума.

### 5.1.10) Что должно проверяться численно

Из лемм выше следуют минимальные критерии верификации:
1. FRW-прогоны с I_grad=0 должны давать J_refr=0 и не включать
   рефрактивный сектор.
2. Выбранная family 𝒩(J) должна оставаться монотонной и конечной на всей
   рабочей области параметров.
3. В сильное поле сканs существует конечный J_refr без смены ветви и без
   blow-up n.
4. При конечной энергии не должно появляться численно достижимого theta=0.

### 5.2) Связь n(theta_eff,Theta,Psi) с тензором

Индекс преломления — функция инварианта Theta. Здесь важно: регуляризованная форма
не является первичным онтологическим постулатом, а только рабочей записью
конечной физической ветки и численной устойчивости решатель'а:
n(theta_eff,Theta,Psi) = 1 + 𝒩(J_refr).


где Phi_eff(theta_eff) — слабое поле прокси:
Phi_eff(theta_eff) = ln(theta_eff / theta_0)  (логарифмическая связь).


При theta_eff → theta_0 и фиксированном слабом режиме:
n ≈ 1 + κ · J_refr ≈ 1 + κ · Phi_eff I_grad.


Канонический выбор κ = 2 задаёт слабое поле нормировку на той ветви, где
J_refr уже сведён к эффективному коэффициенту отклика:
n ≈ 1 + 2 Phi_N,  где Phi_N — ньютоновский потенциал.


### 5.2.1) Откуда в этом языке берётся гравитация

Если Theta_munu неоднороден в пространстве, то неоднороден и theta_eff, а вместе с
ним неоднороден и отклик вакуума n(theta_eff). Это означает, что вакуум задаёт
неодинаковый локальный ход процессов в разных точках.

Для пробного тела:
S_probe = -m ∫ dtau,   dtau = dt / n(theta_eff).


В нерелятивистском пределе:
L ≈ (m/2n) v² - m/n,


и уравнение движения даёт

a = -∇ ln n(theta_eff).


То есть гравитация в этой рамке возникает не как отдельная силовая сущность, а
как отклик тел на пространственный градиент причинно-временной проводимости
вакуума.

### 5.2.2) Что означает «преломление вакуума»

Это выражение следует понимать широко. Преломляется не один фотон и не
"скорость света сама по себе", а режим переноса причинности в вакууме,
определяемый Theta.

- Для фотонов это даёт изгиб лучей и задержку сигнала.
- Для массивных тел это даёт перестройку траекторий через тот же отклик вакуума.
- Для локальных часов это даёт замедление или ускорение темпа хода.

Поэтому "преломление вакуума" в Theta–Psi означает изменение того, как вакуум
проводит причинные процессы, а свет лишь один из наблюдаемых зондов этого
режима.

### 5.3) Тензорные волны из g_hat_munu

Вариация действия по g_hat_munu даёт индуцированный Эйнштейновский член (Сахаров):
Γ_eff[g_hat] ⊃ ∫ d^4x √(-g_hat) · (M_ind²/2) · R(g_hat).


Линеаризация g_hat_munu = eta_munu + h_munu даёт уравнение тензорных волн:
□ h_ij^TT = 0  ⇒  c_T = 1.


**Вывод:** тензорные волны — это TT-моды эффективной метрики g_hat_munu(Theta), а не самого Theta_munu.

**Важно:** в Theta–Psi нет фундаментальных гравитонов. Тензорные волны — эмерджентное
явление, колебания индуцированной метрики g_hat_munu. Они не квантуются в рамках
классической Theta–Psi (квантовый предел требует отдельного вывода).

### 5.4) Следствие для горизонта

Если временной режим обязан оставаться положительным, то редуцированная
оптическая запись не может привести к физически достижимому обнулению времени
за конечный ход. В конечной форме это выглядит как конечность
n(theta_eff) и, следовательно, конечность redshift/delay.

Смысловой первичен не верхний предел на n, а положительный статус времени как
объекта; конечный n здесь является рабочей вычислительной формой этого
требования.

### 6.2) Строгий proof для текущего канона

Ниже зафиксирован не рекламный текст, а рабочий proof-скелет для уже
прописанного reduced canon. Он строгий в том смысле, что каждый вывод опирается
на явно выписанные предпосылки. Он не выдумывает новую микрофизику; он
закрывает то, что уже содержится в S_red, L_refr, stability_реестр и
theory/adm_symbolic.py.

#### Теорема 1. Ковариантная замкнутость reduced канона

Пусть выполнены условия:
1. theta_eff^2 = Theta_munuu^muu^nu и Phi_eff = ln(theta_eff/theta_0) являются скалярами;
2. S_red = ∫ √(-g)[ EH + L_Theta + L_Psi - U_red + M_*^4 F(J_refr)] d^4x;
3. J_refr = Phi_eff · I_grad, I_grad = l_theta^2|∇_⊥theta|^2 + l_Psi^2|∇_⊥Psi|^2;
4. matter couples only through the sing <= propagation law encoded by g_hat_munu = g_munu + (n^2-1)u_muu_nu;
5. F depends only on J_refr and not on gauge-dependent objects.

Тогда:
1. каждый член S_red диффеоморфизм-инвариантен;
2. вариация по g_munu даёт один согласованный T^(red)_munu;
3. FRW-фон и stationary сильное поле сектор являются двумя предельными
   реализациями одного и того же action, а не разными теориями;
4. редуцированный канон замкнут на уровне полевых инвариантов theta_eff,
   Phi_eff, J_refr, g_hat_munu.

Доказательство:
1. EH, √(-g), ∇_mu и g_munu являются стандартно ковариантными объектами.
2. theta_eff и Phi_eff построены из полной свертки по u^mu, поэтому они
   скаляры; I_grad и J_refr составлены из скалярных норм пространственных
   градиентов и потому также скаляры.
3. U_red зависит только от theta_eff и Psi, а L_refr зависит только от
   J_refr; следовательно, ни один член не требует внешнего порога, ручного
   переключателя или скрытого поля.
4. На FRW-фоне ∇_i theta_eff = ∇_i Psi = 0, значит I_grad = 0, J_refr = 0,
   L_refr = 0. На stationary неоднородном фоне I_grad > 0, и тот же член
   включается без смены закона.
5. Следовательно, ковариантный reduced canon замкнут.

#### Теорема 2. Отсутствие ghost и лишних мод в reduced каноне

Пусть выполнены условия:
1. theta_eff != 0 в рассматриваемой области;
2. gamma >= 0, V0 >= 0;
3. ctheta2 > 0, cpsi2 > 0;
4. F(0)=0, F'(J)>0, F''(J)<=0 для J>=0;
5. J_refr >= 0;
6. Лапса/Шифта не имеют собственных кинетических членов и остаются
   algebraic constraints;
7. для stationary оператора выполнено M_11 > 0, M_22 > 0, det(M) > 0.

Тогда:
1. FRW kinetic matrix положительно определена;
2. FRW gradient matrix положительно определена;
3. stationary radial operator эллиптичен;
4. дополнительные propagating modes отсутствуют;
5. ghost/gradient instabilities исключены в рабочей области reduced канона.

Доказательство:
1. FRW kinetic matrix в текущем каноне равна
   K_FRW = diag(3/theta_eff² + gamma², 1). При theta_eff != 0 и gamma >= 0
   диагональные элементы строго положительны, значит K_FRW ≻ 0.
2. FRW gradient matrix задан как
   G_FRW = diag(c_theta² K_theta, c_Psi²). При c_theta² > 0, c_Psi² > 0
   и K_theta > 0 она также положительно определена. Следовательно, gradient
   instability отсутствует.
3. F'(J)>0 и F''(J)<=0 означают, что refractive response монотонен и
   saturating, а не разгоняющийся. Это исключает sign flip и runaway в
   рефрактивном секторе.
4. M_11 > 0, M_22 > 0, det(M) > 0 означает положительную
   определённость локального квазилинейного оператора. Значит stationary PDE
   эллиптичен, а hidden propagating branch не возникает.
5. Lapse/shift в ADM-скелете не получают кинетических членов, поэтому они не
   добавляют новых физических мод; они остаются ограничениями, а не
   распространяющимися степенями свободы.
6. Комбинируя пункты 1-5, получаем отсутствие ghost, gradient и extra modes в
   reduced каноне на всей рабочей области, где выполняются перечисленные
   предпосылки.

#### Следствие

Если reduced canon проходит stability_реестр(...).healthy == True, то он
прошёл:
1. ковариантную замкнутость на уровне текущего S_red;
2. проверку на ghost/gradient/tachyon/extra-mode в canonical рабочей области;
3. baseline-условие для перехода к 1-2 primary наблюдаемые и только затем к fit.

#### Теорема 3. Полный perturbation реестр замыкает ADM-канон

Пусть выполнены условия Теоремы 1 и Теоремы 2, а также:
1. для quadratic sector заданы матрицы K, G, M в форме
   L^(2) = 1/2 dq̇ᵀ K dq̇ - 1/2 dqᵀ[ G(k²/a²) + M ]dq;
2. Лапса/Шифта исключаются через Шура complement;
3. прямой и Шура-каналы дают одинаковый результат для K, G, M;
4. stability_реестр даёт healthy = True на рабочей области.

Тогда:
1. ADM-редукция согласована с covariant action реестр;
2. quadratic perturbation sector не содержит скрытого разъезда между direct и
   Шура completion;
3. рабочий perturbation proof закрыт в пределах текущего канона.

Для этого в репозитории существует единый реестр:
- theory/perturbation_proof.py

Его выходы:
1. S_cov как полный covariant completion;
2. K_direct/G_direct/M_direct и K_schur/G_schur/M_schur;
3. routes_match;
4. health;
5. proof_closed.

### 6.3) Минимальный derived S_micro

S_micro -> S_red -> S_cov фиксируется как рабочая иерархия. Чтобы не
начинать канон с S_cov как с постулата, ниже задаётся минимальный derived
microphysics layer, выводимый из базы Theta_munu, Psi. Он нужен только как
локальный слой, из которого текущий S_red получается редукцией, а S_cov
- добавлением matter completion. Он не вводит новую геометрию.

#### 6.3.1) Поля и структура

Фундаментальные поля derived-слоя:
1. Theta_munu - clock sector;
2. Psi - energy sector;
3. χ_a - материальные поля;
4. local deviation scalar R_micro, fixed as the minimal local quadratic
   deviation from the canonical слабое поле point;
5. derived preferred-slicing volume form measure_dt_d3x = dt d^3x;
6. local response coefficients Z_t(R_micro) и Z_s(R_micro).

Минимальный local action в preferred slicing:
S_micro = ∫ dt d^3x [ L_Theta^micro + L_Psi^micro + L_matter^micro + L_int^micro ].


Состав:
R_micro = (theta_eff/theta_0 - 1)^2 + Psi^2/M_pl^2 + chi^2/M_pl^2 + |nabla theta_eff|^2/Lambda_theta^4 + |nabla Psi|^2/Lambda_Psi^4 + |nabla chi|^2/Lambda_chi^4,
A_Theta = 1 + R_micro,
B_Theta = gamma^2 + R_micro,
A_Psi = 1 + R_micro,
B_Psi = 1 + R_micro,
Z_t = 1 + R_micro,
Z_s = 1 + 2 * R_micro,
V_Theta_micro = 1/2 * m_theta^2 * theta_eff^2 + 1/4 * lambda_theta * theta_eff^4,
V_Psi_micro = 1/2 * V0 * R^2 + 1/4 * lambda_Psi * R^4,
V_matter_micro = 1/2 * m_m^2 * chi^2 + 1/4 * lambda_m * chi^4,
L_Theta_micro = 3/2 * M_pl^2 * A_Theta * |nabla ln theta_eff|^2 + 1/2 * B_Theta * |nabla theta_eff|^2 - V_Theta_micro,
L_Psi_micro = 1/2 * A_Psi * |nabla R|^2 - V_Psi_micro,
L_matter_micro = 1/2 * Z_t * chi_dot^2 - 1/2 * Z_s * |nabla chi|^2 - V_matter_micro,
L_int_micro = (R_micro / M_pl^2) * (theta_eff^2 * R^2 + theta_eff^2 * chi^2).


#### 6.3.2) Слабое поле согласование

В слабое поле limit:
R_micro → 0,
A_Theta → 1,  B_Theta → gamma²,  A_Psi → 1,  B_Psi → 1,
Z_t → 1,  Z_s → 1.


Согласование-правило для рефрактивного сектора:
n² = Z_s / Z_t.


Следовательно:
1. n → 1 в однородном слабое поле пределе;
2. отличие от слабое поле живёт в локальных response coefficients, а не в
   новой онтологии геометрии;
3. S_red является reduced/covariant completion этого local action после
   интегрирования тяжёлых мод;
4. S_cov получается как S_red + S_m[g_hat, matter], то есть через
   добавление matter completion поверх reduced слоя.
5. В коде мост фиксируется как совпадение S_red_from_micro = S_red и
   S_cov_from_micro = S_cov на уровне canonical реестрs.
6. Переход от S_micro к S_red теперь algebraic: он идёт через
   R_micro -> 0, а не через свободные функции коэффициентов или отдельную
   ручку силы отклика.

#### 6.3.2.1) Замыкание пунктов 1-3: propagation law

Этот подпункт фиксирует, почему g/g_hat, универсальность связи и слабое поле
согласование не являются отдельными постулатами поверх Theta–Psi, а читаются как одна
цепочка вывода.

**Пункт 1: статус g/g_hat.**

В S_micro первичными остаются только:
Theta_munu, Psi, χ_a, R_micro, Z_t(R_micro), Z_s(R_micro).


Для matter-поля χ локальный квадратичный блок имеет вид:
L_matter^micro =
  1/2 Z_t(R_micro) χ̇²
  - 1/2 Z_s(R_micro) |∇χ|²
  - V_matter(χ).


Отношение коэффициентов задаёт локальный propagation cone:
c_matter² = Z_s/Z_t.


Поэтому запись

n² = Z_s/Z_t


является не новой геометрией, а сжатым способом сказать: локальные часы,
световые сигналы и matter-процессы используют один и тот же response ratio.
g_hat вводится только после этого как bookkeeping-запись:
S_matter[χ; Z_t,Z_s]  ->  S_matter[χ; g_hat],
g_hat = g + (n² - 1) u ⊗ u.


Следовательно, g и g_hat не являются третьей онтологической сущностью.
Они допустимы только если не несут независимых степеней свободы сверх
Theta_munu, Psi и response coefficients.

**Пункт 2: универсальность.**

Универсальность связи означает:
∀χ_a:  Z_t^a(R_micro) = Z_t(R_micro),
      Z_s^a(R_micro) = Z_s(R_micro).


Тогда для любых matter-секторов:
n_a² = Z_s^a/Z_t^a = Z_s/Z_t = n².


Если появляются species-dependent коэффициенты

Z_t^a != Z_t^b  или  Z_s^a != Z_s^b,


то это уже не текущий канон Theta–Psi: разные вещества видят разные локальные
законы распространения, и принцип единого clock-sector response ломается.
Поэтому универсальность Z_t,Z_s является обязательным acceptance criterion
для derived micro layer.

**Пункт 3: слабое поле согласование.**

Слабое поле точка определяется как:
R_micro -> 0,
Z_t -> 1,
Z_s -> 1,
n² -> 1,
n -> 1.


Первая физически значимая поправка записывается через слабое поле scalar
Phi_eff:
n(theta_eff) = 1 + 2 Phi_eff + O(Phi_eff²).


Это не новая онтология, а нормировка observab <= layer. Она фиксирует один
коэффициент так, чтобы один и тот же n давал согласованные слабое поле
эффекты:
Δnu/nu ≈ Δ ln n,
t_signal = ∫ n dl,
α_lens ≈ ∫ ∇_⊥ ln n dl,
a_probe = -∇ ln n.


Иными словами, слабое поле согласование закрыт в каноне тогда и только тогда,
когда:
1. R_micro -> 0 возвращает инертный локальный режим;
2. все matter-секторы имеют один и тот же n;
3. линейная нормировка n = 1 + 2 Phi_eff + O(Phi_eff²) зафиксирована до fit;
4. higher-order поправки не вводят composition-dependent propagation.

**Итоговая цепочка:**

local Theta–Psi basis
  -> R_micro
  -> universal Z_t,Z_s
  -> n² = Z_s/Z_t
  -> g_hat as bookkeeping only
  -> слабое поле наблюдаемые


Это и есть замыкание пунктов 1-3 в текущем каноне. В коде оно закреплено
функцией propagation_law_замыкание_реестр() в theory/micro_action.py и
тестом test_propagation_law_замыкание_pins_points_1_2_3:
1. g_hat_independent_dof = 0;
2. Z_t_chi_a = Z_t_chi_b, Z_s_chi_a = Z_s_chi_b;
3. R_micro -> 0 даёт Z_t -> 1, Z_s -> 1, n -> 1;
4. линейная observable-нормировка n = 1 + 2 Phi_eff + O(Phi_eff²) фиксируется
   до fit.

Следовательно, пункты 1-3 считаются закрытыми на уровне текущего канона,
реестр и тестов. Дальнейшая работа может уточнять higher-order поправки, но
не должна менять эту цепочку без явного изменения канона.

#### 6.3.2.1a) Anti-fit guardrails

Чтобы derived micro layer не превратился в набор ручек для получения нужных
чисел, вводится явное разделение:
canonical = fixed before fit or derived from the action,
diagnostic = useful for сканs, but not a physical claim.


В текущем каноне фиксируются:
primary ontology = Theta_munu, Psi,
theta_eff² = Theta_munuu^muu^nu,
Phi_eff = ln(theta_eff/theta_0),
слабое поле slope n ≈ 1 + 2Phi_eff,
слабое поле profi <= n(Phi_eff) = 1 + 2 asinh(Phi_eff),
n² = Z_s/Z_t,
EPR singlet ru <= E(a,b) = -a·b,
partic <= = stab <= localized Theta-projection of a Psi-sector.


В diagnostic-only слой выносятся:
слабое поле profi <= alternatives: linear/exp/tanh2,
legacy explicit profi <= selector,
F_v1(J) = 1 - exp(-J),
ADM eps*/alpha* mixings,
legacy/exploratory sigma extension,
сильное поле brid >= calibration coefficients,
inverse-BAO required multipliers.


Их нельзя использовать как физический вывод. Любой такой элемент может быть
повышен до канона только при одновременном выполнении:
1. вывод из действия или из более малого набора постулатов;
2. фиксация до просмотра data-facing fit;
3. отсутствие выбора по test set;
4. отсутствие новых лёгких мод без наблюдательной цены;
5. независимое наблюдаемое предсказание.

Запрещённые ходы:
choose_profile_after_data,
retune_bridge_on_test,
promote_inverse_diagnostic_to_correction,
use_adm_eps_without_derivation,
add_sigma_as_physical_field_without_new_canon.


Кодовая фиксация: anti_fit_реестр() и canonical_promotion_check() в
theory/anti_fit.py, тесты в tests/test_anti_fit.py.

#### 6.3.2.1b) Что уже выводится из действия

Из текущего минимального Theta–Psi действия для
q = (d_theta, d_R, deltaφ) следует диагональный high-k квадратичный блок:
K = diag(3/theta² + gamma², 1, R²),
G = diag(3/theta² + gamma², 1, R²),
M = diag(0, V0 + 3Q²/(a⁶R⁴), 0).


Следствие:
epsK* = 0,
epsG* = 0,
alpha_k2* = 0,
c_s² = (1, 1, 1).


Это не дополнительная подгонка, а отрицательный вывод из действия: если в
действии нет недиагональных kinetic/gradient-операторов и higher-spatial
derivative операторов, то соответствующие смешения в квадратичном действии
не появляются. Ненулевые epsK*, epsG*, alpha_k2* могут войти в канон
только вместе с явно записанными новыми операторами в S_cov.

Для преломляющего сектора текущее действие выводит аргумент

J_refr = Phi_eff · I_grad,
L_refr = M_*⁴ F(J_refr),


и требования к функции:
F(0)=0,
F'(J) ≥ 0,
finite response for the no-horizon branch.


Но сам выбор экспоненциальной формы из текущего действия не следует.
Следует только то, что J_* не является отдельным физическим параметром и
может быть поглощён в нормировку J_refr.

Мост сильных полей (theta_c_scale, m2_scale, lam_scale,
powerlaw2d) тоже не следуют из текущего действия. Для их вывода нужно решить
сourced static Euler-Lagran >= equations с граничными условиями, а не
калибровать коэффициенты по EHT/GW цель-набору.

Кодовая фиксация: theory/action_derivation.py,
tests/test_action_derivation.py.

Принцип, который делает экспоненту выводом, а не выбором формы:
S(J) = 1 - F(J),
S(J1 + J2) = S(J1) S(J2).


То есть независимые приращения J действуют без памяти и мультипликативно
на оставшуюся незадействованную часть отклика. При непрерывности такой закон
даёт

S(J) = exp(-J/J_*),
F(J) = 1 - exp(-J/J_*).


В этой записи J_* уже не физическая ручка, а только нормировка остатка
отклика.

#### 6.3.2.2) Vacuum offset source-map

Проверка вакуумной катастрофы в текущем каноне формулируется не как
утверждение "нулевая энергия отсутствует". Нулевые энергии matter-мод могут
существовать и проявляться в boundary-разностях, например в эффекте Казимира.
Проверяется другое: абсолютная аддитивная константа в effective matter action
не является прямым источником для Theta–Psi.

Минимальная запись:
Γ_matter^eff = Γ_response[Z_t,Z_s,n,boundary data] + ∫ dt d³x Λ_vac_offset.


Source map для базиса Theta_munu, Psi берётся только от вариаций по локальным
response data, то есть по тем величинам, которые меняют режим распространения:
source_ThetaPsi  deltaΓ_eff/d_theta_eff, deltaΓ_eff/d_R, deltaΓ_eff/d_R_micro.


Для абсолютного offset:
deltaΛ_vac_offset/d_theta_eff = 0,
deltaΛ_vac_offset/d_R = 0,
deltaΛ_vac_offset/d_R_micro = 0.


Значит, такой offset не входит в Theta–Psi источник как огромная космологическая
постоянная. При этом boundary/response-вклады остаются физическими:
Γ_boundary  ΔΓ_boundary · n(R_micro),
Γ_response  Δresponse · n(R_micro),
deltaΓ_boundary/d_R_micro != 0,
deltaΓ_response/d_R_micro != 0.


Иными словами:
1. абсолютная сумма 1/2 sum ħω не является наблюдаемым Theta–Psi response сама по
   себе;
2. Казимир сохраняется, потому что это разность спектров при разных boundary
   data;
3. локальная QFT сохраняет нулевые моды как часть effective action, но
   гравитирующий/source-вклад проходит через response map, а не через
   произвольный additive constant.

Кодовая фиксация: vacuum_offset_source_реестр() и тесты
test_vacuum_offset_source_реестр_* в tests/test_micro_action.py.

#### 6.3.3) Статус

Это derived layer from the Theta_munu, Psi basis, а не отдельная онтология. Он
нужен, чтобы:

2. держать явным правило n² = Z_s/Z_t;

#### 6.3.4) Reduction check

Чтобы мост был не декларативным, а вычислимым, фиксируем слабополевой
редукционный тест:
1. A_Theta -> 1, B_Theta -> 1, A_Psi -> 1, B_Psi -> 1;
2. Z_t -> 1, Z_s -> 1;
3. n² -> 1 и, следовательно, n -> 1;
4. L_Theta^micro и L_Psi^micro возвращаются к canonical quadratic blocks;
5. L_matter^micro возвращается к canonical local matter kinetic block;
6. тяжёлые моды интегрируются и не остаются отдельным propagating sector.

Этот тест не заменяет физический вывод из базы Theta_munu, Psi. Он фиксирует,
что derived microphysics layer не ломает слабое поле согласование и действительно
ведёт к уже принятому reduced канону.

#### 6.3.5) Micro perturbation check

Отдельно фиксируем минимальный perturbation sanity для derived micro layer:
1. A_Theta > 0, B_Theta > 0, A_Psi > 0, B_Psi > 0;
2. Z_t > 0, Z_s > 0;
3. c_Theta² = B_Theta / A_Theta > 0, c_Psi² = B_Psi / A_Psi > 0;
4. n² = Z_s / Z_t > 0;
5. no_new_modes == True на рабочей ветке.

Это не полный UV-proof на уровне фундаментальной микрофизики. Это честный
микропертурбационный барьер: derived layer должен быть здоровым уже на
локальном квадратичном уровне, иначе его нельзя поднимать выше.

#### 6.3.6) Derivation ladder

Чтобы derived слой не расползался в отдельные костыли, в коде и каноне фиксируется
явная лестница вывода:
1. S_micro;
2. weak_field_projection;
3. integrate_heavy_modes;
4. assemble_S_red;
5. add_S_matter_completion;
6. S_cov.

Эта лестница не претендует на UV-единственность. Она фиксирует, что текущий
derived microphysics layer не просто “совместим” с каноном базы Theta_munu, Psi,
а ведёт к нему в одном и том же порядке редукции.

#### 6.3.7) Sing <= сильное поле consequence

Из derived micro layer сохраняем только один сильнопольный выход:
1. finite redshift на compact branch;
2. finite delay на compact branch;
3. ok_no_horizon == True.

Это не второй канал с отдельной физикой и не набор прокси. Это один
сильное поле consequence, который читается через
micro_strongfield_primary_реестр() и micro_to_strongfield_primary_bridge().
Он фиксирует, что derived сильное поле consequence не расползается по
альтернативным сильнопольным индикаторам, а сводится к одному
наблюдаемому классу: compact_no_horizon.

#### 6.3.8) Symbol audit

Для публичного разбора каждая сущность микрослоя получает статус:
1. derived - выводится из projection, согласование или reduction from the Theta–Psi basis;
2. assumed - минимальный объект, без которого reduced completion не
   собирается;
3. diagnostic - итоговая проверка, не базовая физика.

В текущем derived micro layer это распределяется так:
derived: theta_eff, measure_dt_d3x, R_micro, A_Theta, B_Theta, A_Psi, B_Psi, Z_t, Z_s,
         n, n^2, S_micro, S_red, S_cov, bridge_to_covariant
assumed: R, chi
diagnostic: finite redshift, finite delay, ok_no_horizon


Это и есть честная публичная форма ответа на вопрос “почему именно эти
коэффициенты”: большая часть символов не выбирается произвольно, а помечается
как derived от reduction rules; measure_dt_d3x = dt d^3x - derived
preferred-slicing volume form, а R_micro - минимальный локальный
квадратичный deviation scalar, выведенный из локальности, чётности,
positivity и слабое поле normalization.

## 7) Канонический класс n(theta_eff)

Family-v1 (current working class):
n(theta_eff; p, thetas) = 1 + 𝒩(p · thetas / theta_eff),
p>0, thetas>0.


Свойства 𝒩:
1. 𝒩(0)=0;
2. 𝒩 монотонна на рабочей ветке;
3. на физической ветке n остаётся конечным без введения отдельного
   потолка как физической сущности.

Правила:
1. Семейство фиксируется до подгонки.
2. Смена семейства после просмотра результата запрещена.
3. Альтернативы допускаются только как заранее объявленный blind-test.

## 8) Жесткие критерии отбраковки

Модель/ветка отвергается, если стабильно нарушен хотя бы один пункт:
1. Слабое поле тесты (Меркурий/Шапиро/красное смещение/отклонение света).
2. Устойчивость (ghost/gradient) в рабочей области параметров.
3. Совместный контур CC+SN+BAO+сильное поле.
4. Согласие достигается только скрытой подгонкой на test-наборе.
5. Нет стабильного решения при фиксированном n(theta_eff) классе.

## 9) Статус Psi (обязательный nested-тест)

Сектор Psi считается каноническим только при прохождении:
1. M0: Theta-only (без динамического Psi);
2. M1: Theta+Psi;
3. единый train/test протокол;
4. критерий принятия: ΔBIC = BIC(M1)-BIC(M0) < -6 стабильно на k-fold.

Если ΔBIC >= 0 в большинстве fold, Psi исключается из канона.
