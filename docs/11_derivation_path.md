# 11. Линия вывода Θ–Ψ: тензорное время как канон

Цель: зафиксировать без двусмысленности, что в канонической версии фундаментальным объектом времени является тензор `Θ_{μν}`, а скаляр `θ` — только его редуцированная проекция для текущих вычислений FRW.

## 1) Постулаты канона

1. В качестве фундаментального объекта времени принимается `Θ_{μν}` (симметричный тензор 2-го ранга).
2. Энергетический сектор моделируется комплексным полем `Ψ`.
3. Предполагается, что наблюдаемая динамика материи задается эффективным откликом вакуума, который
   определяется инвариантами `Θ_{μν}`.
4. В weak-field пределе рабочая версия теории должна воспроизводить проверенные наблюдаемые тесты.
5. Поскольку время в гипотезе понимается как физический объект, допустимая динамика не должна
   переводить временной режим в нуль или отрицательную ветвь.

## 2) Базовое ковариантное действие (структура)

```text
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_Θ(Θ_{μν}, g) + L_Ψ(Ψ, g) + L_int(Θ,Ψ,g) ].
```

### 2.1) Единое reduced action для FRW и strong-field

Фиксируем единый reduced action, который воспроизводит оба рабочих предела:

```text
S_red = ∫ d^4x √(-g) [
  (M_pl²/2)R
  + L_Theta
  + L_R
  - U_red(θ_eff,R)
  + M_*^4 F(J_refr)
].
```

В этом каноне:

```text
L_Theta = 3/2 · ∇_μ ln θ_eff ∇^μ ln θ_eff + 1/2 · γ² ∇_μ θ_eff ∇^μ θ_eff
L_R     = 1/2 · ∇_μ R ∇^μ R
```

С минимальным потенциалом:

```text
U_red(θ_eff,R) = 1/2 m_θ² θ_eff² + λ_θ/4 θ_eff⁴ + 1/2 V0 R² + λ_{ΘΨ}/2 θ_eff² R².
```

Смысл этой записи:

1. На FRW-фоне `∇_i θ_eff = ∇_i Ψ = 0`, поэтому `J_refr = 0` и остаётся
   однородный лагранжиан.
2. В стационарном неоднородном режиме temporal/energy gradients включают
   `F(J_refr)`, давая strong-field ответ из того же действия.
3. Это единый reduced covariant action, а не два независимых лагранжиана.

### 2.2) Текущий covariant action ledger

В репозитории этот переход теперь поддержан не только текстом, но и отдельным
символическим ledger-модулем:

- `theory/covariant_action.py`

Его роль:

1. зафиксировать `θ_eff² = Θ_{μν}u^μu^ν`;
2. хранить канонический `Φ_eff^can = ln(θ_eff/θ_0)`;
3. держать один symbolic source для `S_red`, stationary `J_refr` и reduced
   stress-energy blocks;
4. позволить проверять, что FRW и stationary strong-field действительно читаются
   как два предела одного reduced action, а не как две независимые подстановки.

Это канонический ledger действия: он существует, тестируется и задаёт рабочий
канон `S_red`.

### 2.3) Уравнения движения и редуцированный `T_{μν}`

Из `S_red` для однородного FRW-среза фиксируем единый minisuperspace ledger:

```text
E_θ = (3/θ² + γ²) θ̈ - 3 θ̇² / θ³ = 0,
E_Ψ = Ψ̈ + V0 Ψ = 0.
```

Соответствующие плотность энергии и давление:

```text
ρ_red = 3/2 · (θ̇/θ)² + 1/2 · Ṙ² + 1/2 · V0 R² + 1/2 · γ² θ̇²,
p_red = 3/2 · (θ̇/θ)² + 1/2 · Ṙ² - 1/2 · V0 R² + 1/2 · γ² θ̇².
```

Следовательно:

```text
T^{(red)}_{00} = ρ_red,
T^{(red)}_{ii} = a² p_red  (без суммирования по i).
```

В reduced каноне это и есть минимальный замкнутый набор: действие,
вариация по полям и сводка по `T_{μν}` в одном источнике.

### 2.4) Полный covariant completion

Полный ковариантный объект канона фиксируется как

```text
S_cov = S_red + S_m[g_hat, matter],
g_hat = g + (n²(θ_eff,Θ,Ψ) - 1) u ⊗ u.
```

Здесь `g_hat` — не новая онтологическая геометрия, а компактная запись
рефракционного и clock-sector отклика вакуума. `S_m[g_hat, matter]` означает,
что материя и излучение получают один и тот же закон распространения через
этот отклик. Это и есть полный ковариантный completion для текущего канона:
reduced ядро плюс единое правило связи материи.

В этой записи `g` и `g_hat` имеют статус языка учета, а не первичных объектов
гипотезы. Первичными остаются `Θ_{μν}` и `Ψ`. Метрическая форма нужна только
для того, чтобы записать локальный закон распространения процессов без
разных правил для света, часов и массивной материи.

Вариация по метрике:

```text
G_{μν} = 8πG · (T^{(m)}_{μν} + T^{(Θ)}_{μν} + T^{(Ψ)}_{μν} + T^{(int)}_{μν}).
```

Критически важно: наблюдаемые тензорные возмущения должны быть коллективными
возмущениями effective propagation law, а не следствием одной скалярной
редукции `θ_eff`.

## 3) Тензорный лагранжиан L_Θ: минимальная форма

### 3.1) Требования к L_Θ

1. **Диффеоморфизм-инвариантность**: скаляр относительно координатных преобразований.
2. **Минимальное число производных**: не более 2-го порядка для гиперболичности.
3. **Положительная энергия**: отсутствие призраков (ghost-free).
4. **Корректный инертный предел**: при Θ_{μν} → const эффективный отклик вакуума
   перестаёт вносить дополнительную неоднородность.

### 3.2) Минимальный лагранжиан на пространстве тензоров

Наиболее общая форма без призраков для симметричного тензора 2-го ранга:

```text
L_Θ = 1/2 · G^{μνρσ}(Θ) · ∇_μ Θ_{νρ} · ∇_σ Θ_{αβ} · g^{αβ} - U(Θ_{μν}Θ^{μν}),
```

где `G^{μνρσ}` — метрика на пространстве конфигураций Θ.

Для изотропного FRW-фона с выделенным временем `u^μ = (1,0,0,0)` в комовинговом кадре:

```text
Θ_{μν} = diag(θ²(t), 0, 0, 0)  (в синхронной калибровке),
```

тогда единственный нетривиальный инвариант:

```text
I_1 = Θ_{μν} u^μ u^ν = θ²(t).
```

### 3.3) Редукция к скаляру θ_eff

Определяем:

```text
θ_eff ≡ √(Θ_{μν} u^μ u^ν),   u^μ u_μ = -1.
```

В FRW метрике `ds² = -dt² + a²(t) dx²`:

```text
u^μ = (1, 0, 0, 0),   u_μ = (-1, 0, 0, 0),
Θ_{00} = θ²(t),   Θ_{ij} = 0,
θ_eff = √(Θ_{00} u^0 u^0) = √(θ² · 1 · 1) = θ(t).
```

### 3.4) Вывод кинетического члена

Из тензорного лагранжиана:

```text
L_Θ ⊃ 1/2 · G^{0000}(Θ) · ∂_0 Θ_{00} · ∂_0 Θ_{00}.
```

Для Θ_{00} = θ²:

```text
∂_0 Θ_{00} = 2θ θ̇,
L_Θ ⊃ 1/2 · G^{0000} · (2θ θ̇)² = 2 G^{0000} · θ² θ̇².
```

Требование канонической нормировки (см. 08_foundations.md):

```text
L_eff ⊃ 3/2 · (θ̇/θ)² = 3/2 · θ^{-2} · θ̇².
```

Сравнивая:

```text
2 G^{0000} · θ² = 3/2 · θ^{-2}  ⇒  G^{0000} = 3/4 · θ^{-4}.
```

**Вывод:** метрика конфигурационного пространства на Θ:

```text
G^{μνρσ}(Θ) = 3/4 · (Θ_{αβ} u^α u^β)^{-2} · P^{μνρσ},
```

где `P^{μνρσ}` — проектор на временную компоненту.

### 3.5) Член смешения γ² θ̇²

Этот член возникает из взаимодействия Θ и Ψ:

```text
L_int ⊃ γ²/2 · (∇_μ Θ_{νρ}) (∇^μ Ψ) (∇^ν Ψ) (∇^ρ Ψ) / (Ψ̄ Ψ).
```

В FRW пределе при Ψ = Ψ(t):

```text
L_int ⊃ γ²/2 · θ̇² · Ψ̇² / |Ψ|².
```

Для однородного фона Ψ̇²/|Ψ|² ≈ 1 (нормировка поля), остаётся:

```text
L_int ⊃ γ²/2 · θ̇².
```

**Физический смысл:** γ — безразмерная связь между градиентом времени и градиентом энергии.

## 4) Итоговый эффективный лагранжиан

После редукции Θ_{μν} → θ_eff:

```text
L_eff = 3/2 · (θ̇/θ)² + 1/2 · Ṙ² - 1/2 · V0 · R² + 1/2 · γ² · θ̇².
```

### 4.1) Проверка пределов

| Предел | L_eff | Физика |
|--------|-------|--------|
| γ → 0 | `3/2 (θ̇/θ)² + 1/2 Ṙ² - 1/2 V0 R²` | Развязанные Θ и Ψ |
| θ → const | `1/2 Ṙ² - 1/2 V0 R²` | инертный предел Ψ-сектора |
| Ψ → 0 | `3/2 (θ̇/θ)² + 1/2 γ² θ̇²` | Чистое Θ-поле (stiff matter) |
| θ̇ → 0 | `1/2 Ṙ² - 1/2 V0 R²` | статический временной режим |

### 4.2) Энергия и давление

```text
ρ = (∂L/∂θ̇)θ̇ + (∂L/∂Ψ̇)Ψ̇ - L
  = 3/2 (θ̇/θ)² + 1/2 Ṙ² + 1/2 V0 R² + 1/2 γ² θ̇²,

p = L = 3/2 (θ̇/θ)² + 1/2 Ṙ² - 1/2 V0 R² + 1/2 γ² θ̇².
```

Уравнение состояния:

```text
w = p/ρ = [3/2 (θ̇/θ)² + 1/2 Ṙ² - 1/2 V0 R² + 1/2 γ² θ̇²] / [3/2 (θ̇/θ)² + 1/2 Ṙ² + 1/2 V0 R² + 1/2 γ² θ̇²].
```

Пределы:
- Кинетическое доминирование: w → +1 (stiff matter)
- Потенциальное доминирование: w → -1 (dark energy)
- θ̇ = 0, Ψ̇ = 0: w = -1 (космологическая постоянная)

## 5) Оптическая метрика из тензора Θ_{μν}

### 5.1) Эффективная метрика для материи

Материя движется по геодезикам эффективной метрики:

```text
ĝ_{μν} = g_{μν} + (n²(θ_eff) - 1) · u_μ u_ν,
```

где `u_μ = ∂_μ θ_eff / √(∂θ·∂θ)` — выделенное временное направление.

В комовинговом кадре `u^μ = (1,0,0,0)`:

```text
ĝ_{00} = n²(θ_eff),   ĝ_{ij} = -δ_{ij}.
```

### 5.1.1) Минимальный рабочий класс `L_refr(Θ,Ψ)`

Связка

```text
Θ, Ψ  ->  J_refr(Θ,Ψ)  ->  n(θ_eff,Θ,Ψ)  ->  ĝ_{μν}
```

фиксирует минимальный рабочий класс преломляющего сектора. Его задача
заключается в том, чтобы задать, как из временного объекта и энергетического
субстрата возникает универсальный рефрактивный отклик вакуума.

Требования к `L_refr`:

1. Он зависит от `Θ` и `Ψ` только через инварианты, совместимые с каноном.
2. Он универсален для материи и излучения.
3. На FRW-фоне его вклад выключается автоматически при `I_grad = 0`.
4. В weak-field пределе он даёт линейное соответствие `ln n ~ κ Φ_eff`.
5. В strong-field режиме он насыщается и не уводит `n` в патологию.

Минимальная запись:

```text
L_refr = M_*^4 · F(J_refr),
J_refr = Φ_eff(θ_eff) · I_grad(Θ,Ψ),
Φ_eff(θ_eff) = ln(θ_eff / θ_0).
```

Здесь:

- `θ_0` — weak-field опорное значение временного режима;
- `Φ_eff` — не ньютоновский потенциал сам по себе, а канонический scalar bridge,
  который в слабом поле совпадает с ньютоновским после калибровки;
- `I_grad` — прямой инвариант пространственной неоднородности системы `Θ–Ψ`.

Тогда эффективный индекс не вводится отдельно "сверху", а определяется как
универсальный отклик вакуума на `J_refr`:

```text
n - 1 = 𝒩(J_refr),
```

где `𝒩(0)=0`, `𝒩'(0)=κ`, `𝒩(J)>=0`, `𝒩(J)` монотонна и имеет конечное
насыщение.

Эквивалентная рабочая форма:

```text
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr).
```

Эта форма фиксирует сразу три свойства:

1. при `I_grad=0` получаем `J_refr=0` и `n=1`, то есть FRW не включает
   рефрактивный сектор;
2. при малом `J_refr` имеем weak-field закон `n ≈ 1 + κ J_refr`;
3. при большом `J_refr` отклик насыщается, а численная итерация не выходит в
   неустойчивую область.

### 5.1.1a) Статус `Φ_eff`: канон против численной регуляризации

Чтобы убрать двусмысленность:

1. **Каноническое определение**:

```text
Φ_eff^can(θ_eff) = ln(θ_eff / θ_0),   θ_eff > 0.
```

2. **Численная регуляризация в solver'ах**:

```text
Φ_eff^num(θ_eff) = гладкая положительная аппроксимация(θ_eff/θ_0),
```

которая совпадает с канонической формой в weak-field matching и используется
только там, где нужна гладкая положительная функция без логарифмической
жёсткости у машинного `θ_eff ~ 0`.

Следствие: канон гипотезы не зависит от конкретной гладкой аппроксимации при
выполнении трёх условий:

- `Φ_eff^num >= 0`,
- `Φ_eff^num -> 0` при `θ_eff -> θ_0`,
- в рабочем слабом поле `Φ_eff^num = Φ_eff^can + O((θ_eff-θ_0)^2)`.

### 5.1.2) Как `L_refr` входит в действие материи

`L_refr` перестраивает эффективный локальный clock sector вакуума. Рабочая
запись:

```text
S_m  ->  S_m[ĝ_{μν}, matter],
ĝ_{μν} = g_{μν} + (n²(θ_eff,Θ,Ψ)-1) u_μ u_ν.
```

### 5.1.2a) Тензор энергии-импульса reduced сектора

Для reduced action определяем:

```text
T_{μν}^{(red)} = -2 / √(-g) · δ(√(-g)L_red) / δg^{μν}
               = T_{μν}^{(θ)} + T_{μν}^{(Ψ)} + T_{μν}^{(refr)} - g_{μν}U_red.
```

Минимальные куски:

```text
T_{μν}^{(θ)} =
  3 · ∇_μ ln θ_eff ∇_ν ln θ_eff
  + γ² ∇_μ θ_eff ∇_ν θ_eff
  - g_{μν}[ 3/2 (∇lnθ_eff)^2 + 1/2 γ² (∇θ_eff)^2 ],

T_{μν}^{(Ψ)} =
  ∇_μ Ψ ∇_ν Ψ
  - g_{μν}[ 1/2 (∇Ψ)^2 ],

T_{μν}^{(refr)} =
  -g_{μν} M_*^4 F(J_refr)
  + 2 M_*^4 F'(J_refr) · δJ_refr/δg^{μν}.
```

На FRW-фоне:

```text
I_grad = 0  =>  J_refr = 0  =>  T_{μν}^{(refr)} = 0,
```

и тензор энергии-импульса сводится к однородному `Θ–Ψ` сектору. В stationary
режиме `T_{μν}^{(refr)}` играет роль добавочного stress-sector, который и
 создаёт конечный optical response без введения отдельного поля-переключателя.

### 5.1.2b) Линейная устойчивость reduced сектора

Минимальные критерии до полного ADM-замыкания:

1. **FRW no-ghost**:

```text
K_FRW = diag( 3/θ_eff² + γ², 1 )  ≻ 0.
```

2. **FRW no-gradient-instability**:

```text
G_FRW = diag( c_θ² K_θ, c_Ψ² )  ≻ 0.
```

3. **Stationary ellipticity** для радиального two-field solver:

```text
M_11 > 0,  M_22 > 0,  det(M) > 0,
```

где `M` — локальная квазилинейная матрица из п. 5.1.3.

4. **No extra modes**:

```text
no_extra_modes = no_ghost ∧ no_gradient_instability ∧ elliptic.
```

Это даёт простой и проверяемый критерий: рабочая область параметров должна
быть одновременно no-ghost на FRW, эллиптичной в stationary strong-field
секторе и без признаков скрытой добавочной моды.

### 5.1.3) Минимальный стационарный сектор и уравнения Эйлера–Лагранжа

Чтобы strong-field solver был лагранжево замкнутым, фиксируем
минимальный стационарный радиальный функционал для редуцированного двухсущностного
сектора:

```text
L_stat = 1/2 · I_grad - U(θ,Ψ) + α · F(J_refr),
I_grad = θ'^2 + w_Ψ Ψ'^2,
J_refr = Φ_eff(θ) · I_grad,
Φ_eff(θ) = ln(θ/θ_0) или его гладкая положительная аппроксимация.
```

Здесь:

- `α` — сила рефрактивной обратной связи;
- `w_Ψ > 0` — относительный вес градиентного вклада `Ψ`;
- штрих означает производную по стационарной радиальной координате `r`.

Минимальный рабочий стационарный потенциал, согласованный с текущим
вычислительным solver:

```text
U(θ,Ψ) =
  1/2 m² θ² + λ/4 θ⁴
  - 1/2 m² Ψ² + c_{ΘΨ} λ/2 · θ² Ψ².
```

Тогда:

```text
U_,θ = m² θ + λ θ³ + c_{ΘΨ} λ θ Ψ²,
U_,Ψ = -m² Ψ + c_{ΘΨ} λ θ² Ψ.
```

Для сферически-симметричного стационарного профиля уравнения Эйлера–Лагранжа:

```text
1/r² · d/dr [ r² A θ' ] + U_,θ - α F'(J_refr) Φ'_(eff) I_grad = 0,
1/r² · d/dr [ r² A w_Ψ Ψ' ] + U_,Ψ = 0,
```

где

```text
A(θ,θ',Ψ') = 1 + 2 α Φ_eff(θ) F'(J_refr).
```

Поскольку `A` зависит от градиентов через `J_refr`, система является
квазилинейной. Её удобно переписать как локальную матричную систему для
вторых производных:

```text
M_11 θ'' + M_12 Ψ'' = R_θ,
M_21 θ'' + M_22 Ψ'' = R_Ψ,
```

с коэффициентами

```text
A_,θ   = 2 α Φ'_(eff) [F' + Φ_eff I_grad F''],
A_,θ'  = 4 α Φ_eff² F'' θ',
A_,Ψ'  = 4 α Φ_eff² F'' w_Ψ Ψ',

M_11 = A + A_,θ' θ',
M_12 = A_,Ψ' θ',
M_21 = A_,θ' w_Ψ Ψ',
M_22 = w_Ψ (A + A_,Ψ' Ψ').
```

Правая часть:

```text
R_θ = -A_,θ θ'^2 - 2Aθ'/r - U_,θ + αF' Φ'_(eff) I_grad,
R_Ψ = -A_,θ w_Ψ θ'Ψ' - 2Aw_ΨΨ'/r - U_,Ψ.
```

Это и есть минимальная форма, в которой текущий two-field solver уже можно
считать лагранжево мотивированным, а не просто ad hoc closure.

### 5.1.4) Weak-field и FRW пределы стационарного сектора

Если `I_grad -> 0`, то:

```text
J_refr -> 0,
F'(J_refr) -> F'(0),
A -> 1,
```

и радиальная система сводится к обычному двум слабо связанным уравнениям
скалярного профиля. На FRW-фоне пространственных градиентов нет, поэтому
стационарный рефрактивный блок автоматически выключен.
```

То есть `L_refr` задаёт универсальный скалярный отклик `n`, а уже через `ĝ`
этот отклик читается всеми секторами материи одинаково.

Это и есть содержательная причина, почему:

- свет линзируется,
- часы замедляются,
- массивные тела испытывают ускорение,

но при этом не вводится три разных механизма для трёх разных явлений.

### 5.1.3) FRW-предел и strong-field включение

На однородном FRW-фоне:

```text
I_grad = 0  =>  J_refr = 0.
```

Поэтому FRW-фон сам по себе не включает рефрактивный сектор.

В strong-field или квазистационарном режиме:

```text
I_grad > 0  =>  J_refr = Φ_eff I_grad > 0
```

и сильнопольный режим включается напрямую, без промежуточного поля режима.

Значит включение strong-field режима происходит так:

1. градиенты `Θ, Ψ` создают `I_grad`;
2. `I_grad` вместе с `Φ_eff(θ_eff)` создаёт `J_refr`;
3. `J_refr` включает конечный рефрактивный отклик `n(θ_eff,Θ,Ψ)`.

### 5.1.4) Как из этого получается насыщение `n`

Для малого `J_refr`:

```text
F(J) = J + O(J²),
```

поэтому

```text
n - 1 = 𝒩(J_refr) ≈ κ J_refr = κ Φ_eff I_grad.
```

То есть в слабом strong-field режиме отклик растёт аналитически и контролируемо.

Для большого `J_refr` функция `𝒩(J)` обязана насыщаться:

```text
𝒩(J) -> 𝒩_∞ < ∞   при J -> +∞.
```

Следовательно, даже при росте `I_grad` временной отклик не уходит в
патологическую ветвь. Это и есть математический механизм того, как из прямого
взаимодействия `Θ` и `Ψ` рождается конечный strong-field режим без ручного
переключателя.

### 5.1.5) Канонический класс функций `F(J)`

Чтобы сектор режима был не произвольным, фиксируем минимальные допустимые
свойства функций ещё до фиттинга и strong-field тестов.

Для `F(J)` канонически требуем:

```text
F(0) = 0,
F'(J) > 0  для J >= 0,
F''(J) <= 0  для J >= 0,
F(J) -> F_∞ < ∞  при J -> +∞.
```

Смысл:

- `F(0)=0` гарантирует отсутствие искусственного рефрактивного сдвига на фоне;
- `F'(J)>0` гарантирует, что усиление режима не уменьшает отклик вакуума;
- `F''(J)<=0` делает отклик насыщаемым, а не взрывным;
- конечный `F_∞` исключает уход в бесконечную оптическую ветвь.

Канонический normalized family-v1:

```text
F_v1(J) = 1 - exp(-J).
```

Любой явный масштаб `J_*` можно убрать перенормировкой `J_refr` до
безразмерной переменной; после этого он не является физическим параметром.

Чтобы сама экспонента стала выводом, нужен отдельный принцип для остатка
отклика:

```text
S(J) = 1 - F(J),
S(J1 + J2) = S(J1) S(J2).
```

То есть независимые приращения `J` действуют без памяти и мультипликативно
на оставшуюся незадействованную часть отклика. При непрерывности такой закон
даёт

```text
S(J) = exp(-J/J_*),
F(J) = 1 - exp(-J/J_*).
```

Именно этот шаг делает экспоненту следствием принципа полугруппы для
остатка отклика, а не отдельным выбором формы.

### 5.1.6) Почему именно этот класс считается каноническим

Для family-v1 получаем:

```text
F_v1'(J) = exp(-J) > 0,
F_v1''(J) = -exp(-J) < 0,
F_v1(∞) = 1.
```

То есть все требования выше выполнены автоматически.

Кроме того, при малом `J`:

```text
F_v1(J) = J + O(J²),
```

и weak-field режим получается аналитическим и контролируемым. При большом `J`
отклик насыщается, что делает family-v1 минимальным гладким кандидатом без
изломов и без дополнительных порогов.

### 5.1.7) Связь family-v1 с рабочей формой для `n`

Если ввести численно регуляризованную family

```text
𝒩(J) = saturating_response(J),
```

то family для `n` принимает вид

```text
n(θ_eff,Θ,Ψ) = 1 + 𝒩(J_refr),
J_refr = Φ_eff(θ_eff) I_grad(Θ,Ψ).
```

Это делает структуру однородной:

- `F(J)` управляет тем, как рефрактивный сектор входит в effective action;
- `𝒩(J)` управляет тем, как этот сектор читается как наблюдаемый индекс в
  solver-реализации;
- обе функции принадлежат одному и тому же классу: монотонный, вогнутый,
  saturating response без порога и без смены ветви.

### 5.1.8) Что допускается как альтернатива, а что нет

Допускается как blind-test family, если зафиксировано заранее:

```text
F_alt,1(J) = J / (1 + J),
F_alt,2(J) = tanh(J),
```

при тех же требованиях монотонности и насыщения.

Не допускается в каноне:

1. функции с `F'(J) < 0` на части области;
2. функции с неограниченным ростом `F(J)`;
3. кусочные пороги вида "если `J>J_c`, то включить режим";
4. переопределение family после просмотра данных.

### 5.1.9) Леммы канона

Ниже фиксируются минимальные утверждения, которые должны быть истинны для
любого допустимого `F(J_refr)`. `F_v1` в нормированной записи `1-exp(-J)`
является каноническим representative-form; явный `J_*` — только нормировка.

#### Лемма A. Выключенность рефрактивного сектора на FRW

Предпосылки:

```text
I_grad = 0,
F(0)=0.
```

Тогда

```text
J_refr = 0,   n = 1.
```

Смысл: FRW не включает рефрактивный сектор.

#### Лемма B. Монотонность и конечность рефрактивного отклика

Предпосылки:

```text
J >= 0,
𝒩(0)=0,
𝒩'(J) > 0,
𝒩''(J) <= 0,
lim_{J->∞} 𝒩(J) < ∞.
```

Тогда

```text
1 <= n(J) = 1 + 𝒩(J),
dn/dJ > 0.
```

Следовательно, усиление режима повышает индекс монотонно, но не позволяет ему
уйти в бесконечность на физической ветке решения; численная regularization
служит только для устойчивости solver'а.

#### Лемма C. Weak-field аналитичность

Если `F(J)` и `𝒩(J)` аналитичны в нуле и

```text
F(J) = J + O(J²),
𝒩(J) = κ J + O(J²),
```

то при малом `I_grad` и слабом `Φ_eff`:

```text
n - 1 = κ Φ_eff I_grad + O((Φ_eff I_grad)²).
```

Следовательно, weak-field отклик не содержит скрытого порога и включается
гладко.

#### Лемма D. Недостижимость истинного горизонта в редуцированном контуре

Предпосылки:

```text
ρ(t) <= ρ_max < ∞,
L_eff ⊃ 3/2 (θ̇/θ)²,
θ(t0) > 0.
```

Тогда

```text
|d ln θ / dt| <= sqrt(2ρ_max/3),
```

и после интегрирования:

```text
θ(t) >= θ(t0) exp(-K |t-t0|) > 0
```

для любого конечного `t`.

Следовательно, временной режим не достигает физического обнуления за конечный
ход, а в конечной optical branch это означает конечные `redshift` и `delay`.

#### Лемма E. Эквивалентность как следствие универсального `ĝ_{μν}`

Если материя входит только через

```text
S_m[ĝ_{μν}, matter],
ĝ_{μν} = g_{μν} + (n²-1) u_μ u_ν,
```

и функция `n` одинакова для всех полей материи, то в нерелятивистском пределе
получаем

```text
L ≈ (m/2n) v² - m/n,
a = -∇ ln n,
```

и масса пробного тела сокращается из уравнения движения.

Следовательно, универсальность свободного падения здесь является не отдельным
постулатом, а следствием универсального отклика вакуума.

### 5.1.10) Что должно проверяться численно

Из лемм выше следуют минимальные критерии верификации:

1. FRW-прогоны с `I_grad=0` должны давать `J_refr=0` и не включать
   рефрактивный сектор.
2. Выбранная family `𝒩(J)` должна оставаться монотонной и конечной на всей
   рабочей области параметров.
3. В strong-field scans существует конечный `J_refr` без смены ветви и без
   blow-up `n`.
4. При конечной энергии не должно появляться численно достижимого `θ=0`.

### 5.2) Связь `n(θ_eff,Θ,Ψ)` с тензором

Индекс преломления — функция инварианта Θ. Здесь важно: регуляризованная форма
не является первичным онтологическим постулатом, а только рабочей записью
конечной физической ветки и численной устойчивости solver'а:

```text
n(θ_eff,Θ,Ψ) = 1 + 𝒩(J_refr).
```

где `Φ_eff(θ_eff)` — weak-field прокси:

```text
Φ_eff(θ_eff) = ln(θ_eff / θ_0)  (логарифмическая связь).
```

При `θ_eff → θ_0` и фиксированном слабом режиме:

```text
n ≈ 1 + κ · J_refr ≈ 1 + κ · Φ_eff I_grad.
```

Канонический выбор `κ = 2` задаёт weak-field нормировку на той ветви, где
`J_refr` уже сведён к эффективному коэффициенту отклика:

```text
n ≈ 1 + 2 Φ_N,  где Φ_N — ньютоновский потенциал.
```

### 5.2.1) Откуда в этом языке берётся гравитация

Если `Θ_{μν}` неоднороден в пространстве, то неоднороден и `θ_eff`, а вместе с
ним неоднороден и отклик вакуума `n(θ_eff)`. Это означает, что вакуум задаёт
неодинаковый локальный ход процессов в разных точках.

Для пробного тела:

```text
S_probe = -m ∫ dτ,   dτ = dt / n(θ_eff).
```

В нерелятивистском пределе:

```text
L ≈ (m/2n) v² - m/n,
```

и уравнение движения даёт

```text
a = -∇ ln n(θ_eff).
```

То есть гравитация в этой рамке возникает не как отдельная силовая сущность, а
как отклик тел на пространственный градиент причинно-временной проводимости
вакуума.

### 5.2.2) Что означает «преломление вакуума»

Это выражение следует понимать широко. Преломляется не один фотон и не
"скорость света сама по себе", а режим переноса причинности в вакууме,
определяемый `Θ`.

- Для фотонов это даёт изгиб лучей и задержку сигнала.
- Для массивных тел это даёт перестройку траекторий через тот же отклик вакуума.
- Для локальных часов это даёт замедление или ускорение темпа хода.

Поэтому "преломление вакуума" в Θ–Ψ означает изменение того, как вакуум
проводит причинные процессы, а свет лишь один из наблюдаемых зондов этого
режима.

### 5.3) Тензорные волны из ĝ_{μν}

Вариация действия по ĝ_{μν} даёт индуцированный Эйнштейновский член (Сахаров):

```text
Γ_eff[ĝ] ⊃ ∫ d^4x √(-ĝ) · (M_ind²/2) · R(ĝ).
```

Линеаризация `ĝ_{μν} = η_{μν} + h_{μν}` даёт уравнение тензорных волн:

```text
□ h_{ij}^{TT} = 0  ⇒  c_T = 1.
```

**Вывод:** тензорные волны — это TT-моды эффективной метрики ĝ_{μν}(Θ), а не самого Θ_{μν}.

**Важно:** в Θ–Ψ нет фундаментальных гравитонов. Тензорные волны — эмерджентное
явление, колебания индуцированной метрики ĝ_{μν}. Они не квантуются в рамках
классической Θ–Ψ (квантовый предел требует отдельного вывода).

### 5.4) Следствие для горизонта

Если временной режим обязан оставаться положительным, то редуцированная
оптическая запись не может привести к физически достижимому обнулению времени
за конечный ход. В конечной форме это выглядит как конечность
`n(θ_eff)` и, следовательно, конечность redshift/delay.

Смысловой первичен не верхний предел на `n`, а положительный статус времени как
объекта; конечный `n` здесь является рабочей вычислительной формой этого
требования.

#### Теорема 1. Ковариантная замкнутость reduced канона

Пусть выполнены условия:

1. `θ_eff^2 = Θ_{μν}u^μu^ν` и `Φ_eff = ln(θ_eff/θ_0)` являются скалярами;
2. `S_red = ∫ √(-g)[ EH + L_Θ + L_Ψ - U_red + M_*^4 F(J_refr)] d^4x`;
3. `J_refr = Φ_eff · I_grad`, `I_grad = l_θ^2|∇_⊥θ|^2 + l_Ψ^2|∇_⊥Ψ|^2`;
4. matter couples only through the single propagation law encoded by `ĝ_{μν} = g_{μν} + (n^2-1)u_μu_ν`;
5. `F` depends only on `J_refr` and not on gauge-dependent objects.

Тогда:

1. каждый член `S_red` диффеоморфизм-инвариантен;
2. вариация по `g_{μν}` даёт один согласованный `T^{(red)}_{μν}`;
3. FRW-фон и stationary strong-field сектор являются двумя предельными
   реализациями одного и того же action, а не разными теориями;
4. редуцированный канон замкнут на уровне полевых инвариантов `θ_eff`,
   `Φ_eff`, `J_refr`, `ĝ_{μν}`.

Доказательство:

1. `EH`, `√(-g)`, `∇_μ` и `g_{μν}` являются стандартно ковариантными объектами.
2. `θ_eff` и `Φ_eff` построены из полной свертки по `u^μ`, поэтому они
   скаляры; `I_grad` и `J_refr` составлены из скалярных норм пространственных
   градиентов и потому также скаляры.
3. `U_red` зависит только от `θ_eff` и `Ψ`, а `L_refr` зависит только от
   `J_refr`; следовательно, ни один член не требует внешнего порога, ручного
   переключателя или скрытого поля.
4. На FRW-фоне `∇_i θ_eff = ∇_i Ψ = 0`, значит `I_grad = 0`, `J_refr = 0`,
   `L_refr = 0`. На stationary неоднородном фоне `I_grad > 0`, и тот же член
   включается без смены закона.
5. Следовательно, ковариантный reduced canon замкнут.

#### Теорема 2. Отсутствие ghost и лишних мод в reduced каноне

Пусть выполнены условия:

1. `θ_eff != 0` в рассматриваемой области;
2. `γ >= 0`, `V0 >= 0`;
3. `ctheta2 > 0`, `cpsi2 > 0`;
4. `F(0)=0`, `F'(J)>0`, `F''(J)<=0` для `J>=0`;
5. `J_refr >= 0`;
6. lapse/shift не имеют собственных кинетических членов и остаются
   algebraic constraints;
7. для stationary оператора выполнено `M_11 > 0`, `M_22 > 0`, `det(M) > 0`.

Тогда:

1. FRW kinetic matrix положительно определена;
2. FRW gradient matrix положительно определена;
3. stationary radial operator эллиптичен;
4. дополнительные propagating modes отсутствуют;
5. ghost/gradient instabilities исключены в рабочей области reduced канона.

Доказательство:

1. FRW kinetic matrix в текущем каноне равна
   `K_FRW = diag(3/θ_eff² + γ², 1)`. При `θ_eff != 0` и `γ >= 0`
   диагональные элементы строго положительны, значит `K_FRW ≻ 0`.
2. FRW gradient matrix задан как
   `G_FRW = diag(c_θ² K_θ, c_Ψ²)`. При `c_θ² > 0`, `c_Ψ² > 0`
   и `K_θ > 0` она также положительно определена. Следовательно, gradient
   instability отсутствует.
3. `F'(J)>0` и `F''(J)<=0` означают, что refractive response монотонен и
   saturating, а не разгоняющийся. Это исключает sign flip и runaway в
   рефрактивном секторе.
4. `M_11 > 0`, `M_22 > 0`, `det(M) > 0` означает положительную
   определённость локального квазилинейного оператора. Значит stationary PDE
   эллиптичен, а hidden propagating branch не возникает.
5. Lapse/shift в ADM-скелете не получают кинетических членов, поэтому они не
   добавляют новых физических мод; они остаются ограничениями, а не
   распространяющимися степенями свободы.
6. Комбинируя пункты 1-5, получаем отсутствие ghost, gradient и extra modes в
   reduced каноне на всей рабочей области, где выполняются перечисленные
   предпосылки.

#### Следствие

Если reduced canon проходит `stability_ledger(...).healthy == True`, то он
прошёл:

1. ковариантную замкнутость на уровне текущего `S_red`;
2. проверку на ghost/gradient/tachyon/extra-mode в canonical рабочей области;
3. baseline-условие для перехода к 1-2 primary observables и только затем к fit.

#### Теорема 3. Полный perturbation ledger замыкает ADM-канон

Пусть выполнены условия Теоремы 1 и Теоремы 2, а также:

1. для quadratic sector заданы матрицы `K`, `G`, `M` в форме
   `L^(2) = 1/2 δq̇ᵀ K δq̇ - 1/2 δqᵀ[ G(k²/a²) + M ]δq`;
2. lapse/shift исключаются через Schur complement;
3. прямой и Schur-каналы дают одинаковый результат для `K`, `G`, `M`;
4. `stability_ledger` даёт `healthy = True` на рабочей области.

Тогда:

1. ADM-редукция согласована с covariant action ledger;
2. quadratic perturbation sector не содержит скрытого разъезда между direct и
   Schur completion;
3. рабочий perturbation proof закрыт в пределах текущего канона.

Для этого в репозитории существует единый ledger:

- `theory/perturbation_proof.py`

Его выходы:

1. `S_cov` как полный covariant completion;
2. `K_direct/G_direct/M_direct` и `K_schur/G_schur/M_schur`;
3. `routes_match`;
4. `health`;
5. `proof_closed`.

### 6.3) Минимальный derived `S_micro`

`S_micro -> S_red -> S_cov` фиксируется как рабочая иерархия. Чтобы не
начинать канон с `S_cov` как с постулата, ниже задаётся минимальный derived
microphysics layer, выводимый из базы `Θ_{μν}, Ψ`. Он нужен только как
локальный слой, из которого текущий `S_red` получается редукцией, а `S_cov`
- добавлением matter completion. Он не вводит новую геометрию.

#### 6.3.1) Поля и структура

Фундаментальные поля derived-слоя:

1. `Θ_{μν}` - clock sector;
2. `Ψ` - energy sector;
3. `χ_a` - материальные поля;
4. local deviation scalar `R_micro`, fixed as the minimal local quadratic
   deviation from the canonical weak-field point;
5. derived preferred-slicing volume form `measure_dt_d3x = dt d^3x`;
6. local response coefficients `Z_t(R_micro)` и `Z_s(R_micro)`.

Минимальный local action в preferred slicing:

```text
S_micro = ∫ dt d^3x [ L_Θ^micro + L_Ψ^micro + L_matter^micro + L_int^micro ].
```

Состав:

```text
R_micro = (θ_eff/θ_0 - 1)^2 + Ψ² + χ² + |∇θ_eff|² + |∇Ψ|² + |∇χ|²,
A_Θ = 1 + R_micro,
B_Θ = γ² + R_micro,
A_Ψ = 1 + R_micro,
B_Ψ = 1 + R_micro,
Z_t = 1 + R_micro,
Z_s = 1 + 2 R_micro,
V_Θ^micro = 1/2 θ_eff² + 1/4 θ_eff⁴,
V_Ψ^micro = 1/2 V0 R² + 1/4 R⁴,
V_matter^micro = 1/2 χ² + 1/4 χ⁴,
L_Θ^micro = 3/2 A_Θ · |∇ln θ_eff|² + 1/2 B_Θ · |∇θ_eff|² - V_Θ^micro,
L_Ψ^micro = 1/2 A_Ψ · |∇R|² - V_Ψ^micro,
L_matter^micro = 1/2 Z_t · χ̇² - 1/2 Z_s · |∇χ|² - V_matter^micro,
L_int^micro = R_micro · (θ_eff² R² + θ_eff² χ²).
```

#### 6.3.2) Weak-field matching

В weak-field limit:

```text
R_micro → 0,
A_Θ → 1,  B_Θ → γ²,  A_Ψ → 1,  B_Ψ → 1,
Z_t → 1,  Z_s → 1.
```

Matching-правило для рефрактивного сектора:

```text
n² = Z_s / Z_t.
```

Следовательно:

1. `n → 1` в однородном weak-field пределе;
2. отличие от weak-field живёт в локальных response coefficients, а не в
   новой онтологии геометрии;
3. `S_red` является reduced/covariant completion этого local action после
   интегрирования тяжёлых мод;
4. `S_cov` получается как `S_red + S_m[g_hat, matter]`, то есть через
   добавление matter completion поверх reduced слоя.
5. В коде мост фиксируется как совпадение `S_red_from_micro = S_red` и
   `S_cov_from_micro = S_cov` на уровне canonical ledgers.
6. Переход от `S_micro` к `S_red` теперь algebraic: он идёт через
   `R_micro -> 0`, а не через свободные функции коэффициентов или отдельную
   ручку силы отклика.

#### 6.3.2.1) Замыкание пунктов 1-3: propagation law

Этот подпункт фиксирует, почему `g/g_hat`, универсальность связи и weak-field
matching не являются отдельными постулатами поверх `Θ–Ψ`, а читаются как одна
цепочка вывода.

**Пункт 1: статус `g/g_hat`.**

В `S_micro` первичными остаются только:

```text
Θ_{μν}, Ψ, χ_a, R_micro, Z_t(R_micro), Z_s(R_micro).
```

Для matter-поля `χ` локальный квадратичный блок имеет вид:

```text
L_matter^micro =
  1/2 Z_t(R_micro) χ̇²
  - 1/2 Z_s(R_micro) |∇χ|²
  - V_matter(χ).
```

Отношение коэффициентов задаёт локальный propagation cone:

```text
c_matter² = Z_s/Z_t.
```

Поэтому запись

```text
n² = Z_s/Z_t
```

является не новой геометрией, а сжатым способом сказать: локальные часы,
световые сигналы и matter-процессы используют один и тот же response ratio.
`g_hat` вводится только после этого как bookkeeping-запись:

```text
S_matter[χ; Z_t,Z_s]  ->  S_matter[χ; g_hat],
g_hat = g + (n² - 1) u ⊗ u.
```

Следовательно, `g` и `g_hat` не являются третьей онтологической сущностью.
Они допустимы только если не несут независимых степеней свободы сверх
`Θ_{μν}, Ψ` и response coefficients.

**Пункт 2: универсальность.**

Универсальность связи означает:

```text
∀χ_a:  Z_t^a(R_micro) = Z_t(R_micro),
      Z_s^a(R_micro) = Z_s(R_micro).
```

Тогда для любых matter-секторов:

```text
n_a² = Z_s^a/Z_t^a = Z_s/Z_t = n².
```

Если появляются species-dependent коэффициенты

```text
Z_t^a != Z_t^b  или  Z_s^a != Z_s^b,
```

то это уже не текущий канон `Θ–Ψ`: разные вещества видят разные локальные
законы распространения, и принцип единого clock-sector response ломается.
Поэтому универсальность `Z_t,Z_s` является обязательным acceptance criterion
для derived micro layer.

**Пункт 3: weak-field matching.**

Weak-field точка определяется как:

```text
R_micro -> 0,
Z_t -> 1,
Z_s -> 1,
n² -> 1,
n -> 1.
```

Первая физически значимая поправка записывается через weak-field scalar
`Φ_eff`:

```text
n(θ_eff) = 1 + 2 Φ_eff + O(Φ_eff²).
```

Это не новая онтология, а нормировка observable layer. Она фиксирует один
коэффициент так, чтобы один и тот же `n` давал согласованные weak-field
эффекты:

```text
Δν/ν ≈ Δ ln n,
t_signal = ∫ n dl,
α_lens ≈ ∫ ∇_⊥ ln n dl,
a_probe = -∇ ln n.
```

Иными словами, weak-field matching закрыт в каноне тогда и только тогда,
когда:

1. `R_micro -> 0` возвращает инертный локальный режим;
2. все matter-секторы имеют один и тот же `n`;
3. линейная нормировка `n = 1 + 2 Φ_eff + O(Φ_eff²)` зафиксирована до fit;
4. higher-order поправки не вводят composition-dependent propagation.

**Итоговая цепочка:**

```text
local Θ–Ψ basis
  -> R_micro
  -> universal Z_t,Z_s
  -> n² = Z_s/Z_t
  -> g_hat as bookkeeping only
  -> weak-field observables
```

Это и есть замыкание пунктов 1-3 в текущем каноне. В коде оно закреплено
функцией `propagation_law_closure_ledger()` в `theory/micro_action.py` и
тестом `test_propagation_law_closure_pins_points_1_2_3`:

1. `g_hat_independent_dof = 0`;
2. `Z_t_chi_a = Z_t_chi_b`, `Z_s_chi_a = Z_s_chi_b`;
3. `R_micro -> 0` даёт `Z_t -> 1`, `Z_s -> 1`, `n -> 1`;
4. линейная observable-нормировка `n = 1 + 2 Φ_eff + O(Φ_eff²)` фиксируется
   до fit.

Следовательно, пункты 1-3 считаются закрытыми на уровне текущего канона,
ledger и тестов. Дальнейшая работа может уточнять higher-order поправки, но
не должна менять эту цепочку без явного изменения канона.

#### 6.3.2.1a) Anti-fit guardrails

Чтобы derived micro layer не превратился в набор ручек для получения нужных
чисел, вводится явное разделение:

```text
canonical = fixed before fit or derived from the action,
diagnostic = useful for scans, but not a physical claim.
```

В текущем каноне фиксируются:

```text
primary ontology = Θ_{μν}, Ψ,
θ_eff² = Θ_{μν}u^μu^ν,
Φ_eff = ln(θ_eff/θ_0),
weak-field slope n ≈ 1 + 2Φ_eff,
weak-field profile n(Φ_eff) = 1 + 2 asinh(Φ_eff),
n² = Z_s/Z_t,
EPR singlet rule E(a,b) = -a·b,
particle = stable localized Θ-projection of a Ψ-sector.
```

В diagnostic-only слой выносятся:

```text
weak-field profile alternatives: linear/exp/tanh2,
legacy explicit profile selector,
F_v1(J) = 1 - exp(-J),
ADM eps*/alpha* mixings,
legacy/exploratory sigma extension,
strong-field bridge calibration coefficients,
inverse-BAO required multipliers.
```

Их нельзя использовать как физический вывод. Любой такой элемент может быть
повышен до канона только при одновременном выполнении:

1. вывод из действия или из более малого набора постулатов;
2. фиксация до просмотра data-facing fit;
3. отсутствие выбора по test set;
4. отсутствие новых лёгких мод без наблюдательной цены;
5. независимое наблюдаемое предсказание.

Запрещённые ходы:

```text
choose_profile_after_data,
retune_bridge_on_test,
promote_inverse_diagnostic_to_correction,
use_adm_eps_without_derivation,
add_sigma_as_physical_field_without_new_canon.
```

Кодовая фиксация: `anti_fit_ledger()` и `canonical_promotion_check()` в
`theory/anti_fit.py`, тесты в `tests/test_anti_fit.py`.

#### 6.3.2.1b) Что уже выводится из действия

Из текущего минимального `Θ–Ψ` действия для
`q = (δθ, δR, δφ)` следует диагональный high-k квадратичный блок:

```text
K = diag(3/θ² + γ², 1, R²),
G = diag(3/θ² + γ², 1, R²),
M = diag(0, V0 + 3Q²/(a⁶R⁴), 0).
```

Следствие:

```text
epsK* = 0,
epsG* = 0,
alpha_k2* = 0,
c_s² = (1, 1, 1).
```

Это не дополнительная подгонка, а отрицательный вывод из действия: если в
действии нет недиагональных kinetic/gradient-операторов и higher-spatial
derivative операторов, то соответствующие смешения в квадратичном действии
не появляются. Ненулевые `epsK*`, `epsG*`, `alpha_k2*` могут войти в канон
только вместе с явно записанными новыми операторами в `S_cov`.

Для преломляющего сектора текущее действие выводит аргумент

```text
J_refr = Φ_eff · I_grad,
L_refr = M_*⁴ F(J_refr),
```

и требования к функции:

```text
F(0)=0,
F'(J) ≥ 0,
finite response for the no-horizon branch.
```

Но сам выбор экспоненциальной формы из текущего действия не следует.
Следует только то, что `J_*` не является отдельным физическим параметром и
может быть поглощён в нормировку `J_refr`.

Strong-field bridge scales (`theta_c_scale`, `m2_scale`, `lam_scale`,
`powerlaw2d`) тоже не следуют из текущего действия. Для их вывода нужно решить
сourced static Euler-Lagrange equations с граничными условиями, а не
калибровать коэффициенты по EHT/GW target-набору.

Кодовая фиксация: `theory/action_derivation.py`,
`tests/test_action_derivation.py`.

Принцип, который делает экспоненту выводом, а не выбором формы:

```text
S(J) = 1 - F(J),
S(J1 + J2) = S(J1) S(J2).
```

То есть независимые приращения `J` действуют без памяти и мультипликативно
на оставшуюся незадействованную часть отклика. При непрерывности такой закон
даёт

```text
S(J) = exp(-J/J_*),
F(J) = 1 - exp(-J/J_*).
```

В этой записи `J_*` уже не физическая ручка, а только нормировка остатка
отклика.

#### 6.3.2.2) Vacuum offset source-map

Проверка вакуумной катастрофы в текущем каноне формулируется не как
утверждение "нулевая энергия отсутствует". Нулевые энергии matter-мод могут
существовать и проявляться в boundary-разностях, например в эффекте Казимира.
Проверяется другое: абсолютная аддитивная константа в effective matter action
не является прямым источником для `Θ–Ψ`.

Минимальная запись:

```text
Γ_matter^eff = Γ_response[Z_t,Z_s,n,boundary data] + ∫ dt d³x Λ_vac_offset.
```

Source map для базиса `Θ_{μν}, Ψ` берётся только от вариаций по локальным
response data, то есть по тем величинам, которые меняют режим распространения:

```text
source_ΘΨ ~ δΓ_eff/δθ_eff, δΓ_eff/δR, δΓ_eff/δR_micro.
```

Для абсолютного offset:

```text
δΛ_vac_offset/δθ_eff = 0,
δΛ_vac_offset/δR = 0,
δΛ_vac_offset/δR_micro = 0.
```

Значит, такой offset не входит в `Θ–Ψ` источник как огромная космологическая
постоянная. При этом boundary/response-вклады остаются физическими:

```text
Γ_boundary ~ ΔΓ_boundary · n(R_micro),
Γ_response ~ Δresponse · n(R_micro),
δΓ_boundary/δR_micro != 0,
δΓ_response/δR_micro != 0.
```

Иными словами:

1. абсолютная сумма `1/2 Σ ħω` не является наблюдаемым `Θ–Ψ` response сама по
   себе;
2. Казимир сохраняется, потому что это разность спектров при разных boundary
   data;
3. локальная QFT сохраняет нулевые моды как часть effective action, но
   гравитирующий/source-вклад проходит через response map, а не через
   произвольный additive constant.

Кодовая фиксация: `vacuum_offset_source_ledger()` и тесты
`test_vacuum_offset_source_ledger_*` в `tests/test_micro_action.py`.

#### 6.3.3) Статус

Это derived layer from the `Θ_{μν}, Ψ` basis, а не отдельная онтология. Он
нужен, чтобы:

1. не начинать канон с `S_cov` как с постулата;
2. держать явным правило `n² = Z_s/Z_t`;
3. показывать, где именно появляется matter response.

#### 6.3.3.1) Закрытие stationary strong-field branch

Стационарная strong-field ветка закрывается отдельным ledger:

```text
strongfield_branch_closure_ledger()
```

Этот ledger закрывает стационарный профиль. Его область:

```text
closure_scope = stationary_strongfield_branch_only,
dynamic_collapse_closed = handled_by_dynamic_collapse_closure_ledger.
```

Критерии закрытия стационарной ветки:

1. `theta_positive = 1`: `θ_eff` остается на положительной ветви;
2. `n_finite = 1`: профиль `n` конечен;
3. `finite_redshift = 1`: `z_surface` конечен;
4. `finite_delay = 1`: optical delay конечен;
5. `outgoing_characteristic_exists = 1`: исходящая характеристика не исчезает;
6. `finite_energy = 1`: радиальный интеграл энергии конечен;
7. `radial_operator_elliptic = 1`: стационарный радиальный оператор не теряет
   эллиптичность;
8. `regular_center = 1`: центр профиля регулярный.

Граничные условия стационарной ветки:

```text
θ_eff(0) > 0,
θ_eff'(0) = 0,
R'(0) = 0,
θ_eff(r -> infinity) = θ_0,
R(r -> infinity) = R_0.
```

Forward map фиксируется как обязательная цепочка:

```text
n_profile
  -> z_surface
  -> delay
  -> ray_map_from_n_profile
  -> transfer_function_from_finite_delay.
```

Ветка фальсифицируется, если выполняется хотя бы одно:

```text
θ_eff reaches zero,
n diverges,
energy diverges,
radial operator is not elliptic,
no outgoing characteristic,
forward observables incompatible.
```

В коде это закреплено тестом
`test_strongfield_branch_closure_pins_horizonless_criteria`.

#### 6.3.3.2) Закрытие dynamic collapse как evolution contract

Динамический collapse закрывается не как численный PDE-solver, а как
канонический evolution contract:

```text
dynamic_collapse_closure_ledger()
```

Область:

```text
closure_scope = dynamic_collapse_ledger_contract,
dynamic_collapse_closed = 1,
requires_numeric_pde_implementation = 1.
```

Смысл: solver может быть разным, но он обязан реализовать один и тот же
контракт. Эволюционные переменные:

```text
θ_eff(t,r),
R(t,r),
n(t,r),
E_total(t),
min θ_eff(t),
max n(t).
```

Критерии закрытия:

1. `initial_finite_energy = 1`;
2. `theta_eff_positive_all_t = 1`;
3. `n_finite_all_t = 1`;
4. `energy_finite_all_t = 1`;
5. `no_blow_up_before_branch_decision = 1`;
6. `outgoing_characteristic_exists_all_t = 1`;
7. `hyperbolic_evolution_operator = 1`;
8. `stationary_or_dispersive_outcome = 1`.

Разрешённые исходы:

```text
settle_to_stationary_branch,

```

Запрещённые исходы:

```text

no_outgoing_characteristic.
```

Итоговая цепочка:

```text

  -> evolve_Theta_Psi
  -> track_theta_min_and_n_max
  -> preserve_outgoing_characteristic
  -> avoid_blow_up
  -> settle_or_disperse
  -> reject_invalid_outcomes.
```

В коде это закреплено тестом
`test_dynamic_collapse_closure_contract_pins_evolution_criteria`.

#### 6.3.4) Reduction check

Чтобы мост был не декларативным, а вычислимым, фиксируем слабополевой
редукционный тест:

1. `A_Θ -> 1`, `B_Θ -> 1`, `A_Ψ -> 1`, `B_Ψ -> 1`;
2. `Z_t -> 1`, `Z_s -> 1`;
3. `n² -> 1` и, следовательно, `n -> 1`;
4. `L_Θ^micro` и `L_Ψ^micro` возвращаются к canonical quadratic blocks;
5. `L_matter^micro` возвращается к canonical local matter kinetic block;
6. тяжёлые моды интегрируются и не остаются отдельным propagating sector.

Этот тест не заменяет физический вывод из базы `Θ_{μν}, Ψ`. Он фиксирует,
что derived microphysics layer не ломает weak-field matching и действительно
ведёт к уже принятому reduced канону.

#### 6.3.5) Micro perturbation check

Отдельно фиксируем минимальный perturbation sanity для derived micro layer:

1. `A_Θ > 0`, `B_Θ > 0`, `A_Ψ > 0`, `B_Ψ > 0`;
2. `Z_t > 0`, `Z_s > 0`;
3. `c_Θ² = B_Θ / A_Θ > 0`, `c_Ψ² = B_Ψ / A_Ψ > 0`;
4. `n² = Z_s / Z_t > 0`;
5. `no_new_modes == True` на рабочей ветке.

Это не полный UV-proof на уровне фундаментальной микрофизики. Это честный
микропертурбационный барьер: derived layer должен быть здоровым уже на
локальном квадратичном уровне, иначе его нельзя поднимать выше.

#### 6.3.6) Derivation ladder

Чтобы derived слой не расползался в отдельные костыли, в коде и каноне фиксируется
явная лестница вывода:

1. `S_micro`;
2. `weak_field_projection`;
3. `integrate_heavy_modes`;
4. `assemble_S_red`;
5. `add_S_matter_completion`;
6. `S_cov`.

Эта лестница не претендует на UV-единственность. Она фиксирует, что текущий
derived microphysics layer не просто “совместим” с каноном базы `Θ_{μν}, Ψ`,
а ведёт к нему в одном и том же порядке редукции.

#### 6.3.7) Single strong-field consequence

Из derived micro layer сохраняем только один сильнопольный выход:

1. `finite redshift` на compact branch;
2. `finite delay` на compact branch;
3. `ok_no_horizon == True`.

Это не второй канал с отдельной физикой и не набор прокси. Это один
strong-field consequence, который читается через
`micro_strongfield_primary_ledger()` и `micro_to_strongfield_primary_bridge()`.
Он фиксирует, что derived strong-field consequence не расползается по
альтернативным сильнопольным индикаторам, а сводится к одному
наблюдаемому классу: `compact_no_horizon`.

#### 6.3.8) Symbol audit

Для публичного разбора каждая сущность микрослоя получает статус:

1. `derived` - выводится из projection, matching или reduction from the Θ–Ψ basis;
2. `assumed` - минимальный объект, без которого reduced completion не
   собирается;
3. `diagnostic` - итоговая проверка, не базовая физика.

В текущем derived micro layer это распределяется так:

```text
derived: theta_eff, measure_dt_d3x, R_micro, A_Θ, B_Θ, A_Ψ, B_Ψ, Z_t, Z_s,
         n, n^2, S_micro, S_red, S_cov, bridge_to_covariant
assumed: R, chi
diagnostic: finite redshift, finite delay, ok_no_horizon
```

Это и есть честная публичная форма ответа на вопрос “почему именно эти
коэффициенты”: большая часть символов не выбирается произвольно, а помечается
как derived от reduction rules; `measure_dt_d3x = dt d^3x` - derived
preferred-slicing volume form, а `R_micro` - минимальный локальный
квадратичный deviation scalar, выведенный из локальности, чётности,
positivity и weak-field normalization.

## 7) Канонический класс `n(θ_eff)`

Family-v1 (current working class):

```text
n(θ_eff; p, θs) = 1 + 𝒩(p · θs / θ_eff),
p>0, θs>0.
```

Свойства `𝒩`:

1. `𝒩(0)=0`;
2. `𝒩` монотонна на рабочей ветке;
3. на физической ветке `n` остаётся конечным без введения отдельного
   потолка как физической сущности.

Правила:
1. Семейство фиксируется до подгонки.
2. Смена семейства после просмотра результата запрещена.
3. Альтернативы допускаются только как заранее объявленный blind-test.

## 8) Жесткие критерии отбраковки

Модель/ветка отвергается, если стабильно нарушен хотя бы один пункт:

1. Weak-field тесты (Меркурий/Шапиро/красное смещение/отклонение света).
2. Устойчивость (ghost/gradient) в рабочей области параметров.
3. Совместный контур `CC+SN+BAO+strong-field`.
4. Согласие достигается только скрытой подгонкой на test-наборе.
5. Нет стабильного решения при фиксированном `n(θ_eff)` классе.

## 9) Статус Ψ (обязательный nested-тест)

Сектор `Ψ` считается каноническим только при прохождении:

1. `M0`: `Θ`-only (без динамического `Ψ`);
2. `M1`: `Θ+Ψ`;
3. единый train/test протокол;
4. критерий принятия: `ΔBIC = BIC(M1)-BIC(M0) < -6` стабильно на k-fold.

Если `ΔBIC >= 0` в большинстве fold, `Ψ` исключается из канона.

from theory.tensor_symbolic import (
