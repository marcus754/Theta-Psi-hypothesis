# FRW reduced EOM (V6)

Короткая рабочая версия фоновых уравнений в той форме, которая реально
используется в background/frw_background.py.

## Переменные

- N = ln a.
- x — регуляризация поля времени: theta = (sqrt(3)/gamma) tanh(x).
- y = dx/dN.
- R — reduced amplitude Ψ = R e^{iφ}, u = dR/dN.

## Ограниченная кинетическая загрузка


A_raw(x,y,u) = 3/2 * (sech^4 x / tanh^2 x) * y^2 + 1/2 * u^2 + 3/2 * sech^4 x * y^2
A = 3 * tanh^2( sqrt(max(A_raw,0)/3) ),   0 <= A < 3


## Фридман

В N-форме:


H^2 =
  [ rho_m + rho_r + rho_L
    + 1/2 V0 R^2
    + 1/2 Q^2/(a^6 R^2) ] /
  [ 3 - A ] .


На однородном FRW пространственный источник рефрактивного режима отсутствует:
I_grad=0, поэтому J_refr=0 и strong-field/refractive сектор не входит в
фоновые уравнения.

## Уравнение для Hdot


Hdot = -1/2 * [ gamma^2 theta_dot^2 + R_dot^2 + 3 (theta_dot/theta)^2
                + Q^2/(a^6 R^2)
                + rho_m + 4/3 rho_r ] .


## Strong-field сектор на FRW

На однородном фоне пространственный источник режима отсутствует:


I_grad = 0   (на однородном фоне)


Итог:


J_refr = Φ_eff(theta_eff) * I_grad = 0 .


Отдельного режимного поля sigma в каноне больше нет. Старые sigma*
параметры в коде допустимы только как legacy-совместимость и не должны
использоваться для вывода физических формул.
