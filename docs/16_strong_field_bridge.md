# 16. Мост сильных полей

Этот документ фиксирует пошаговый вывод связи между фоновыми параметрами и характеристиками компактных объектов в гипотезе Тета-Пси.

1. Постановка стационарной задачи
Рассматривается сферически-симметричная область вокруг компактного источника. Поля Тета (время) и Пси (энергия) зависят только от радиуса r.
Стационарный радиальный функционал системы:
L_stat = 1/2 I_grad - U(theta, Psi) + alpha F(J_refr)
где:
I_grad = (theta')^2 + w_Psi (Psi')^2 - инвариант неоднородности.
J_refr = Phi_eff(theta) I_grad - аргумент рефрактивного отклика.
Phi_eff(theta) = ln(theta/theta_0) - логарифмический потенциал отклонения от фона.

2. Уравнения Эйлера-Лагранжа
Варьирование по полям дает систему квазилинейных уравнений:
1/r^2 d/dr [ r^2 A(r) theta' ] = - dU/d_theta + alpha F'(J) (1/theta) I_grad
1/r^2 d/dr [ r^2 A(r) w_Psi Psi' ] = dU/d_Psi
Функция A(r) = 1 + 2 alpha Phi_eff F'(J) определяет эффективную проводимость вакуума.

3. Асимптотика слабого поля (Согласование)
На больших расстояниях (r -> infinity):
I_grad -> 0, Phi_eff -> 0, следовательно A(r) -> 1.
Уравнения переходят в стандартную форму Пуассона:
Delta theta = Source.
Решение: theta(r) = theta_0 (1 - r_g/r).
Это связывает градиент Тета на бесконечности с Ньютоновской массой M (через радиус r_g).
Индекс преломления в этом пределе: n = 1 + 2 Phi_eff = 1 + 2 ln(1 - r_g/r) = 1 + 2 Phi_N.

4. Возникновение масштаба моста (Brid >= Scale)
При приближении к источнику градиент (theta')^2 растет. Влияние рефрактивного сектора (alpha) становится значимым, когда член 2 alpha Phi_eff F' становится сопоставим с единицей.
Из структуры J_refr следует возникновение естественного масштаба длины r_c:
r_c^2  alpha / V0
Этот масштаб определяет область, в которой нелинейные эффекты Тета-Пси начинают доминировать над классической гравитацией, предотвращая коллапс в сингулярность.

В текущей версии стационарный сильнопольный режим считается замкнутым как эффективная задача.

В рамках этой постановки получены:
- конечный показатель преломления;
- отсутствие бесконечного красного смещения;
- существование исходящей световой характеристики;
- согласование со слабым полем на больших расстояниях;
- регулярное поведение решений.

Пока не закрыты:
- динамический коллапс;
- образование компактного объекта из произвольных начальных условий;
- полный нестационарный режим.

5. Насыщение и отсутствие горизонта
Благодаря экспоненциальному насыщению F(J) = 1 - exp(-J), производная F'(J) стремится к нулю при росте градиентов.
Это приводит к тому, что проводимость вакуума A(r) перестает расти, и значение поля Тета в центре остается конечным и положительным:
lim(r->0) theta(r) = theta_c > 0.
Следствие: показатель преломления n остается конечным, что исключает формирование математического горизонта событий.

Итог
Мост сильных полей является прямым следствием динамики эффективной проводимости вакуума A(r). Он связывает массу (дальнее поле), параметр связи alpha (интенсивность отклика) и потенциал V0 (упругость субстрата), полностью определяя структуру компактных объектов без привлечения внешних допущений.
