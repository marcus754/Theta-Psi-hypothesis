# 08. Foundations — ядро и доказуемые условия


## Канон времени

- Фундаментальный объект времени в гипотезе: тензор `Θ_{μν}` (см. `docs/11_derivation_path.md`).
- В формулах этого раздела используется редуцированная FRW-переменная
  `θ ≡ θ_eff`, где `θ_eff^2 = Θ_{μν} u^μ u^ν`, `u^μ u_μ = -1`.
- Все записи вида `n(θ)` в этом файле следует читать как `n(θ_eff)`.
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику `Θ_{μν}`.

## Каноническая постановка (обязательно для всех разделов)

В этом файле фиксируется именно математическая твердь гипотезы. В текущем
каноне первичны только `Θ_{μν}` и `Ψ`, а strong-field отклик строится
напрямую через составной инвариант

```
J_refr(Θ,Ψ)=Φ_eff(θ_eff)·I_grad(Θ,Ψ),
I_grad(Θ,Ψ)=l_θ^2 · |∇_⊥θ|^2 + l_Ψ^2 · |∇_⊥Ψ|^2.
```

На FRW-фоне:

```
I_grad = 0,   J_refr = 0.
```

Индекс вакуума задаётся конечной рабочей формой:

```
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr),
𝒩(0)=0,\quad 𝒩(J)\ge 0,\quad 𝒩(J)\ \text{монотонна и конечна}.
```

Невалидная ветка фона исключена параметризацией:

```
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).
```

Цель: зафиксировать «твердь» — формулы и достаточные условия, на которых
строится вся феноменология на базе `Θ_{μν}, Ψ`, с прозрачными связями код ↔ формулы.

## Единицы и масштабы

- Выбраны 8πG=1, c=1 ⇒ H² = ρ_tot / 3. Конверсия H0 из км/с/Мпк: см. 02_background.md.
- γ — безразмерный в этих единицах. Реалистичный режим: γ ≲ 10⁻¹⁸.

## Однородный лагранжиан и регуляризация

Поля θ(t), Ψ(t)=R e^{iφ}:

```
L_eff = 3/2 (θ̇/θ)² + 1/2 Ṙ² − 1/2 V0 R² + 1/2 γ² θ̇²,
Q = a³ R² φ̇,
ρ_phase = p_phase = 1/2 · Q²/(a⁶ R²).
```

Уравнение движения для θ имеет полюс при |θ|=√3/|γ|. Регуляризация
```
θ = (√3/γ) tanh x,   θ̇ = (√3/γ) sech²x · ẋ
```
устраняет полюс: |x|→∞ соответствует границе по θ. См. 02_background.md.

С точки зрения физики ещё более прозрачная переменная — канонический логарифм
χ ≡ √3 ln θ. Тогда кинетика становится просто ½χ̇², а граница θ→0 соответствует
χ→−∞ без особого поведения в лагранжиане. Такой переход фиксирует нормировку
динамики, потому что коэффициент 3/2 перед (θ̇/θ)² обеспечивает каноническую
метрику на логарифмической координате. Можно свободно работать в этой χ-параметризации
или в x=artanh(γθ/√3) — оба атласируют одну и ту же физику, но χ явно показывает,
что никакой физической сингулярности нет, просто координата выходит в бесконечность.
Символьные и численные формулы приведены в background/frw_symbolic.py, где χ-интерпретацию
берём как исходную, а регуляризация x остаётся удобной для численных схем.

## Каноническая конечная параметризация (без ручного отсечения ветки)

Чтобы «нежелательная» ветка не отрезалась постфактум, а отсутствовала в самой
математической постановке, используем конечную рабочую форму эффективной
нагрузки в уравнении Фридмана:

```
A_raw = A_raw(x, y_N, u_N)  >= 0,
χ_A = sqrt(A_raw / 3),
A = 3 tanh^2(χ_A)  in [0, 3),
H^2 = (ρ_bg + V/2 + ρ_phase) / (3 - A).
```

Свойства:
- при `A_raw << 1` получаем `A ≈ A_raw` (слабопольный режим не искажается);
- область `A >= 3` исключена конструктивно, а не «фильтром после расчета».

## Скалярные возмущения и ADM‑исключение ограничений

После линеаризации по δq=(δθ, δR, δφ) получаем квадратичное действие

```
L^(2) = 1/2 · δq̇ᵀ K δq̇ − 1/2 · δqᵀ A δq,
A(a,k) = G (k²/a²) + M.
```

Lapse/shift (алгебраические) переменные c=(δN,B) вносят
```
L^(2) ⊃ − δqᵀ B c − 1/2 · cᵀ C c,
```
и исключаются по стационарности c* = − C⁻¹ Bᵀ δq ⇒

```
A_eff = A − B C⁻¹ Bᵀ,
G_eff и M_eff — как части A_eff при (k²/a²) и константе.
```

Символьная реализация: theory/adm_symbolic.py (`effective_matrices_sym`).
Численный API: src/linear_modes_adm.py (параметр `elim={...}`).

Базовые диагональные ядра (согласованы с L_eff и Ψ=R e^{iφ}):
```
Kθθ = 3/θ² + γ²,   KRR = 1,   Kφφ = R²;
Gθθ = c_θ² · Kθθ,  GRR = c_R², Gφφ = c_φ² · R²;
MRR = V0,          Mθθ = Mφφ = 0.
```

Смешения и корректировки α_k² допускаются параметрически и контролируются
в интерфейсе (см. theory/adm_symbolic.py).

## Достаточные условия гиперболичности и субсветовости

1) Отсутствие призраков (K ≻ 0):
   - Без смешений: Kθθ>0, KRR=1>0, Kφφ=R²>0 ⇒ K ≻ 0.
   - Со смешениями: условия Сильвестра на главные миноры K.

2) Положительность «градиентного оператора» в кинетической метрике:
   - В high‑k: положительные собственные значения K⁻¹ G_eff.
   - При конечных k: положительность собственных значений K⁻¹[ G_eff (k²/a²) + M_eff ].

3) Субсветовость скалярных мод: max eig(K⁻¹ G_eff) ≤ 1 + tol.

Замечание о Шур‑комплементе:
Если C ≻ 0, то A_eff = A − B C⁻¹ Bᵀ симметрична. Для достаточно малых ‖B‖
и положительных диагоналей G,M сохраняется положительность A_eff. Это отражено
в тестах стабильности с малыми смешениями (см. tests/test_linear_modes_adm.py и новый тест с `elim`).

## Consistency: weak-field и тензорный сектор

- PPN-слой в текущей реализации рассматривается как weak-field диагностический
  модуль. Интерфейс `checks/ppn_full.compute_ppn_params` возвращает текущую
  рабочую конфигурацию до внедрения полного PN‑вывода.
- Тензорный сектор: α_T=0 ⇒ c_T=1. См. docs/07_tensor_optical_ppn.md и tests/test_tensor_speed.py.

## Реалистичный режим и инвариантность фона к γ

Для γ ≈ 10⁻¹⁸ и начальному состоянию без динамики (Ψ≈0, θ̇≈0) фон H(a) совпадает
с собственным инертным фоновым пределом в пределах численной точности. См.
tests/test_gamma_limit.py.

## Weak-field matching как граничное условие

Оптический индекс вводится в конечной рабочей форме и одновременно
фиксируется по слабому полю:

```
n(θ_eff) = 1 + 𝒩(Φ_eff(θ_eff)),
𝒩(0)=0,\quad 𝒩\ge 0,\quad 𝒩\ \text{монотонна и конечна},
```

где `Φ_eff(θ_eff) >= 0` — гладкая weak-field рабочая форма.

При `Φ_eff -> 0`:

```
n(θ_eff) ≈ 1 + κ Φ_eff.
```

Канонический выбор `κ=2` задаёт нормировку weak-field отклика
`n ≈ 1 + 2Φ_eff` (в `c=1`) как наблюдательное граничное условие.



## Минимальная двухсущностная модель для поля времени и субстрата

Выбираем минимально‑сцепленное действие двух скаляров в редуцированном
поле‑пространстве:
- Метрика поля: G_{θθ}(θ) = γ² + 3/θ²,  G_{RR}=1,  G_{φφ}=R².
- Градиенты (high‑k): c_s=1 для всех мод ⇒ G=K.
- Масс‑блок: M_{RR} = V0 + 3 Q²/(a⁶ R⁴),  M_{θθ}=M_{φφ}=0 (при V(θ)=const, V(φ) отсутствует).

Исключение δN,B даёт стандартный квадратичный функционал с K=G_ab и G=G_ab на градиентах; конечнок‑k эффекты и вся динамика
массового блока собраны в M (см. theory/adm_symbolic.py).

## Каузальность
- Переносчик причинности — фотон; c=1 фиксирует световой конус.
- Тензорный сектор немодифицирован (EH) ⇒ c_T=1.
- Скалярные моды в high‑k имеют c_s=1 (K=G). Любой случай c_s²>1 запрещён veto.
- На конечных k учитываются k^0‑поправки ΔM_ab, не меняющие high‑k скорость.
