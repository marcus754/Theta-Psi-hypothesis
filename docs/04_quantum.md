# 04. Квантовая гауссова надстройка

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор Theta_munu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная
  theta ≡ theta_eff, где theta_eff^2 = Theta_munu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику Theta_munu.


## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются Theta_munu и Psi; редуцированный контур не требует внешнего порога и не использует ручное отсечение веток.
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + M_*^4 F(J_refr) ],
J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi).

Источник режима (без Weyl-порога в каноне):
I_grad(Theta,Psi) = l_theta^2 · |∇_⊥theta|^2 + l_Psi^2 · |∇_⊥Psi|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^mu).

Смысл:
- В FRW (однородный фон): I_grad=0, поэтому J_refr=0.
- В сильное поле (большие пространственные градиенты): I_grad>0, поэтому преломляющий сектор включается напрямую через F(J_refr).
- Отдельного режимного поля в каноне нет.

Индекс вакуума задаётся конечной рабочей формой:
n(theta_eff,Theta,Psi)=1+𝒩(J_refr),
𝒩(0)=0,𝒩(J) >= 0,𝒩(J) монотонна и конечна.

Невалидная ветка фона исключена параметризацией (а не фильтром):
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).

В сравнении с GR проверяется не «ручное включение», а прямой ответ на пространственные неоднородности Theta_munu, Psi.

Идея: квантуем линейные моды dq_k = (d_theta_k, d_R_k, deltaφ_k). Состояние — гауссово, полностью задаётся матрицей ковариации sum(t,k).

Квадратичный гамильтониан (после исключения Лапса/Шифта):
L^(2) = 1/2 · dq̇ᵀ K dq̇ − 1/2 · dqᵀ A dq,
A = G (k²/a²) + M.

Переходя к каноническим переменным x = (dq, π) с π = K dq̇, получаем матрицу Гамильтона
H = 1/2 · xᵀ H_x x,   H_x = [[ A, 0 ], [ 0, K⁻¹ ]].

Каноника и HUP
[x_i, x_j] = i Ω_ij  ⇒  sum + i Ω / 2 ≥ 0,
Ω = [[0, I], [−I, 0]].

Эволюция sum (линейная):
Ẋ = Ω H_x X  ⇒  Ẋ  →  suṁ = Ω H_x sum + sum H_xᵀ Ωᵀ.
Это ODE для каждой моды (t,k) и позволяет вычислять выжатие, когерентность и запутанность пар (k,−k).

Ожидаемые эффекты: выжатие при k/a ≲ H, рост запутанности, стабилизация в режиме k/a ≫ H. Источник выжатия — неадиабатичность фонового Theta–Psi режима

Метрики
- Энтропия запутанности (для пар мод), меры когерентности/декогеренции (по sum).
- Энергетическая плотность кванто‑флуктуаций: ⟨H⟩ = 1/2 Tr(H_x sum) (внимание к ультрафиолетовой регуляризации: k‑срез или аддитивные контрчлены).

## Казимир как граничная проверка согласованности

Эффект Казимира в этой рамке не добавляет новую сущность к базису
Theta_munu, Psi. Он относится к производному микрофизическому слою matter-сектора: границы меняют допустимые моды поля χ, а вакуумная энергия появляется как разность спектров при разных boundary data.

Минимальная локальная запись matter-сектора:
L_matter^micro =
  1/2 Z_t χ_dot² - 1/2 Z_s |∇χ|² - V_matter(χ).

При однородных коэффициентах отклика дисперсия граничных мод в координатах локального временного слоения имеет вид:
ω_k = sqrt(Z_s/Z_t) |k|.

Поэтому для параллельных пластин на расстоянии a Casimir-вклад масштабируется тем же множителем:
E_C/A = - pi²/(720 a³) * sqrt(Z_s/Z_t),
P_C   = - pi²/(240 a⁴) * sqrt(Z_s/Z_t).

В слабополевом пределе:
R_micro -> 0,
Z_t -> 1,
Z_s -> 1,
n² = Z_s/Z_t -> 1,

получается стандартный результат:
E_C/A = - pi²/(720 a³),
P_C   = - pi²/(240 a⁴).

Смысл проверки: Казимир является граничным квантовым вкладом matter-мод в одном и том же Theta–Psi-слое отклика. Он не должен включать FRW-рефрактивный сектор сам по себе и не должен вводить независимую энергию вакуума как третью онтологию. Математическая граничная проверка согласованности закреплена в theory.micro_perturbation.casimir_boundary_реестр().
