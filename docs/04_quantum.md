# 04. Квантовая гауссова надстройка

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор `Θ_{μν}` (см. `docs/11_derivation_path.md`).
- В формулах этого раздела используется редуцированная FRW-переменная
  `θ ≡ θ_eff`, где `θ_eff^2 = Θ_{μν} u^μ u^ν`, `u^μ u_μ = -1`.
- Все записи вида `n(θ)` в этом файле следует читать как `n(θ_eff)`.
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику `Θ_{μν}`.


## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются `Θ_{μν}` и `Ψ`; редуцированный контур не
требует внешнего порога и не использует ручное отсечение веток.

```
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_{ΘΨ}
                   + M_*^4 F(J_refr) ],
J_refr(Θ,Ψ)=Φ_eff(θ_eff)·I_grad(Θ,Ψ).
```

Источник режима (без Weyl-порога в каноне):

```
I_grad(Θ,Ψ) = l_θ^2 · |∇_⊥θ|^2 + l_Ψ^2 · |∇_⊥Ψ|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^μ).
```

Смысл:
- В FRW (однородный фон): `I_grad=0`, поэтому `J_refr=0`.
- В strong-field (большие пространственные градиенты): `I_grad>0`, поэтому
  преломляющий сектор включается напрямую через `F(J_refr)`.
- Отдельного режимного поля в каноне нет.

Индекс вакуума задаётся конечной рабочей формой:

```
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr),
𝒩(0)=0,\quad 𝒩(J)\ge 0,\quad 𝒩(J)\ \text{монотонна и конечна}.
```

Невалидная ветка фона исключена параметризацией (а не фильтром):

```
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).
```

В сравнении с GR проверяется не «ручное включение», а прямой ответ на
пространственные неоднородности `Θ_{μν}, Ψ`.

Идея: квантуем линейные моды δq_k = (δθ_k, δR_k, δφ_k). Состояние — гауссово, полностью задаётся матрицей ковариации Σ(t,k).

Квадратичный гамильтониан (после исключения lapse/shift):

```
L^(2) = 1/2 · δq̇ᵀ K δq̇ − 1/2 · δqᵀ A δq,
A = G (k²/a²) + M.
```

Переходя к каноническим переменным x = (δq, π) с π = K δq̇, получаем матрицу Гамильтона

```
H = 1/2 · xᵀ H_x x,   H_x = [[ A, 0 ], [ 0, K⁻¹ ]].
```

Каноника и HUP
```
[x_i, x_j] = i Ω_{ij}  ⇒  Σ + i Ω / 2 ≥ 0,
Ω = [[0, I], [−I, 0]].
```

Эволюция Σ (линейная):
```
Ẋ = Ω H_x X  ⇒  Ẋ  →  Σ̇ = Ω H_x Σ + Σ H_xᵀ Ωᵀ.
```
Это ODE для каждой моды (t,k) и позволяет вычислять выжатие, когерентность и запутанность пар (k,−k).

Ожидаемые эффекты: выжатие при k/a ≲ H, рост запутанности, стабилизация в deep‑subhorizon режиме k/a ≫ H.

Метрики
- Энтропия запутанности (для пар мод), меры когерентности/декогеренции (по Σ).
- Энергетическая плотность кванто‑флуктуаций: ⟨H⟩ = 1/2 Tr(H_x Σ) (внимание к ультрафиолетовой регуляризации: k‑срез или аддитивные контрчлены).

## Казимир как boundary sanity-check

Эффект Казимира в этой рамке не добавляет новую сущность к базису
`Θ_{μν}, Ψ`. Он относится к matter-сектору derived microphysics layer:
границы меняют допустимые моды поля `χ`, а вакуумная энергия появляется как
разность спектров при разных boundary data.

Минимальная локальная запись matter-сектора:

```
L_matter^micro =
  1/2 Z_t χ_dot² - 1/2 Z_s |∇χ|² - V_matter(χ).
```

При однородных response coefficients дисперсия граничных мод в координатах
preferred slicing имеет вид:

```
ω_k = sqrt(Z_s/Z_t) |k|.
```

Поэтому для параллельных пластин на расстоянии `a` Casimir-вклад масштабируется
тем же множителем:

```
E_C/A = - pi²/(720 a³) * sqrt(Z_s/Z_t),
P_C   = - pi²/(240 a⁴) * sqrt(Z_s/Z_t).
```

В weak-field limit:

```
R_micro -> 0,
Z_t -> 1,
Z_s -> 1,
n² = Z_s/Z_t -> 1,
```

получается стандартный результат:

```
E_C/A = - pi²/(720 a³),
P_C   = - pi²/(240 a⁴).
```

Смысл проверки: Казимир является граничным квантовым вкладом matter-мод в
одном и том же `Θ–Ψ` response layer. Он не должен включать FRW-refractive
сектор сам по себе и не должен вводить независимую "энергию вакуума" как
третью онтологию. Кодовый sanity-check закреплён в
`theory.micro_perturbation.casimir_boundary_ledger()`.
