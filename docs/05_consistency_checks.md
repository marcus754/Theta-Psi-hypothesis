# 05. Критерии здоровья и отсечки

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор Theta_munu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная theta ≡ theta_eff, где theta_eff^2 = Theta_munu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную тензорную динамику Theta_munu.


## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются Theta_munu и Psi; редуцированный контур не
требует внешнего порога и не использует ручное отсечение веток.

S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + M_*^4 F(J_refr) ],
J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi).

Источник режима (без Weyl-порога в каноне):
I_grad(Theta,Psi) = l_theta^2 · |∇_⊥theta|^2 + l_Psi^2 · |∇_⊥Psi|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^mu).

Смысл:
- В FRW (однородный фон): I_grad=0, поэтому J_refr=0 и рефрактивный сектор не получает источника.
- В сильное поле (большие пространственные градиенты): I_grad>0, поэтому преломляющий сектор включается напрямую как функция F(J_refr).
- Отдельного режимного поля σ в каноне нет. Это убирает третью сущность и оставляет первичными только Theta_munu и Psi.

Конечность индекса вакуума обеспечивается конструкцией модели:
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).

Численные коэффициенты регуляризации относятся только к интегратору и не записываются как физическая сущность.

Невалидная ветка фона исключена параметризацией (а не фильтром):
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).

В сравнении с GR проверяется не «ручное включение», а прямой ответ F(J_refr) на пространственные неоднородности Theta_munu, Psi.

Базовые проверки (используются в сканах):
- K ≻ 0 (нет призраков)
- eig(K⁻¹G) > 0 (нет градиентных инстабильностей)
- Субсветовость: c_s² ≤ 1 (с запасом по tol)
- Конечные k: ω²(k) > 0 для всех мод (если включено)
- Ранняя Вселенная (BBN): H(a) монотонно убывает на a ∈ [1e−10, 1e−6] (опционально)

Режим по умолчанию — строгий режим (только следствия из лагранжиана и уравнений движения). Эвристические фильтры (предел эффективной теории и PPN‑прокси) по умолчанию отключены и могут быть включены явно для быстрых сканов; они не считаются частью физики модели.

В проекте есть унифицированный агрегатор проверок checks/health.py, который принимает траекторию фона и параметры и возвращает флаги ok_*.

Примечание: канонический ADM-контур использует поля q=(d_theta,d_R,deltaφ). Старые sigma-расширения и strict sigma-model проверки не считаются частью канона и могут оставаться только как устаревшие тестовые каркасы до удаления из API.

## Приложение: аналитическая область параметров (встроено из бывшего раздела 09)

## Аналитические условия здоровья и область параметров

Здесь собраны простые и прозрачные условия, обеспечивающие «здоровье» скалярного сектора при текущей аппроксимации, и указаны рабочие области параметров. Они полезны как априори до сравнений с данными.

Обозначения: q=(d_theta,d_R,deltaφ), K,G,M из L^(2)=1/2 dq̇ᵀKdq̇ − 1/2 dqᵀ[ G (k²/a²)+M ]dq. 
Диагональные ядра:
Kthetatheta = 3/theta² + gamma²,   KRR = 1,   Kφφ = R²,
Gthetatheta = c_theta² Kthetatheta,     GRR = c_R², Gφφ = c_φ² R²,
MRR = V0,           Mthetatheta = Mφφ = 0.

## 1) Отсутствие призраков (K ≻ 0)

Без смешений (ε=0) — достаточно:
Kthetatheta > 0  (всегда при |theta|<√3/|gamma|),  KRR = 1 > 0,  Kφφ = R² > 0  ⇒ R ≠ 0.

Со смешениями (малые ε_K): по Сильвестру для 3×3

Kthetatheta > 0,
det [[Kthetatheta, εthetaR],[εthetaR, KRR]] > 0     ⇒  εthetaR² < Kthetatheta·KRR,
det K > 0                           ⇒  |εthetaφ|, |εRφ| достаточно малы (явные формулы опускаем).

Практически: держать |ε_K,ij| ≪ minKthetatheta, KRR, Kφφ.

## 2) Отсутствие градиентной неустойчивости (собственные значения K⁻¹G положительны)

Без смешений: собственные c_s² = c_theta², c_R², c_φ² ⇒ требуются c_theta² > 0,  c_R² > 0,  c_φ² > 0.

Субсветовость накладывает c_i² ≤ 1 (с небольшим запасом >1 для учёта округления в численных расчётах).

Со смешениями (малые ε_G): аналогично п.1, но для матрицы K⁻¹G. Достаточно требовать малость ε_G и положительность диагональных c_i².

## 3) Положительность дисперсии при конечном k (ω² > 0)

ω² — собственные значения K⁻¹ [ G (k²/a²) + M ].

При диагонали и V0 ≥ 0: вклад G (k²/a²) положителен, а M не отрицателен ⇒ ω² > 0 ∀ k>0. В точке k→0 одна мода (R) получает массу V0, остальные — нулевые (ожидаемо для theta и фазы φ).

## 4) Регуляризация theta и пределы

Рабочая область по фону обеспечивает |theta| < theta⋆ = √3/|gamma| через замену theta=(√3/gamma) tanh x. Это устраняет особенность параметризации и сохраняет Kthetatheta>0.

## 5) предел эффективной теории (фон)

Грубая область применимости эффективной теории по фону:
|thetȧ/theta| < Λ,   |gamma thetȧ| < Λ,    Λ ≪ 1  (в планковских единицах).

Типично в численных прогонах берём Λ  1e−1 как жёсткий срез.

## 6) PPN‑прокси (локальные тесты)

Быстрая эвристика (не строгий предел):
ε_loc ≃ 1 / (gamma² theta² + 3)  <  ε_max.

Порог ε_max подбирается как мягкий фильтр (в коде дефолтно 0.5 в тестах); строгие PPN‑границы потребуют полноценного вывода слабополевого эффективного описания. Не следует трактовать этот прокси как финальный результат.

## 6.1) Статус слабополевых проверок

- Reference: GPS clock correction, Hafele-Keating, Pound-Rebka, Shapiro,
  solar redshift, solar deflection, Mercury, Lense-Thirring, binary pulsar
  timing, LLR, atomic limit, twin-paradox, c_T=1.

## 7) Итоговый «рецепт» рабочей области

- Фон: |theta| < √3/|gamma|,   |thetȧ/theta| < Λ, |gamma thetȧ| < Λ.
- Кинетика: Kthetatheta>0, R≠0; при ε_K ≠ 0 — проверка миноров по Сильвестру.
- Градиенты: c_i² ∈ (0, 1] и малые ε_G.
- Конечное k: V0 ≥ 0 ⇒ ω²(k) > 0 при k>0.
- Локальные тесты: ε_loc < ε_max (прокси), уточняется в будущем полном выводе PPN.
- Спутниковые часы и GPS: эталонная проверка с опубликованным эталоном порядка 38 mus/day.
- Лабораторные и транспортные часы: Pound-Rebka, Hafele-Keating.
- Орбитальная и релятивистская динамика: Shapiro, solar redshift, solar deflection, Mercury, Lense-Thirring, тайминг двойных пульсаров, лазерная локация Луны.
- Атомный / химический предел: в слабое поле пределе n(theta_eff) → 1, gamma_PPN = 1, beta_PPN = 1; никаких дополнительных сдвигов для стандартной релятивистской химии.

Приемлемые диапазоны параметров в кодовых единицах только для демонстрационных численных прогонов:
gamma  O(1),  V0 ∈ [1e−3, 1],  Q ≈ 0 (или |Q| ≪ 1),
ctheta², cR², cphi² ∈ (0,1],  |ε_K|, |ε_G| ≪ 1.

Сопоставление с реальностью требует gamma_phys ≲ 1e−18; это условие применяется как априори при анализе данных.
