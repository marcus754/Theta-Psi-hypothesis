# 05. Критерии здоровья и отсечки

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор `Θ_{μν}` (см. `docs/11_derivation_path.md`).
- В формулах этого раздела используется редуцированная FRW-переменная
  `θ ≡ θ_eff`, где `θ_eff^2 = Θ_{μν} u^μ u^ν`, `u^μ u_μ = -1`.
- Все записи вида `n(θ)` в этом файле следует читать как `n(θ_eff)`.
- Этот раздел описывает вычислительный редуцированный контур, а не полную
  тензорную динамику `Θ_{μν}`.


## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются `Θ_{μν}` и `Ψ`; редуцированный контур не
требует внешнего порога и не использует ручное отсечение веток.

```
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_{ΘΨ}
                   + M_*^4 F(J_refr) ],
J_refr(Θ,Ψ)=Φ_eff(θ_eff)·I_grad(Θ,Ψ).
```

Источник режима (без Weyl-порога в каноне):

```
I_grad(Θ,Ψ) = l_θ^2 · |∇_⊥θ|^2 + l_Ψ^2 · |∇_⊥Ψ|^2,
∇_⊥ = пространственная часть градиента (ортогонально локальному u^μ).
```

Смысл:
- В FRW (однородный фон): `I_grad=0`, поэтому `J_refr=0` и рефрактивный
  сектор не получает источника.
- В strong-field (большие пространственные градиенты): `I_grad>0`, поэтому
  преломляющий сектор включается напрямую как функция `F(J_refr)`.
- Отдельного режимного поля `σ` в каноне нет. Это убирает третью сущность и
  оставляет первичными только `Θ_{μν}` и `Ψ`.

Индекс вакуума ограничен конструктивно:

```
n(θ_eff,Θ,Ψ)=1+𝒩(J_refr).

Численные regularization-коэффициенты относятся только к solver'у и не
записываются как физическая сущность.
```

Невалидная ветка фона исключена параметризацией (а не фильтром):

```
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).
```

В сравнении с GR проверяется не «ручное включение», а прямой ответ `F(J_refr)` на
пространственные неоднородности `Θ_{μν}, Ψ`.

Базовые проверки (используются в сканах):
- K ≻ 0 (нет призраков)
- eigen(K⁻¹G) > 0 (нет градиентных инстабильностей)
- Субсветовость: c_s² ≤ 1 (с запасом по tol)
- Конечные k: ω²(k) > 0 для всех мод (если включено)
- Ранняя Вселенная (BBN): H(a) монотонно убывает на a ∈ [1e−10, 1e−6] (опционально)

Режим по умолчанию — strict (только следствия из лагранжиана и EOM). Эвристические
фильтры (EFT‑срез и PPN‑прокси) по умолчанию отключены и могут быть включены явно
для быстрых сканов; они не считаются частью физики модели.

В проекте есть унифицированный агрегатор проверок `checks/health.py`, который
принимает траекторию фона и параметры и возвращает флаги `ok_*`.

Примечание: канонический ADM-контур использует поля `q=(δθ,δR,δφ)`. Старые
`sigma`-расширения и strict sigma-model проверки не считаются частью канона и
могут оставаться только как deprecated/legacy тестовые каркасы до удаления из API.

## Приложение: аналитическая область параметров (встроено из бывшего раздела 09)

# 09. Условия здоровья и область параметров (аналитика)

Здесь собраны простые и прозрачные условия, обеспечивающие «здоровье» скалярного сектора при текущей аппроксимации, и указаны рабочие области параметров. Они полезны как априори до сравнений с данными.

Обозначения: `q=(δθ,δR,δφ)`, `K,G,M` из `L^(2)=1/2 δq̇ᵀKδq̇ − 1/2 δqᵀ[ G (k²/a²)+M ]δq`. Диагональные ядра:

```
Kθθ = 3/θ² + γ²,   KRR = 1,   Kφφ = R²,
Gθθ = c_θ² Kθθ,     GRR = c_R², Gφφ = c_φ² R²,
MRR = V0,           Mθθ = Mφφ = 0.
```

## 1) Отсутствие призраков (K ≻ 0)

Без смешений (ε=0) — достаточно:

```
Kθθ > 0  (всегда при |θ|<√3/|γ|),  KRR = 1 > 0,  Kφφ = R² > 0  ⇒ R ≠ 0.
```

Со смешениями (малые ε_K): по Сильвестру для 3×3

```
Kθθ > 0,
det [[Kθθ, εθR],[εθR, KRR]] > 0     ⇒  εθR² < Kθθ·KRR,
det K > 0                           ⇒  |εθφ|, |εRφ| достаточно малы (явные формулы опускаем).
```

Практически: держать |ε_K,ij| ≪ min{Kθθ, KRR, Kφφ}.

## 2) Отсутствие градиентной неустойчивости (eigen K⁻¹G > 0)

Без смешений: собственные `c_s² = {c_θ², c_R², c_φ²}` ⇒ требуются

```
c_θ² > 0,  c_R² > 0,  c_φ² > 0.
```

Субсветовость накладывает `c_i² ≤ 1` (с небольшим запасом >1 для учёта округления в численных расчётах).

Со смешениями (малые ε_G): аналогично п.1, но для матрицы K⁻¹G. Достаточно требовать малость ε_G и положительность диагональных c_i².

## 3) Положительность дисперсии при конечном k (ω² > 0)

```
ω² — с.зн. K⁻¹ [ G (k²/a²) + M ].
```

При диагонали и V0 ≥ 0: вклад `G (k²/a²)` положителен, а M не отрицателен ⇒ ω² > 0 ∀ k>0. В точке k→0 одна мода (R) получает массу V0, остальные — нулевые (ожидаемо для θ и фазы φ).

## 4) Регуляризация θ и пределы

Рабочая область по фону обеспечивает `|θ| < θ⋆ = √3/|γ|` через замену `θ=(√3/γ) tanh x`. Это исключает полюс и обеспечивает Kθθ>0.

## 5) EFT‑срез (фон)

Грубая область применимости эффективной теории по фону:

```
|θ̇/θ| < Λ,   |γ θ̇| < Λ,    Λ ≪ 1  (в планковских единицах).
```

Типично в численных прогонах берём Λ ~ 1e−1 как жёсткий срез.

## 6) PPN‑прокси (локальные тесты)

Быстрая эвристика (не строгий предел):

```
ε_loc ≃ 1 / (γ² θ² + 3)  <  ε_max.
```

Порог ε_max подбирается как мягкий фильтр (в коде дефолтно 0.5 в тестах); строгие PPN‑границы потребуют полноценного вывода метрики. Не следует трактовать этот прокси как финальный результат.

## 6.1) Статус weak-field checks

- Reference: GPS clock correction, Hafele-Keating, Pound-Rebka, Shapiro,
  solar redshift, solar deflection, Mercury, Lense-Thirring, binary pulsar
  timing, LLR, atomic limit, twin-paradox, c_T=1.

## 7) Итоговый «рецепт» рабочей области

- Фон: |θ| < √3/|γ|,   |θ̇/θ| < Λ, |γ θ̇| < Λ.
- Кинетика: Kθθ>0, R≠0; при ε_K ≠ 0 — проверка миноров по Сильвестру.
- Градиенты: c_i² ∈ (0, 1] и малые ε_G.
- Конечное k: V0 ≥ 0 ⇒ ω²(k) > 0 при k>0.
- Локальные тесты: ε_loc < ε_max (прокси), уточняется в будущем полном выводе PPN.
- Спутниковые часы и GPS: reference check с опубликованным эталоном порядка
  `38 μs/day`.
- Лабораторные и транспортные часы: Pound-Rebka, Hafele-Keating.
- Орбитальная и релятивистская динамика: Shapiro, solar redshift, solar
  deflection, Mercury, Lense-Thirring, binary-pulsar timing, lunar-laser ranging.
- Атомный / химический предел: n(θ_eff) → 1, γ_PPN = 1, β_PPN = 1; никаких
  дополнительных сдвигов для стандартной релятивистской химии.

Приемлемые диапазоны параметров в кодовых единицах для демонстраций:

```
γ ~ O(1),  V0 ∈ [1e−3, 1],  Q ≈ 0 (или |Q| ≪ 1),
ctheta², cR², cphi² ∈ (0,1],  |ε_K|, |ε_G| ≪ 1.
```

Сопоставление с реальностью требует γ_phys ≲ 1e−18; это условие применяется как априори при анализе данных.
