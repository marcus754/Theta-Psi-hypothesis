# 15. Эмерджентность квантовых и электродинамических эффектов

Статус раздела: Это согласованная реконструкция, а не строгий первопринципный вывод из действия. Показывается, что известные структуры (квантование, электродинамика, уравнение Шрёдингера) допускают естественную интерпретацию в рамках Theta–Psi, но не выводятся из него единственным образом на данном этапе.

Проверка EPR/CHSH в текущей версии считается закрытой:
- воспроизводится квантовая граница Цирельсона;
- сохраняется отсутствие сверхсветовой передачи сигнала;
- корреляционный сектор не использует свободные подгоночные параметры.

При этом полный вывод квантовой механики из базового действия пока не завершён.

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор Theta_munu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная theta ≡ theta_eff, где theta_eff^2 = Theta_munu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную тензорную динамику Theta_munu.

## Каноническая постановка (обязательно для всех разделов)

В проекте базой принимаются Theta_munu и Psi; редуцированный контур не требует внешнего порога и не использует ручное отсечение веток.
S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + M_*^4 F(J_refr) ],
J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi).

На однородном FRW-фоне: I_grad = 0, J_refr = 0.

Документ фиксирует возможный механизм возникновения квантовых и электродинамических эффектов из базиса Theta–Psi. Здесь не утверждается, что вся квантовая теория уже строго выведена из первых принципов. Речь идет о согласованной реконструкции, в которой известные структуры появляются как естественные пределы динамики Theta–Psi.

## 1. Возникновение постоянной Планка hbar

### Шаг 1.1. Фазовый заряд

В гипотезе энергия представлена комплексным полем:
Psi = R · exp(i φ)

Глобальная симметрия относительно фазового сдвига
φ -> φ + const

порождает сохраняющийся ток Нётер и соответствующий заряд.

В координатах предпочтительного временного слоения заряд имеет вид:
Q = ∫ d³x √(-g) · R² · φ̇

Фазовый заряд имеет размерность действия, то есть энергии, умноженной на время.

### Шаг 1.2. Топологические сектора

Поскольку фаза φ определена на окружности S¹, поле допускает топологически нетривиальные конфигурации.

Для замкнутого контура вокруг дефекта, где R = 0, выполняется условие:
∮ dφ = 2πn, где n — целое число.

Это определяет дискретные winding-сектора поля Psi.

Для стационарных конфигураций с конечной энергией фазовый заряд связывается с топологическим числом сектора:
Q_n = n · q0

где q0 — минимальная устойчивая порция фазового действия.

Таким образом, квантование возникает как следствие топологической структуры фазового сектора Psi.

### Шаг 1.3. Идентификация с hbar

В Theta–Psi минимальный квант фазового действия отождествляется с наблюдаемой постоянной Планка:
hbar = q0

В этой интерпретации hbar не является внешней фундаментальной константой, а представляет собой минимальный устойчивый квант фазовой динамики Psi.

## 2. Физическая природа волн де Бройля

Рассмотрим локализованную солитоноподобную конфигурацию Psi, движущуюся со скоростью v:
Psi(x,t) = R(x − vt) · exp(i[φ₀(x − vt) + kx − ωt])

Фазовая часть действия определяет канонический импульс:
p = ħ · ∇φ

Для плоской составляющей:
grad(phi) = k = 2 pi / lambda

Следовательно:
p = ħ · 2π/λ = h/λ

Таким образом, длина волны де Бройля возникает как пространственный период фазовой структуры Psi.

## 3. Возникновение электродинамики

### Шаг 3.1. Локальная фазовая ковариантность

Тензор Theta задает локальную структуру причинного распространения. Если фазовая динамика Psi должна оставаться локально согласованной, глобальная U(1)-симметрия естественным образом расширяется до локальной:
Psi(x) -> exp(iα(x)) · Psi(x)

Для сохранения локальной ковариантности требуется компенсирующее поле связи.

### Шаг 3.2. Калибровочное поле

Минимальная локально-инвариантная производная имеет вид:
Dmu = ∂mu − i(e/ħ) Amu

Поле A_mu возникает как gauge-связность, обеспечивающая согласованность локальной фазовой структуры Psi.

### Шаг 3.3. Динамика и распространение

Динамика gauge-сектора задается стандартным инвариантом:
Fmunu = ∂mu Anu − ∂nu Amu

Низкоэнергетическое действие gauge-сектора имеет вид:
L_gau >= = −1/4 · √(−g_hat) · Fmunu F^munu

Все поля распространяются в эффективной метрике:
g_hat_mu_nu = g_mu_nu + (n^2 - 1) · u_mu u_nu

где n(theta) — универсальный индекс преломления Theta-сектора.

Вариация действия по Amu даёт:
∇mu F^munu = J^nu

Таким образом, электродинамика возникает как низкоэнергетический gauge-сектор фазовой динамики Psi в среде переменной причинной структуры.

## 4. Нерелятивистский предел и уравнение Шрёдингера

### Шаг 4.1. Релятивистский Psi-сектор

Релятивистская динамика Psi задаётся уравнением Клейна–Гордона:
(Dmu D^mu + m²) Psi = 0

### Шаг 4.2. Нерелятивистский предел

Выделим быструю фазу покоя:
Psi = psi · exp(-i m t / hbar)

В пределе малых скоростей v << c медленная огибающая psi удовлетворяет:
i hbar · dpsi/dt = [ -hbar^2 / (2m) · grad^2 + V_ext ] psi

Таким образом, уравнение Шрёдингера возникает как нерелятивистский предел Psi-сектора.

### Шаг 4.3. Гравитационный потенциал

В Theta–Psi внешний потенциал является следствием неоднородности event-frequency поля:
V_ext = m · Phi_N

где Phi_N — ньютоновский предел Theta-сектора.

## 5. Unified Layer

Полное низкоэнергетическое действие записывается как:
S = integral d4x sqrt(-g) · [ (M_Pl^2 / 2) R + L_Theta + L_gau >= + L_Psi + L_refr - U_tot ]

где:
L_Theta = 3/2 · (thetȧ/theta)² + 1/2 · gamma² · thetȧ²
(L_Theta здесь — кинетическая часть Тета-сектора; потенциальный член −1/2 V0 R² включён в L_Psi канонического L_eff)
L_gau >= = −1/4 · F²
L_Psi = |DmuPsi|² − 1/2 V0 R²
L_refr = M_*⁴ · F(J_refr)

Индекс преломления определяется как:
n(theta) = 1 + 𝒩(J_refr)

## Заключение

В рамках Theta–Psi квантование связывается с топологической структурой фазового сектора Psi. Электродинамика возникает как локально-ковариантная фазовая динамика. Уравнение Шрёдингера появляется как нерелятивистский предел Psi-сектора. Гравитация проявляется через неоднородность causal/event-frequency структуры Theta.

Модель не претендует на завершенный вывод всей квантовой теории из первых принципов, однако задает внутренне связный emergent-framework, в котором основные структуры современной физики возникают как согласованные пределы единой Theta–Psi динамики.
