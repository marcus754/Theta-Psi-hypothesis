# 02. Фон: единицы, лагранжиан, уравнения

## Канон времени

- Фундаментальный объект времени в гипотезе: тензор Theta_munu (см. docs/11_derivation_path.md).
- В формулах этого раздела используется редуцированная FRW-переменная theta ≡ theta_eff, где theta_eff^2 = Theta_munu u^mu u^nu, u^mu u_mu = -1.
- Все записи вида n(theta) в этом файле следует читать как сокращённую запись полного отклика n(theta_eff, Theta, Psi).
- Этот раздел описывает вычислительный редуцированный контур, а не полную тензорную динамику Theta_munu.

## Каноническая постановка (обязательно для всех разделов)

Этот файл относится только к FRW-модулю. Он использует физический базис гипотезы, но не заменяет собой ни онтологию времени, ни сильное поле слой.

S = ∫ d^4x √(-g) [ (M_pl^2/2)R + L_m + L_ThetaPsi + L_refr(Theta,Psi) ].

В текущем каноне первичны только Theta_munu и Psi. В полной записи Psi рассматривается как комплексный параметр порядка Psi = R e^iφ; в FRW-редукции используется амплитуда R и фазовый заряд Q = a^3 R^2 dot φ, а |Psi|^2 естественно читается как плотность амплитуды. Сильное поле отклик параметризуется напрямую составным инвариантом

J_refr(Theta,Psi)=Phi_eff(theta_eff)·I_grad(Theta,Psi),
I_grad(Theta,Psi)=l_theta^2 · |∇_⊥theta|^2 + l_Psi^2 · |∇_⊥Psi|^2.

На однородном FRW-фоне:
I_grad = 0,   J_refr = 0.

Поэтому фоновая космология сама по себе не должна включать сильное поле рефрактивную ветвь. Рабочая форма индекса остаётся
n(theta_eff,Theta,Psi)=1+𝒩(J_refr).

Невалидная ветка фона исключена параметризацией:
A = 3 tanh^2( sqrt(A_raw/3) ) in [0,3),
H^2 = (ρ_bg + V/2 + ρ_phase)/(3-A).

Раздел задает только базовые единицы, однородный лагранжиан и FRW-уравнения, которые используются во всех численных проверках фонового модуля.
